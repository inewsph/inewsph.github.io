9. TUGAS MENULISKAN NOTASI ALGORITMA DALAM BENTUK PSEUDOCODE1. Menghitung luas segitiga2. Menghitung keliling persegi panjang jika diketahui panjang dan luas​


Jawaban:

program menghitung_luas_segitiga

deklarasi

var alas : integer

var tinggi : integer

var diketahui : array { alas, tinggi }

algoritma:

INPUT diketahui

IF (diketahui INCLUDES alas AND diketahui INCLUDES tinggi) then

output Math: ½ * alas * tinggi

ELSE then

output "Tolong masukkan alas dan tinggi segitiga!"

program menghitung_keliling_persegi

deklarasi

var panjang : integer

var lebar : integer

var diketahui : array { panjang, lebar }

algoritma:

INPUT diketahui

IF (diketahui INCLUDES panjang AND diketahui INCLUDES lebar) then

output Math: 2 * (panjang + lebar)

ELSE then

output "Tolong masukkan panjang dan lebar persegi panjang!"

Pseudocode merupakan kode yang mengilustrasikan suatu program dengan Bahasa yang mudah dipahami, jadi ini hanya sebagai contoh cara kerja kodenya nanti dan apa yang dilakukan kodenya. Karena biasanya Algoritma berupa kalimat deskriptif lebih rumit dipahami maka pseudocode lebih banyak dipakai dalam membuat algoritma yang memudahkan programmer untuk merealisasikan algoritmanya menjadi kode.

Jadi, kode diatas ini bukan kode yang work atau bekerja di suatu engine atau machine yaa, ini hanya ilustrasi suatu algoritma kode.

14. buatlah algoritma menghitung luas Limas segi empat menggunakan pseudocode​


Algoritma Luas Limas Segi Empat

Deklarasi: s1, s2, t sebagai bilangan real

[tex] \: [/tex]

1. Baca s1, s2, t

2. luas_segitiga = (s1 * t) / 2

3. luas_segiempat = s1 * s2

4. luas_limas = luas_segitiga + luas_segiempat

5. Tampilkan "Luas Limas Segi Empat adalah:", luas_limas

[tex] \: [/tex]

Akhir Algoritma Luas Limas Segi Empat

[tex] \: [/tex]

Catatan:

s1 dan s2 adalah panjang dan lebar segi empat dasar Limas.t adalah tinggi Limas.Luas segitiga diperoleh dengan mengalikan setengah dari s1 dengan t.Luas segi empat diperoleh dengan mengalikan s1 dengan s2.Luas Limas diperoleh dengan menambahkan luas segitiga dan luas segi empat.
