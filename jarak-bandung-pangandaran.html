1. Jarak Bandung ke Pangandaran pada peta 11 cm. Skala pada peta 1:2.000.000. Berapa jarak sebenarnya antara Bandung ke Pangandaran ?​


Diketahui:

jp: 11 cm

skala: 1:2.000.000

Ditanya:

jarak sebenarnya

Dijawab:

js = jp × skala

js = 11 × 2.000.000

js = 22.000.000 cm

js = 220 km

jarak sebenarnya = 220 km

Diket:

JP = 11 cm

Skala = [tex]\frac{1}{2.000.000}[/tex]

Dit:

JS = ?

Jwb:

JS = JP ÷ Skala

JS = 11 ÷ [tex]\frac{1}{2.000.000}[/tex]

JS = 11 × [tex]\frac{2.000.000}{1}[/tex]

JS = 22.000.000 cm

JS = 220 km

17. Sebuah bus berangkat dari bandung menuju pangandaran pukul 20.30 sampai di Pangandaran pukul 03.00 pagi hari dengan kecepatan 52 km / jam, jarak bandung ke pangandaran adalah...tolong jawab dengan caranya...​


Jawaban:

Sebuah bus berangkat dari bandung menuju pangandaran pukul 20.30 sampai di Pangandaran pukul 03.00 pagi hari dengan kecepatan 52 km / jam, jarak bandung ke pangandaran adalah 338 km.

Penjelasan dengan langkah-langkah:

Diketahui :

t = 20.30 + 03.00

= 6,5 jam

v = 52 km/jam

Ditanya :

Jarak dari Bandara ke Pangandaran (s)?

Jawab :

v = s/t

52 = s/6,5

s = 52 × 6,5

= 338 km

Kesimpulan :

Maka, jarak dari Bandara ke Pangandaran adalah 338 km.
