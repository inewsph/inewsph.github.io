3. setarakan persamaan reaksi tersebut!NaOH + H2SO4 = Na2SO4(aq) + H2O (l)


NaOH + H2SO4 =>. Na2SO4 + H2O

Asumsikan koefisien zat dengan a, b, c, d, sehingga menjadi :
a NaOH + b H2SO4 => c Na2SO4 + d H2O

Ingat!
atom dikiri = atom dikanan

Na => a = 2c ... (1)
S => b = c .... (2)
O => a + 4b = 4c + d ....(3)
H => a + 2b = 2d .....(4)

Misal : a = 1 ; maka :
a = 2c ...(1)
c = ½

b = c = ½ ..... (2)

a + 4b = 4c + d .... (3)
d = a + 4b - 4c
d = 1

Jadi persamaan reaksi :
NaOH + ½ H2SO4 => ½ Na2SO4 + H2O
- Atau bisa juga persamaan reaksi di atas dikali 2-
2 NaOH + H2SO4 => Na2SO4 + 2 H2O

13. setarakanlah persamaan reaksi dari NaOH + H2SO4 = Na2SO4 + H2Otolong pake jalan yaa​


Jawaban:

NaOH + H2SO4 = Na2SO4 + H2O

2NaOH + H2SO4 = Na2SO4 + 2H2O

Penjelasan:

Bandingkan kiri dengan kanan

Awal kiri = Awal kanan

Na (1) = Na (2)

O (5) = O (5)

H (3) = H (2)

S (1) = (S) (1)

Samakan kiri dengan kanan, agar Na kiri sama dengan Na kanan, maka kira berikan koefisien 2.

Karena koefisien molekul NaOH = 2, maka Na, O dan H jumlahnya menjadi 2.

Na (2) = Na (2)

O (6) = O (5)

H (4) = H (2)

S (1) = (S) (1)

Karena mengganti koefisien NaOH belum setimbang, maka kita tambah koefisien suatu molekul agar setimbang di bagian kanan. yaitu kita berikan koefisien 2 ke molekul H2O. Karena koefisien H2O = 2, maka H dan O jumlahnya menjadi 4 dan 2, jadi-

Na (2) = Na (2)

O (6) = O (6)

H (4) = H (4)

S (1) = (S) (1)

Setara, deh.

22. setarakan NaOH+H2So4---->Na2So4+H2O


Cara penyetaraan reaksi

(1). aNaOH(aq) + bH₂SO₄(aq) → cNa₂SO₄(aq) + dH₂O(l)

Koefisien ruas kiri = koefisien ruas kanan
Na ⇒ a = 2c
O ⇒ a + 4b = 4c + d
H ⇒ a + 2b = 2d
S ⇒ b = c
Jika a = 2, maka c = 1, dan b = 1, lalu substitusikan ke a + 4b = 4c + d
menjadi 2 + 4 = 4 + d, sehingga diperoleh d = 2
Reaksi sudah setara:
2NaOH(aq) + H₂SO₄(aq) → Na₂SO₄(aq) + 2H₂O(l)
