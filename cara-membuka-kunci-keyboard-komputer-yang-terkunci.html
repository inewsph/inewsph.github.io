21. Jelas 5 Kata kunci Dari keyboard


Jawaban:

Ctrl + C berfungsi untuk melakukan copy pada obyek yang dipilih (memilih obyek, fungsinya hampir sama seperti Ctrl + X namun shortcut ini berfungsi untuk memilih).

Ctrl + S berfungsi untuk melakukan save pada program yang sedang diproses, biasanya berlaku pada microsoft word, microsoft powerpoint, notepad, paint, ataupun microsoft excel.

Ctrl + A berfungsi untuk memilih atau select seluruh obyek pada program yang sedang diproses.

Ctrl + X berfungsi untuk melakukan cut pada obyek yang dipilih (memotong obyek, fungsinya hampir sama seperti Ctrl + C namun shortcut ini berfungsi untuk memotong).

Ctrl + V berfungsi untuk melakukan paste obyek yang telah di cut ataupun copy

Jawaban:

1. Fungsi Tombol Esc

Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.

2. Fungsi Tombol Tab

Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.

3. Fungsi Tombol Caps Lock 

Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.

4. Fungsi Tombol Shift

Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).

5. Fungsi Tombol Ctrl

Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.

6. Fungsi Tombol Alt

Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.

7. Fungsi Tombol Enter

Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.

8. Fungsi Tombol Back Space 

Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.

9. Tombol Delete

Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

10. Tombol Macro 

Macro ini jika ditekan akan muncul seperti ini "\".

11. Tombol Insert 

Untuk menyisipkan karakter di posisi kursor.

12. Tombol Home

Fungsinya untuk langsung menuju bagian atas halaman atau lembar kerja.

13. Tombol End

Fungsinya untuk langsung menuju bagian bawah halaman atau lembar kerja.

14. Tombol Page Up

Fungsinya untuk menggerakkan baris satu demi satu layar ke atas.

15. Tombol Page Down

Fungsinya untuk menggerakkan baris satu demi satu layar ke bawah.

24. bagaimana cara menghubungkan keyboard dengan komputer?


masukkan kabel keyboard ke port yg ada di CPU.
Tutorial Cara Menghubungkan Keyboard MIDI / MIDI Controller dengan Laptop Untuk bisa memainkan program instrument musik di computer, khususnya software/vst seperti keyboard synthesizer atau piano, diperlukan sebuah alat yang disebut: MIDI Controller atau bisa juga disebut Keyboard Controller. Keyboard musik biasa, baik itu tipe arranger, synthesizer, maupun workstation dapat saja difungsikan sebagai MIDI Controller.
--> Hanya saja kan harganya tinggi, dan peruntukkannya lebih maksimal digunakan untuk live play. Untuk proses music production / recording mending pakai Keyboard Controller saja. Mengenai Keyboard Controller Gwe bahas di tulisan lain. Jadi, di tulisan kali ini kita langsung saja, ada 3 (tiga) benda penting untuk bisa menghubungkan Keyboard Controller dengan laptop, yaitu:


1. Laptop (sebenarnya bisa juga untuk Desktop Komputer) yang dilengkapi program music DAW, contohnya seperti: FL Studio, Pro Tools, Logic Pro, Ableton, dll.

2. MIDI Controller, atau Keyboard musik biasa yang memiliki output berupa MIDI atau USB. Ada beberapa merek Keyboard/MIDI Controller seperti E-MU, M-AUDIO, dll.

3. Kabel USB yang digunakan untuk menghubungkan keyboard dengan laptop. Ini adalah Kabel tipe yang sama yang digunakan utuk menghubungkan printer dengan komputer/laptop.

-->
Selanjutnya, langkah-langkah yang perlu dilakukan untuk mewujudkan hal itu adalah:

1. Hidupkan MIDI / Keyboard Controller. Posisi On. Beberapa Keyboard MIDI / MIDI Controller tidak memiliki tombol ON / OFF. Dia otomatis ON bila dikoneksikan ke laptop/komputer;

2. Setelah laptop dalam posisi On dan ready, jalankan program DAW semacam FL Studio atau program musik lain;

3. Masuk ke menu option program musik (dalam hal ini FL Studio). Biasanya di situ ada pilihan/opsi: Input MIDI. Pilih: Enable atau Aktifkan;

4. Coba tekan tombol Keyboard Controller untuk memastikan hubungan/koneksi sudah aktif. Pilih instrumen/channel di program musik komputer sesuai dengan pilihan / keinginan Anda. Sebagai tes Anda dapat menggunakan intrumen keyboard atau piano. Apabila sudah aktif, akan muncul/bunyi yang sesuai dengan instrumen/channel yang aktif;

5. Untuk mengatur volume di program musik, pengaturannya langsung dengan komputer/laptop. Anda juga dapat menghubungkan / assign salah satu kenop yang ada di Keyboard Controller untuk menjalankan fungsi pengaturan Volume. Namun umumnya penghubungan / assignment ini bersifat temporer. Artinya harus anda setting ulang tiap kali menjalankan program, tidak otomatis hardware dan software ini terhubung / terkoneksi.

28. jelaskan cara mematikan komputer melalui keyboard


ADA 4 CARA UNTUK MEMATIKAN KOMPUTER DENGAN KEYBOARD YAITU:

1) Pada sistem operasi Windows XP; dengan menekan kombinasi tombol Alt + F4 + U, atau menekan kombinasi tombol Windows + U + U.

2) Untuk sistem operasi Windows 7, kombinasi tombol keyboard yang biasa digunakan untuk mematikan komputer dengan cara dengan menekan tombol Windows + tombol panah kanan + Enter. Di samping itu juga dapat menekan kombinasi tombol Alt + F4.

3) Langkah-langkah mematikan komputer dengan keyboard di Windows 8 dan 10 diawali pada mode deskop, tekanlah ALT + F4. Nantinya akan muncul kotak dialog dengan tulisan "What do you want the computer to do?", jika option pada pilihan Shut Down, langsung saja tekan Enter. Jika moda option pada pilihan sleep atau restart, maka pilih terlebih dahulu ke mode Shut Down baru tekan tombol Enter.

4) Cara mematikan komputer dengan Linux (ubuntu)

Untuk mematikan Linux dengan menggunakan shortcurt keyboard dengan langkah menekan tombol alt + SysRq secara bersamaan dan tahan kemudian tekan tombol R + E + I + S + U + O satu persatu. Kemudian Linux akan langsung melakukan proses shutdown secara otomatis.

30. Huruf A – Z, tanda baca, spasi, caps lock adalah bagian dari keyboard yang disebut. *Tombol angkaTombol fungsiKunci primerKetikkan kunci tulis​


Huruf A – Z, tanda baca, spasi, caps lock ialah bagian dari keyboard yang dinamakan Kunci primer. Kunci primer ialah kunci utama pada keyboard.

Pembahasan

Keyboard ialah sebuah perangkat keras pada komputer yang berfungsi sebagai alat untuk memasukkan data berupa huruf, angka, dan simbol. Fungsionalitas keyboard laptop sangat penting karena perintah tekstual dan kontrol lainnya dimasukkan ke dalam sistem komputer melalui keyboard. Simbol ditampilkan di layar komputer, dicetak pada dokumen, atau sengaja dibuat secara digital di komputer. Keyboard ini merupakan alat yang penting dalam dilakukannya penggunaan komputer.
