12. Sebutkan langkah langkah mengaktifkan microsoft word 2010


Jawaban:

Misalkan memakai sistem operasi Windows XP:

1. Langkah pertama, klik Start

2. Langkah kedua, klik All Programs

3. Langkah ketiga, klik Microsoft Office

4. Langkah keempat, klik Microsoft Office Word 2010

Setelah langkah keempat dilakukan, maka akan tampil program Ms. Word 2010 yang menandakan bahwa program itu telah aktif.

14. bagaimana cara mengubah font pada Microsoft word 2010​


Cara Mengganti Jenis Huruf/Font Default Di Microsoft Word

1. Buka menu ribbon home.

2. Pada grup styles, cari style Normal lalu klik kanan pada style tersebut.

3. Pada popup menu yang muncul, klik item modify.

4. Selanjutnya pada dialog Modify Style ganti jenis huruf (font) serta ukurannya.

5. Kemudian pada bagian bawah dialog, pilih opsi New documents based on this template.

Semoga Membantu
