16. Sebutkan 4 cara mengaktifkan dan menutup microsoft excel ?


aktifkan :
1. klik 2x dr shortcut di desktop
2. klik dr start > all program > ms. office > ms. excel
3. start > run (atau pencet windows+r), lalu ketik excel, enter
4. dr cmd prompt, ketik "start excel". tanpa petik
5. klik 2x langsung file excel nya

tutup :
1. klik tanda x pada pojok kanan atas
2. dr tab file > close
3. pencet alt+f4
4. pencet ctrl+w
