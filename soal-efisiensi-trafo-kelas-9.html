jwb

kumparan primer ( No ) = 500 lilitan

kumparan sekunder ( Ns ) = 220 Volt

ditanya : tegangan sekunder ( Vs )

rumus transformator

V s / Ns = Vp / N p

1. Sebuah transformator mempunyai kumparan primer dan sekunder dengan jumlah lilitan masing-masing 500 dan 5000, dihubungkan dengan jaringan bertegangan arus bolak-balik 220 V. Berapakah tegangan keluarannya?  

Pembahasan

Diketahui :

Kumparan primer (NP) = 500 lilitan  

Kumparan sekunder (Ns) = 5000 lilitan  

Tegangan primer (VP) = 220 Volt

Ditanya : Tegangan sekunder (VS)

Jawab  :

Rumus transformator :

Vs / Ns = Vp / Np

Vs / 5000 = 220 / 500

Vs / 5000 = 0,44

Vs = (0,44)(5000)  

Vs = 2.200 Volt

24. Efisiensi sebuah trafo 60. Energi listrik yang dikeluarkan sebuah trafo adalah 300J, berapakah energi listrik yang masuk trafo, jika efisiensinya 60%​


"Efesiensi Sebuah Travo 60%. Energi Listrik yang di keluarkan 300 J. berapa energi listrik yang masuk Travo"


Penjelasan Dengan Langkah-langkah :

Diketahui :

μ  = Efesiensi = 60%

WS =  Energi Sekunder = 300 J


Ditanyakan :

WP = Energi Primer = ...?


Penyelesaian :

μ  = WS/WP x 100%

60% = 300/WP x 100%

60WP = 300 x 100

   WP =  30.000/60

   WP =  500 J


Kesimpulan :Jadi Energi yang masuk Travo adalah 500 J

30. Efisiensi sebuah trafo 60. Energi listrik yang dikeluarkan sebuah trafo adalah 300J, berapakah energi listrik yang masuk trafo, jika efisiensinya 50%​


Transformator ( Trafo )

alat untuk mengubah tegangan arus bolak balik menjadi lebih rendah atau lebih tinggi.  

Effisiensi trafo :

η = ( Ps / Pp ) x 100%

Ps  =  Vs . Is

Pp  =  Vp . Ip

Ws =  Ps . t

Wp = Pp . t


dimana :

Ps  =  daya sekunder ( watt )

Pp  =  daya primer ( watt )

Ws =  Energi listrik keluar ( joule )

Wp = Energi listrik masuk ( joule )

________________________________________________________

diketahui :

η = 60%

Ws = 300 J

ditanya

Wp

jawab :

W = P x t

P =  W / t


              η  = ( Ps / Pp ) x 100%

                  = ( Ws/t / Wp/t ) x 100%    ( t keduanya di coret )

         60%  = ( 300 / Wp ) x 100%

60% / 100% = 300 / Wp

           0,6  = 300 / Wp

           Wp  = 300/0,6

                  = 500 J

jadi energi listrik yang masuk pada trafo sebesar 500 Joule
