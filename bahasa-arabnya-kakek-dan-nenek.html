3. Kakek = جَدٌّ (mufrad) = أَجْدّادٌ (jama')

Kakek = jaddun = ajdaadun

4. Nenek = جَدَّةٌ (mufrad) = جَدَّاتٌ (jamak)

Nenek = jaddatun = jaddaatun

5. Paman (dari ayah) = عَمٌّ (tunggal) = أَعْمَامٌ (jamak)

Paman (dari ayah) = 'ammun = a'maamun

6, Bibi/Tante (dari ayah) = عَمَّةٌ (mufrad) = عَمَّاتٌ (jamak)

Bibi/Tante (dari ayah) = 'ammatun = 'ammaatun

7. Paman (dari ibu) = خَالٌ (mufrad) = أَخْوَالٌ (jama')

Paman/Om (dari ibu) = khaalun = akhwaalun

8. Bibi/Tante (dari ibu) = خَالَةٌ (mufrad) = خَالاَتٌ (jama')

Bibi/tante (dari ibu) = khaalatun = khaalaatun

9. Saudara laki-laki = أَخٌ (mufrad) = إِخْوَانٌ / إِخْوَةٌ (jamak)

Saudara laki-laki = akhun = ikhwatun/ikhwaanun

10. Saudara perempuan = أُخْتٌ (mufrad) = أَخْوَاتٌ (jamak)

Saudara perempuan = ukhtun = akhwaatun

11. Anak laki-laki = اِبْنٌ (mufrad) = أَبْنَاءٌ (jamak)

Anak laki-laki = ibnun = abnaa-un

12. Anak perempuan = بِنْتٌ (mufrad) = بَنَاتٌ (jamak)

Anak perempuan = bintun = banaatun

13. Cucu laki-laki = سِبْطٌ (mufrad) = أَسْبَاطٌ (jamak).

Cucu laki-laki = حَفِيْدٌ (mufrad) = حَفَدَةٌ (jamak)

Cucu laki-laki = sibthun/hafiidun = asbaathun/hafadatun

30. bahasa jawa dari kakek dan nenek


Bahasa Jawa dari :

Kakek: Simbah Kakung, dan Kaki.Nenek: Simbah Putri, dan Nini.

～ Penjelasan ～

Kakek adalah Ayah dari sang Ibu/Ayah. Nenek adalah Ibu dari sang Ayah/Ibu. Sebagai contoh: Seseorang bernama “Lushi” mempunyai anak bernama “Kato” dan si “Kato” memiliki anak bernama “Chika”, jadi si “Chika” memanggil “Lushi” dengan sebutan Nenek.

Untuk bahasa Jawa dari Kakek yaitu simbah kakung atau bisa disingkat menjadi Mbah Kakung, Begitu juga bahasa Jawa dari Nenek yaitu Simbah Putri bisa disingkat menjadi Mbah uti. Untuk mengajak berbincang dengan Simbah Anda harus berbahasa Krama.

～ Silsilah Keluarga Jawa ～

Kaki Uyut : Kakek Buyut.Nini Uyut : Nenek Buyut.Kaki, Simbah Kakung : Kakek.Nini, Simbah Putri : Nenek.Bapak : Ayah.Mamak : Ibu.Kakang, Mas : Kakak (lk).Yayu/Mbakayu, Mba : Kakak (pr).Adhi Lanang : Adik (lk).Adhi Wadhon : Adik (pr).Lilik : Om atau Tante.Dhulur : Kerabat.Batir, Kanca : Teman.
