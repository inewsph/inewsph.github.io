27. A box measuring by 64 cm by 16 cm by 8 cm is packed with cubes. The length of each cube is 4 cm. how many cubes are there in the box?Pls Tolong ;-;​


Jawab:
There should be 128 cubes
to fill the box with the
aforementioned dimension

Penjelasan dengan langkah-langkah:

Given that a box (cuboid)
which has the:
--> Length (L) = (of) = 64 cm
--> Width (W) = (of) = 16 cm
--> Height (H) = (of) = 8 cm
Is filled with cubes,
each with the:
--> Sidelength (S) = (of) = 4 cm

Find the number of cubes
which can fill the box (n)

Number =
cuboid volume ÷ cube volume =
n = (L×W×H) ÷ (S³)
n = 64×16×8 ÷ 4³
n = 8192 ÷ 64
n = 128 cubes

Conclusion: There should
be 128 cubes to fill the box
with the aforementioned
dimension <(7o7)>
