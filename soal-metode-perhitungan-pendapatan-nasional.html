2. buatlah soal tentang metode perhitungan pendapatan nasional dengan cara pendapatan produksi lengkap dengan rumus​


Soal :

Diketahui jumlah harga barang dan jumlah yang diproduksi pada negaraXYZ pada tahun 2017 sebagai berikut :

a. Sandal harga Rp. 145.000 jumlah barang 3.000

b. Tas harga Rp. 450.000 jumlah barang 6.500

c. Kemeja batik Rp. 230.000 jumlah barang 150.000

d. Sepatu harga Rp. 275.000 jumlah barang 700

Berapa besar pendapatan nasional berdasarkan pendekatan produksi!

Jawaban :

Pendekatan Produksi

Y = ( P1 × Q1 ) + ( P2 × Q2 ) + .... ( Pn × Qn )

Y = ( 145.000 × 3.000 ) + ( 450.000 × 6.500 ) + ( 230.000 × 150.000 ) + ( 275.000 × 700 )

Y = Rp38.052.500.000,00

Pembahasan :

Metode Perhitungan Pendapatan Nasional :

1. Pendekatan Produksi

Rumus :

Y = ( Q1 × P1 ) + ( Q2 × P2 ) + .... ( Qn × Pn )

2. Pendekatan Pengeluaran

Rumus :

Y = C + I + G + ( X - M )

Ket :

Y = pendapatan nasionalC = konsumsi rumah tanggaI = investasiG = pengeluaran pemerintahX = eksporM = impor

3. Pendekatan Pendapatan

Rumus :

Y = w + r + i + p

Ket :

w = upah tenaga kerjar = sewa tanahi = bunga modalp = laba pengusaha

Rumus umum mencari pendapatan nasional (Y) dengan metode pendekatan yaitu :

Y = C + I + G + (X - M)

dimana

Y = pendapatan nasional.

C = Consumption, pengeluaran dalam konsumsi rumah tangga.

I = Investment, pengeluaran investasi dalam rumah tangga.

G = Government Expenditure, pengeluaran yang dilakukan pemerintah.

X = Eksport, pengeluaran yang dilakukan ke luar negeri (eksport) dan dalam luar negeri (import)

I = Import

Perhitungan pendapatan nasionalnya adalah (dalam juta):

Y = C + I + G + (X - M)

= 80 + 75 + 250 + (50 – 35)

= 405 + 15 = 420

Jadi besarnya pendapatan nasional dengan metode pengeluaran adalah Rp 420.000.000.

200P = 2.000Q + 1.200.000
