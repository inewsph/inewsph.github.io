11. Segitiga PQR mempunyai keliling 70 cm. Jika panjang sisi QR = 25 cm dan PR = 17 cm, luas segitiga tersebut adalah mas dayat


Kelas : X

Pelajaran : Matematika

Kategori : Dalil-dalil Segitiga

Kata Kunci : segitiga, keliling, luas

Diketahui

Keliling ΔDEF = 70 cm

Panjang sisi EF = 25 cm

Panjang sisi DF = 17 cm

Ditanya

Luas ΔDEF

Penyelesaian

Step-1

Hitung panjang sisi DE

DE + EF + DF = 70

DE = 70 - 25 - 17

∴ DE = 28 cm

Step-2

Hitung luas ΔDEF

Siapkan setengah keliling s = ¹/₂ x 70

Nilai s = 35 cm

Misalkan

a = DE = 28 cm

b = EF = 25 cm

c = DF = 17 cm

⇔ Luas ΔDEF = √ [s.(s - a)(s - b)(s - c)]

⇔ Luas ΔDEF = √ [35.(35 - 28)(35 - 25)(35 - 17)]

⇔ Luas ΔDEF = √ [(35)(7)(10)(18)]

⇔ Luas ΔDEF = √ [5 x 7 x 7 x 5 x 2 x 9 x 2]

⇔ Luas ΔDEF = √ [(25)(49)(4)(9)

⇔ Luas ΔDEF = 5 x 7 x 2 x 3

∴ Luas ΔDEF = 210 cm²

29. pak dayat meminjam uang di bank yang memberlakukan bunga pinjaman 20% per tahun untuk melunasi pinjaman tersebut setiap bulan selama 1 tahun pak dayat mengangsur sebesar rp600.000 besar pinjaman pak dayat adalah​


Pak dayat meminjam uang di bank yang memberlakukan bunga pinjaman 20% per tahun. Untuk melunasi pinjaman tersebut, setiap bulan selama setahun pak dayat harus mengangsur sebesar Rp600.000,00. Besar pinjaman Pak dayat adalah Rp6.000.000,00 Selengkapnya dapat disimak pada pembahasan di bawah ini!

PENDAHULUAN

Aritmetika sosial adalah materi yang dapat diaplikasikan dalam kehidupan sehari-hari. Misalnya, menentukan untung atau rugi berikut persentasenya, menentukan tabungan awal atau tabungan akhir, mencari besar bunga jika diketahui suku bunganya, dan lain sebagainya.

Untuk menyelesaikan soal ini, kita akan menggunakan rumus berikut :

dimana :

○ A = angsuran

○ B = bunga

○ b = lama meminjam (bulan)

○ i = suku bunga

○ M = pinjaman semula

○ Mt = pinjaman akhir

Kembali ke soal, mari simak penyelesaiannya pada pembahasan di bawah ini!

PEMBAHASAN

Diketahui :

b = 1 tahun → 12 bulan

i = 20% per tahun

A = Rp600.000,00

Ditanya : pinjaman semula (M) = . . . ?

Jawab :

❖ Menentukan pinjaman akhir (MTA)

diperoleh: pinjaman akhir (Mt) = Rp7.200.000,00

❖ Menentukan bunga (B)

diperoleh: bunga (B) = ⅕M

❖ Sehingga, pinjaman semula Pak Abid

∴ Kesimpulan : Jadi, Besar pinjaman Pak dayat adalah Rp6.000.000,00.

30. Besarnya kalor yang diperlukan untuk menguraikan 4,9 l hcl pada suhu 25°c dan tekanan 1 atm adalah mas dayat


Jawaban:

kalor yang diperlukan sebesar 9,23 kJ

Penjelasan:

Diketahui:

Persamaan termokimia:

2HCl (g) → H₂ (g) + Cl₂ (g)   ΔH = +92,3 kJ

V HCl = 4,9 L

T = 25°C = 298 K

P = 1 atm

Ditanyakan:

kalor

Jawab:

P V = n R T

1 x 4,9 = n x 0,082 x 298

4,9 = 24,436n

n HCl = 0,2 mol

ΔH untuk 0,2 mol HCl = 0,2/2 x +92,3 kJ

ΔH untuk 0,2 mol HCl = +9,23 kJ

kalor yang diperlukan = 9,23 kJ
