4. Sebutkan diagonal sisi, diagonal ruang, dan bidang diagonal dari prisma segi-3, prisma segi-4, prisma segi-5, prisma segi-6 ... Fast!! PR nya dikumpulin besok!


Prisma Segi Empat
Diagonal sisi = n(n-1)
                   = 4(4 - 1)
                   = 4(3)
                   = 12
Diagonal ruang = n(n-3)
                      = 4(4-3)
                      = 4(1)
                      = 4
Bidang diagonal = (n(n -3))/2
                        = (4(4-3))/2
                        = (4(1))/2
                        = 2
Prisma segi lima
Diagonal sisi = n(n - 1)
                   = 5(5 - 1)
                   = 5(4)
                   = 20
Diagonal ruang = n(n-3)
                      = 5(5-3)
                      = 5(2)
                      = 10
Bidang diagonal = (n(n-3))/2
                        = (5(5-3))/2
                        = (5(2))/2
                        = 5
Prisma segi enam
Diagonal sisi = n(n-1)
                    = 6(6-1)
                    = 6(5)
                    = 30
Diagonal ruang = n(n-3)
                       = 6(6-3)
                       = 6(3)
                       = 18
Bidang diagonal = (n(n-3))/2
                        = (6(6-3)/2
                        = (6(3)/2
                        = 9

Bangun prisma adalah bangun ruang sisi datar yang memiliki alas dan tutup yang sama bentuknya.

Banyak sisi pada prisma segi-n dapat dirumuskan sebagai n + 2

Banyak rusuk pada prisma segi-n dapat dirumuskan sebagai 3n

Banyak titik sudut pada prisma segi-n dapat dirimuskan sebagai 2n

19. terdapat bangun prisma segi 5 dengan luas alas 60cm² jika tinggi prisma adalah 8cm hitunglah volume prisma segi 5 tersebut adalah? ​


Hi Semoga Jawaban Kakak Membantu Ya!

Sesuai dengan namanya, prisma segi lima memiliki ciri khusus berupa alas dan tutup yang berbentuk segilima.

Banyak rusuk prisma segi lima yakni 3 kali sisi tutup atau alas, sementara banyak titik sudutnya yakni 2 kali titik sudut tutup atau alas.

Penjelasan dengan langkah-langkah:

Untuk menghitung volume prisma segi lima, kita perlu mengalikan luas alas dengan tinggi prisma. Rumusnya adalah sebagai berikut:

Volume = Luas Alas x Tinggi Prisma

Dari soal, diketahui luas alas prisma segi lima adalah 60 cm² dan tinggi prisma adalah 8 cm. Maka, kita dapat menghitung volumenya sebagai berikut:

Volume = Luas Alas x Tinggi Prisma

Volume = 60 cm² x 8 cm

Volume = 480 cm³

Jadi, volume prisma segi lima tersebut adalah 480 cm³.
