30. Kll 88 berapa luasnya lingkaran


Diketahui Keliling lingkaran = 88 satuan

Ditanyakan Luas Lingkaran = ...?

Pembahasan :

Keliling Lingkaran = 2πr

   Luas Lingkaran = πr²

------------------------------------------------------------------------------------------------------

Cari jari-jari lingkaran :

  Keliling = 88

         2πr = 88

              r = 88 : 2π

              r = 44 : π

              r = 44 : 22/7

              r = 2 : 1/7

              r = 2 x 7

              r = 14 satuan panjang

Maka, Luas Lingkaran tersebut :

 Luas = πr²

          = [tex]\frac{22}{7}[/tex] x 14²

          = 22 x 2 x 14

          = 616 satuan Luas
