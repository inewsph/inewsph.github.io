14. contoh soal cara mencari modal akhir


Jika rugi, modal akhir = modal awal - rugi - prive (jika ada prive/pengambilan untuk pribadi)
Jika laba, modal akhir = modal awal + laba - prive (jika ada prive)
Jika mendapat Laba Bersih :
Modal Akhir
= Modal Awal + (Laba - Prive)

Jika mendapat Rugi Bersih :
Modal Akhir
= Modal Awal - (Rugi - Prive)

---
Prive = Pengambilan uang dari perusahaan untuk pribadi

Bunga majemuk adalah bunga yang diberikan berdasarkan modal awal dan akumulasi bunga pada periode sebelumnya

Rumus Bunga Majemuk

Besar bunga = Mn - M₀

Mn = Modal akhir

M₀ = Modal awal

i = suku bunga

n = lama menabung

Pembahasan

Diketahui :

Besar modal (M₀) = Rp 4.000.000,-

suku bunga majemuk pertahun (i) = 15 % = 0,06

Lama (n) = 3 tahun

Ditanya :

Besar modal keseluruhan?

Jawab :

Menentukan besar modal setelah 3 tahun

Mn = M₀ (1 + i)ⁿ

M₃ = 4.000.000 (1 + 0,06)³

= 4.000.000 × (1,06)³

= 4.000.000 × 1,191016

= 4.764.064

Jadi besar modal itu setelah 3 tahun adalah Rp 4.764.064
