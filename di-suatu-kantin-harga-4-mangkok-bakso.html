Di suatu kantin harga 4 mangkok bakso dan 6 gelas es tidak lebih dari Rp.35.000.00, sedangkan harga 8 mangkok bakso dan 4 gelas es tidak lebih dari Rp.50.000.00 . Jika memesan 5 mangkok bakso dan 3  gelas es, maka biaya maksimum yg harus dibayar adalah ... rupiah.

Program linear adalah salah satu bagian dari matematika terapan yang digunakan untuk memecahkan masalah pengoptimalan (memaksimalkan atau meminimalkan suatu tujuan).

Pembahasan

Diketahui :

4 mangkok bakso dan 6 gelas es,  membayar tidak lebih dari Rp. 35.000,00

8 mangkok bakso dan 4 gelas es, membayar tidak lebih dari Rp. 50.000,00

Ditanya :

5 mangkok bakso dan 3 gelas es , maka maksimum yang harus mereka bayar ?

Jawab :

Misalkan :  x = banyak mangkok bakso

                 y = banyak gelas es

Model matematika dari pernyataan diatas

4x + 6y ≤ 35.000  ... (1)

8x + 4y ≤ 50.000  ... (2)

Banyak mangkok bakso dan gelas es tidak mungkin negatif sehingga    

x ≥ 0  dan  y ≥ 0

Menentukan titik potong sumbu x dan sumbu y

Titik potong pertidaksamaan garis 4x + 6y ≤ 35.000

4x + 6y = 35.000

x = 0 → 0 + 6y = 35.000  

                   y = [tex]\frac{35.000}{6}[/tex]

                   y = [tex]\frac{17.500}{3}[/tex]

        titik (0 , [tex]\frac{17.500}{3}[/tex])

y = 0 → 4x + 0 = 35.000  

                   x = [tex]\frac{35.000}{4}[/tex]

                   x = 8750

       titik (8750 , 0)

Titik potong pertidaksamaan garis 8x + 4y ≤ 50.000  

8x + 4y = 50.000  

x = 0 → 0 + 4y = 50.000

                   y = [tex]\frac{50.000}{4}[/tex]

                   y = 12.500  

      titik (0 , 12500)  

y = 0 → 8x + 0 = 50.000  

                   x = [tex]\frac{50.000}{8}[/tex]

                   x = 6250

       titik (6250 , 0)

Apabila koefisiennya positif dan mempunyai tanda ≤, maka daerah arsirannya kebawah.

Menentukan nilai x dan y

Eliminasikan pers I dan II

4x + 6y = 35.000   |×2|    8x + 12y = 70.000

8x + 4y = 50.000   |×3|  24x + 12y = 150.000

                                      ------------------------------- --

                                       -16x         = -80.000

                                                   x = [tex]\frac{-80.000}{-16}[/tex]

                                                   x = 5000

Subtitusikan x = 5000 ke dalam pers I

4x + 6y = 35.000  

4 (5000) + 6y = 35.000

20.000 + 6y = 35.000

                6y = 35.000 - 20.000

                6y = 15.000

                  y = [tex]\frac{15.000}{6}[/tex]

                  y = 2500

titik potong dari kedua pertidaksamaan adalah (5000, 2500)

Untuk gambar grafik dan daerah arsiran terdapat pada lampiran

Menentukan Nilai maksimum fungsi objektif f(x,y) = 5x + 3y

Uji titik pojok

A (6250 , 0)  → 5 (6250) + 3 (0) = 31250 + 0 = 31.250

B (5000, 2500) → 5 (5000) + 3 (2500) = 25000 + 7500 = 32.500   (maks)

C (0, 17500/3)  → 5 (0) + 3 (17500/3) = 0 + 17.500 = 17.500

Jadi pesanan 5 mangkok bakso dan 3 gelas es maksimum yang harus mereka bayar adalah Rp 32.500.

9. harga 5 mangkok bakso dan 2 mangkok es campur adalah Rp.37.500.00. harga 7 mangkok dan 4 mangkok es campur adalah Rp.55.500.00. tentukan harga 1 mangkok bakso dan 1 mangkok es campur​


Jawab:

Harga 1 mangkok bakso adalah Rp6.500,00, dan harga 1 mangkok es campur adalah Rp2.500,00.

Penjelasan dengan langkah-langkah:

"Harga 5 mangkok bakso dan 2 mangkok es campur adalah Rp.37.500.00" dimodelkan oleh persamaan: 5x + 2y = 37.500 ...(1)"Harga 7 mangkok dan 4 mangkok es campur adalah Rp.55.500.00" dimodelkan oleh persamaan: 7x + 4y = 55.500 ...(2)

Mengurangi persamaan;

7x + 4y = 55.500

5x + 2y = 37.500

------------------------  –

2x + 2y = 18.000

2y = 18.000 - 2x ...(3)

y = 9.000 - x ...(4)

Substitusi nilai 2y dari persamaan (3) ke persamaan (1):

5x + 2y = 37.500

5x + 18.000 - 2x = 37.500

3x = 37.500 - 18.000

3x = 19.500

x = 6.500

Substitusi nilai x ke persamaan (4):

y = 9.000 - 6.500

y = 2.500

Jadi, harga 1 mangkok bakso adalah Rp6.500,00, dan harga 1 mangkok es campur adalah Rp2.500,00.
