Jika kalian memiliki graphic calculator atau aplikasi GeoGebra, kalian boleh menggunakannya. Grafik yang digambar adalah y = f(x) dengan f(x) adalah fungsi kuadrat yang berbentuk f(x) = ax²+bx+c Lakukan langkah-langkah yang sama untuk setiap fungsi kuadrat ini, gunakan kertas berpetak. a. f(x)=x² b. f(x)=x²-2x - 3 c. f(x)=x²-x+2 d. f(x) = 3x² - 6x-9 e. f(x)=2x² - 2x + 4 f. f(x)=2x² - 4x + 2 g. f(x)=x² + 4x h. f(x) = 2x² + 4x + 2 i. f(x)=-3x² - 12z - 15 j. f(x) = 2x² + 1 Hasil kerja kalian akan digunakan untuk eksplorasi.​


Ilustrasi beberapa grafik fungsi kuadrat dapat kamu lihat di lampiran. Perhatikan bahwa semua grafik fungsi kuadrat tersebut memiliki bentuk yang identik, yaitu kurva terbuka berbentuk parabola.

Pembahasan

Fungsi kuadrat adalah fungsi yang derajat (pangkat) tertinggi sukunya adalah 2. ax² + bx + c adalah bentuk umum dari fungsi kuadrat, dengan x = akar kuadrat, a = koefisien x², b = koefisien x, dan c adalah konstanta. Fungsi ini menghasilkan grafik berupa kurva terbuka berupa parabola.

Fungsi kuadrat dapat memiliki akar riil maupun akar imajiner. Akar riil ini bisa berjumlah dua ataupun satu.

Jika akar riilnya dua, maka terdapat dua titik potong sumbu x. Jika akar riilnya satu, maka terdapat satu titik potong sumbu x.Jika akarnya imajiner, maka kurvanya tidak akan tidak akan menyentuh sumbu x.

Kurva terbukanya ini bisa terbuka ke atas maupun ke bawah, tergantung dari positif-negatifnya koefisien a. Jika positif, kurvanya terbuka ke atas seperti grafik dari hampir semua fungsi kuadrat yang diberikan. Namun jika negatif, maka kurvanya terbuka ke bawah (coba lihat grafik f(x) = - 3x² - 12x - 15).

Pembahasan

Halo adik-adik! Balik lagi di Brainly!! Gimana, masih semangat belajar kah? Supaya adik-adik semua bisa lebih paham dan mengerti, kakak ingetin soal ini masuk ke materi tentang fungsi atau persamaan kuadrat. Pertama kakak akan jelaskan tentang definisi dari persamaan kuadrat. Suatu persamaan yang memiliki variabel dimana variabel tersebut mempunyai pangkat yang paling tinggi dua maka persamaan tersebut disebut sebagai fungsi kuadrat. Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Diketahui : Suatu persegi panjang kelilingnya 60 cm

Ditanyakan : ukuran persegi panjang agar mempunyai luas maksimum

Dimisalkan :

p = panjang dari persegi panjang l = lebar dari persegi panjang

jawab :

[tex]keliling = 2(p + l)\\60 = 2(p + l)\\30 = p + l\\30 - p = l[/tex]

Akan dicari fungsi luasnya yaitu  

[tex]Luas = p \times l\\Luas = p \times (30 - p)\\Luas = 30 p - p^2[/tex]

Fungsi [tex]Luas = 30p - p^2[/tex] adalah fungsi kuadrat dengan variabel p, dan dari fungsi kuadrat [tex]Luas = 30p - p^2[/tex] diperoleh nilai a = -1, b= 30 dan c = 0. Kemudian agar didapat Luas yang maksimum maka akan dicari sumbu simetri p nya

[tex]p = \frac{-b}{2a}\\p = \frac{-30}{2(-1)}\\p = \frac{30}{2}\\p = 15[/tex]

Jadi didapat panjang dari persegi panjang adalah 15

Akan dicari lebarnya

[tex]l = 30 - p\\l = 30 - 15\\l = 15[/tex]

Jadi ukurannya adalah panjang = 15 cm dan lebar = 15 cm

Pembahasan

Halo adik-adik! Balik lagi di Brainly!! Gimana, masih semangat belajar kah? Nah untuk pertanyaan di atas itu sedikit masuk ke materi tentang fungsi atau persamaan kuadrat. Yuukk langsung aja kakak jelasin materi kali ini, oke? Oke! Pertama-tama apa itu fungsi kuadrat? Kakak jawab yaaa! Suatu persamaan yang memiliki variabel dimana variabel tersebut mempunyai pangkat yang paling tinggi dua maka persamaan tersebut disebut sebagai fungsi kuadrat. Bentuk umum dari fungsi kuadrat adalah [tex]y = f(x) = ax^2 + bx + c[/tex], dengan keterangan x disebut sebagai variabel, kemudian a, b adalah koefisien dan c adalah suatu konstanta, serta nilai dari a tidak sama dengan 0. Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Nomor 1

Diketahui : Suatu persegi panjang kelilingnya 60 cm.

Diperoleh :  

[tex]2(p + l) = 60\\p + l = 30\\p = 30 - l[/tex]

akan dicari fungsi kuadrat untuk luas persegi panjang

[tex]L = p \times l\\L = (30 - l) \times l\\L = 30l - l^2[/tex]

Dari fungsi kuadrat diatas diperoleh a = -1, b = 30, c = 0, sehingga dapat dicari nilai diskriminannya [tex]D = b^2 - 4ac = 30^2 - 4(-1)(0) = 900[/tex], maka luas maksimumnya adalah  

[tex]L = -\frac{D}{4a}\\L = -\frac{900}{4(-1)}\\L = 225[/tex]

Nomor 2

Diketahui : sebuah segitiga siku-siku jumlah kedua sisi siku-sikunya adalah 50 cm

Dimisalkan :

a = alas segitiga siku-sikub = tinggi segitiga siku-siku

Diperoleh :  

[tex]a + b = 50\\a = 50 - b[/tex]

akan dicari fungsi kuadrat untuk luas segitiga siku-siku

[tex]L = \frac{1}{2} \times a \times b \\L = \frac{1}{2} \times (50 - b) \times b\\L = 25b - \frac{1}{2}b^2[/tex]

Dari fungsi kuadrat diatas diperoleh a = -1/2, b = 25, c = 0, sehingga dapat dicari nilai diskriminannya [tex]D = b^2 - 4ac = 25^2 - 4(-\frac{1}{2})(0) = 625[/tex], maka luas maksimumnya adalah  

[tex]L = -\frac{D}{4a}\\L = -\frac{625}{4\big(-\frac{1}{2}\big)}\\L = 312.5[/tex]

Nomor 3

Diketahui : seorang siswa memotong selembar kain. Kain hasil potongannya berbentuk persegi panjang dengan keliling 80 cm

Diperoleh :  

[tex]2(p + l) = 80\\p + l = 40\\p = 40 - l[/tex]

akan dicari fungsi kuadrat untuk luas persegi panjang

[tex]L = p \times l\\L = (40 - l) \times l\\L = 40l - l^2[/tex]

Dari fungsi kuadrat diatas diperoleh a = -1, b = 40, c = 0, akan dicari lebar maksimumnya agar diperoleh luas maksimum

[tex]l = -\frac{b}{2a}\\l = -\frac{40}{2(-1)}\\l = 20[/tex]

sehingga [tex]p = 40 - l = 40 - 20 = 20[/tex], jadi agar diperoleh luas maksimum ukuran panjangnya 20 cm dan lebarnya 20 cm

Nomor 4

Diketahui :  

Sebuah peluru ditembakkan vertikal ke atas.  Tinggi peluruh (dalam meter) sebagai fungsi waktu t (dalam detik) dirumuskan dengan [tex]h(t) = -4t^2 +40t[/tex]

Dari fungsi kuadrat tersebut diperoleh a = -4, b= 40, c = 0

Akan dicari diskriminannya [tex]D = b^2 - 4ac = 40^2 - 4(-4)(0) = 1600[/tex]

Akan dicari tinggi maksimum yang dapat dicapai peluru

[tex]h(x) = -\frac{D}{4a}\\h(x) = -\frac{1600}{4(-4)}\\h(x) = 100[/tex]

Akan dicari waktu yang diperlukan

[tex]t = -\frac{b}{2a}\\t = -\frac{40}{2(-4)}\\t = 5[/tex]
