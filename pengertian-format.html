6. macam macam format gambar dan pengertiannya


Macam-macam format gambar antara lain:

JPEG (Joint Photographic Experts Group) atau dikenal dengan format gambar yang paling umum digunakan ketika mengambil gambar menggunakan perangkat seluler, kamera digital, ataupun kamera profesional.GIF (Graphic Interchange Format) ialah sebuah format gambar yang dipakai ketika awal kemunculan internet. TIFF (Tagged Image File) ialah sebuah format gambar yang sering kali dipakai untuk industri penerbitan.PNG (Portable Network Graphics) ialah sebuah format gambar yang dirancang untuk menggantikan format GIF yang bisa menyimpan foto atau gambar dengan kualitas bagus.BMP ini ialah merujuk pada format gambar Bitmap yang merepresentasikan citra grafis yang terdiri dari susunan titik atau pixel yang tersimpan di memori komputer. Pembahasan

Pada dasarnya format gambar sendiri biasanya dipilih berdasarkan kegunaannya.

Format gambar yang paling umum di dunia digital sendiri ada 5 macam yakni: JPEG, TIFF, PNG, GIF, dan BMP.

14. Apa pengertian dari range format file?


Row adalah baris horisontal pada lembar kerja excel yang di wakili oleh nomor (1,2,3,4…..) dengan jumlah baris maksimum adalah 1.048.576 row pada excel

Column adalah kolom vertikal pada lembar kerja excel yag diwakili oleh Huruf (A,B,C,D,E,….) dengan jumlah kolom maksimum adalah 16.384 atau kolom (XFD)kolom pada excel

Cell adalah tempat untuk menginput dan menampung data pada excel yang didalamnya terdapat alamat cell. alamat cell merupakan titik temu antara kolom dan baris. Sebagai contoh jika sebuah kursor berada di kolom C dan baris 6, maka alamat cellnya adalah C6. alamat cell ini akan ditampilkan pada name box, lihat gambar dibawah.Pengertian cell

Range adalah gabungan dari beberapa cell atau bisa juga di artikan gabungan dari beberapa kolom atau beberapa baris. Range juga mempunyai alamat yaitu dengan cara membaca alamat cell awal = A1 dengan alamat cell akhir = D4. Dari gambar di bawah ini maka alamat range dapat diartikan Range A1:D4.pengertian range

16. apa pengertian format ip address


 Format IP address yaitu IP address  yg dinyatakan dalam struktur bilangan biner yang terdiri atas 32 bit dengan bentuk sebagai berikut.   xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx  
Contoh 11000000000010100001111000000010 Agar kita mudah membaca IP address, maka 32 bit bilangan itu dibagi ke dalam 4 segmen yang masing-masing berisi 8 bit. Kedelapan bit itu bisa disebut oktat.

22. Jelaskan Pengertian Format di komputer


Komputer adalah alat hardware yang digunakan untuk melakukan pengolahan data, yang sudah dirumuskan.

Pembahasan

Format di komputer adalah struktur dari sebuah file, dan bagaimana proses file tersebut terbentuk, dan juga penamaan sesuai jenis-jenisnya.

Macam-macam format file didalam komputer:

EOF

Format data yang ada akhrinya eof, biasanya datanya berisi tentang header, metadata, kontent dan lainya.

JPEG

format file yang biasanya isi berupa file foto, gambar dan lainya.

XML

Biasanya formal file ini berisikan tentang penyimpanan data list item

Txt

Text, biasa berupa catatan tulis, seperti dari proram notepad atau lainya

MP4, MKV

adalah suatu file yang isinya suatu video, namun video ada jenis-jenis sendiri tergantung kualitasnya. Seperti MP4 dengan MKV, kualitas gambarnya lebih bagus MKV, ataupun HD dan lainya.
