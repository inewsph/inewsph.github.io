3. Pak Ali bekerja di PT. Angkasa. Setiap bulan ia memperoleh gaji sebesar 3,5 jt rupiah. Penghasilan tidak kena pajak. Rp. 1.160.000,00. Jika pajak penghasilan (ppn) 10% berapa besar gaji bersih yang diterima pak Ali?


dik :
besar gaji = 3.500.000
penghasiln non pajak = 1.160.000
pph = 10%

mka :
Penghasilan kena pajak = 3.500.000 - 1.160.000 = Rp 2.340.000
Besar pph = 10% x 2340.000 = Rp 234.000

Jadi besar gaji = 1.160.000 + (2.340.000 - 234.000)
                      = 1.160.000 + 2.106.000
                      = Rp 3.266.000

17. Gapura kiri = gapura kanan panjang gapura kanan 3500 cm jarak antara gapura kiri dan kanan 400 cm Berapa panjang gapura ketiga


Gapura kiri = gapura kanan panjang gapura kanan 3500 cm jarak antara gapura kiri dan kanan 400 cm. Panjang gapura ketiganya adalah 7.400 cm

Soal ini membahas tentang pengukuran panjang / jarak

dimana jarak didefinisikan angka yang menunjukkan berapa jauh posisi antara objek yang satu dengan objek yang lainnya

Satuan untuk jarak / panjang dalam Sistem Internasional (SI) adalah meter

Pembahasan :

Diketahui :

Gapura kiri = gapura kanan

Panjang gapura kanan = 3500 cm

Jarak antara gapura kiri dan kanan 400 cm

Ditanya :

Panjang gapura ketiganya ?

Dijawab :

Perhatikan gambar terlampir, kira-kira ilustrasi seperti itu

Karena panjang garura kiri = gapura kanan maka

gapura kanan = 3.500 cm

gapura kiri = 3.500 cm

jarak antara gapura kiri dan kanan = 400 cm

Maka jarak seluruhnya = gapura kiri + gapura kanan + jarak antara gapura kiri dan kanan

= 3.500 + 3.500 + 400

= 7.400 cm

Jadi panjang ketiga gapura adalah 7.400 cm

maka bruto gaji Pak Andre:

penghasilan bruto = 1.8 milyar + 54 juta = 1.854 milyar / tahun

tahap selanjutnya dgn menghitung penghasilan neto setelah pajak (PNSP) dgn cara penghasilan bruto dikurangi Pajak Penghasilan (PPh) yg terutang. utk menghitung PPh terutang perlu diketahui berapa besar PTKP (Penghasilan Tidak Kena Pajak) dan tarif pajak yg berlaku.

pd tahun 2021 PTKP untuk orang yg sudah menikah dan memiliki satu anak sebesar 54 juta rupiah. tarif pajak penghasilan yg berlaku untuk PKP 50 juta - 250 juta sebesar 15%.

maka PPh yg terutang dapat dihitung menggunakan rumus:

PPh = (penghasilan bruto - PTKP) x tarif pajak

= (1.854 milyar - 54 juta) x 15%

= Rp273.900.000

jdi PPh yg terutang selama 1 tahun bagi Pak Andre sebesar Rp273.900.000
