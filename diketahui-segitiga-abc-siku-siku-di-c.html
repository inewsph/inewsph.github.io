2. Diketahui segitiga ABC,siku-siku di C dan besar


Soal:

Diketahui segitiga ABC siku-siku di C dan besar AC = (x - 7) cm, BC = (x - 14) cm, dan AB = x cm. Panjang sisi AC adalah ...

Jawab:

AB² = (BC² + AC²) 
x² = (x-14)² + (x-7)² 
x² = x² - 28x + 196 + x² - 14x + 49 
x² = 2x² - 42x + 245 
2x² - x² - 42x + 245 = 0 
x² - 42x + 245 = 0 
(x - 35) (x - 7) = 0 
x - 35 = 0  atau  x - 7 = 0 
x₁ = 35 atau x₂ = 7 

AC = x - 7 
Untuk x = 35, maka panjang sisi AC = 35 - 7 = 28 (memenuhi) 
Untuk x = 7, maka panjang sisi AC = 7 - 7=0 (tidak memenuhi) 

Jadi panjang AC = 28 cm

8. diketahui segitiga abc dengan titik a(-1,5).b(-1,1).dan c(2,1).apakah segitiga abc merupakan segitiga siku siku ?jelaskan


itu ya, semoga membantu.

Pembahasan :

segitiga dengan titik A(-1,5), titik B(-1,1) dan titik C(2,1) apakah merupakan segitiga siku-siku?

kita cari panjang masing-masing sisi
panjang AB = √{(5-1)²+(-1-(-1))²}
= √(4²+0²)
= √16
= 4 satuan
panjang BC = √{(2-(-1))² + (1-1)²}
= √(3² + 0²)
= √9
= 3 satuan
panjang dari AC = √{(5-1)² + (-1-2)²}
= √(4²+(-3)²)
= √(16+9)
= √25
= 5 satuan

setelah tau panjang masing-masing sisi, sekarang kita cek apakah segitiga itu siku-siku?
jika AB² + BC² = AC², maka segitiga itu siku-siku
4² + 3² = 5²
16 + 9 = 25 → terbukti segitiga ABC siku-siku

jawabanya iya segitiga ABC adalah segitiga siku-siku
karena terbukti AB² + BC² = AC²

misal, sudut B adalah x, maka :

sudut b = 180⁰ - ( 90⁰ + 35⁰)

sudut b = 180⁰ -125⁰

sudut b = 55⁰

28. Diketahui segitiga siku-siku ABC, siku-siku disudut C seperti pada gambar,maka tan A=​

Tan A = BC/BA

Penjelasan dengan langkah-langkah:

Diketahui :

∆ siku-siku ABC, siku-siku di C.

Ditanya :

tan A

Jawab :

tan A adalah perbandingan sisi di hadapan sudut A dengan sisi di samping sudut A (bukan sisi miring segitiga ).

Perbandingannya dapat dituliskan sebagai berikut :

sin A = BC / AB

cos A = AC / AB

tan A = BC / AC

Jadi tan A = BC / AC
