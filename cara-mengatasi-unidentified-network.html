11. Jelaskan cara yang digunakan untuk merepresentasikan sebuah subnet mask dengan menggunakan bit yang mendefinisikan network identifier sebagai sebuah network prefix dengan menggunakan notasi network prefix.!


1. Tentukan dahulu berapa jumlah host dari prefix yang telah ditentukan (contoh /30 = 4 host)
2. kurangi jumlah keseluruhan ip (256 ip) dengan host yang terdapat pada prefix tersebut (256 - 4 = 252)
3. subnet mask dari /30 = 255.255.255.252

15. Cara mengakses folder di network tanpa pin ?


Cara mengakses folder di network tanpa pin iyalah sebagai berikut :
1. Sebelumnya kita harus terkoneksi dalam sebuah jaringan
WiFi/LAN
2. Pastikan terlebih dahulu kita mengaktifkan "Turn on File and Printer Sharing" pada settingan dari control panel, jika ingin berbagi file dalam folder.
3. Cara mengaktifkan settingan sharing tersebut iyalah, terlebih dahulu kita masuk Control panel dengan cara tekan tombol "Win+R" pada keyboard , lalu ketik "Control Panel" atau "Control" pada kolom open, lalu klik "OK"
4. Jika sudah masuk control panel, masuk pada bagian "Network and Internet"
5. Masuk lagi pada bagian "Network and Sharing Center"
6. Lalu ada di bagian kiri tertulis "Changes Advanced Sharing Settings" klik lah pada tulisan tersebut.
7. Pada bagian "Password Protected Sharing" pilih "Turn off password protected sharing"
8. jika sudah, lalu simpan settingan tersebut dengan menekan tombol "Save Changes"

17. 1. Jelaskan cara yang digunakan untuk merepresentasikan sebuah subnet mask dengan menggunakan bit yang mendefinisikan network identifier sebagai sebuah network prefix dengan menggunakan notasi network prefix.!


1. Dalam banyak kasus, sebuah alamat network identifier adalah sama dengan segmen jaringan fisik dengan batasan yang dibuat dan didefinisikan oleh router IP.

2. Memberikan pengalamatan komputer dalam jaringan dengan berdasarkan kelas

3. Dengan pengalamatan IP classful, jaringan yang dapat dibentuk hanya sebatas kapasitas masing-masing kelas.

4. Secara standar dalam jaringan yang menggunakan subnetting classfull, sudah terdapat subnet mask yang menandakan jumlah host yang dapat digunakan masih sama dengan jumlah host maksimal dari masing-masing kelas tersebut.
