Teorama Phytagoras untuk Segitiga Tumpul pada soal ini adalah a² + b² < c²

Jika terpenuhi, maka segitiga tersebut adalah segitiga tumpul.

Maka Langsung saja

Pilihan A ( Segitiga Tumpul )

{ a = 14, b = 16, c = 22 }

\________________/

a² + b² < c²

Pilihan B ( Bukan Segitiga Tumpul )

{ a = 10, b = 41, c = 41 }

\________________/

a² + b² < c²

10² + 40² < 41²

100 + 1.600 < 1.681

1.700 < 1.681 [ × Salah ]

Pilihan C ( Bukan Segitiga Tumpul )

\________________/

a² + b² < c²

26. 2.Ukuran sisi yang merupakan segitiga tumpul.tolong ya kak​


Jika B^2 > A^2+C^2 maka segitiga tsb termasuk segitiga tumpul

Jika B^2 = A^2+C^2 maka segitiga tsb termasuk segitiga siku siku

Jika B^2 < A^2+C^2 maka segitiga tsb termasuk segitiga lancip

Contohnya:

7^2 > 2^2+4^2

49 > 4+16

49 > 20

Ini termasuk segitiga tumpul.

28. ukuran sisi yang membentuk segitiga tumpul ditunjukkan oleh


Kelas         : 8 
Mapel         : Matematika 
Kategori     : Bab 5 - Teorema Pythagoras
Kata kunci : menentukan segitiga tumpul

Kode : 8.2.5 [Kelas 8 Matematika Bab 5 Teorama Pythagoras

Penjelasan : 

Dengan berdasarkan teorama pythagoras kita bisa menentukan jenis segitiga.
Untuk menentukan sebuah segitiga harus memenuhi syarat yaitu →  a + b > c

Gambar jenis segitiga bisa dilihat pada lampiran.

Menentukan jenis segitiga

Dalam Δ ABC, apabila a, b, dan c adalah sisi-sisi di hadapan sudut A, B, dan C, maka berlaku kebalikan teorama Pythagoras, yaitu :

-  Jika a² = b² + c² , maka Δ ABC adalah segitiga siku-siku di ∠ A
-  Jika a² < b² + c² , maka Δ ABC adalah segitiga lancip di ∠ A
   Sisi a terletak dihadapan ∠ A
-  Jika b² < a² + c², maka Δ ABC adalah segitiga lancip di ∠ B
   Sisi b terletak di hadapan ∠ B 
-  Jika a² > b² + c², maka Δ ABC adalah segitiga tumpul di ∠ A
   Sisi a terletak di hadapan ∠ A
---------------------------------------------------

Untuk menentukan jenis segtiga kita kuadrat sisi terpanjang di ruas kiri dan ruas kanan merupakan jumlah kuadrat dua sisi yg lain.

(i).   3 cm , 5 cm dan 6 cm

6² > 3² + 5²
36 > 9 + 25
36 > 34

Jadi jenis segitiganya adalah segitiga tumpul, karena a² > b² + c²

(ii).  5 cm , 12 cm dan 13 cm

13²  = 5² + 12²
169 = 25 + 144
169 = 169

Jadi jenis segitiganya adalah segitiga siku-siku, karena a² = b² + c²

(iii).  8 cm , 12 cm dan 16 cm

16² > 8² + 12²
256 > 64 + 144
256 > 208

Jadi jenis segitiganya adalah segitiga tumpul, karena a² > b² + c²

(iv).  10 cm , 15 cm dan 17 cm

17² < 10² + 15²
289 < 100 + 225
289 < 325

Jadi jenis segitiganya adalah segitiga lancip, karena a² < b² + c²

Jadi ukuran sisi yang membentuk segitiga tumpul ditunjukkan oleh B.  (i) dan (iii)

PEMBAHASAN

Jenis segitiga berdasarkan sudutnya (beserta cara menentukan-nya) , yaitu:

Segitiga lancip, jika c² < a² + b²Segitiga siku-siku, jika c² = a² + b²Segitiga tumpul, jika c² > a² + b²

​

dengan syarat:

c = sisi miring/hipotenusa dengan kata lain sisi terpanjang, otomatis besar sisi miring lebih besar dari pada sisi lainnyaa dan b = sisi tegak

Maka:

1. ) 5, 5, 9

dengan:

a = 5b = 5c = 9

sehingga:

c² = a² + b²

9² = 5² + 5²

81 = 25 + 2

81 = 50

81 > 50 → Segitiga tumpul

__

2. ) 5, 9, 10

dengan:

a = 5 b = 9 c = 10

sehingga:

c² = a² + b²

10² = 5² + 9²

100 = 25 + 81

100 = 106

100 < 106 → Segitiga lancip

__

3. ) 7, 10, 25

dengan:

a = 7b = 10c = 25

sehingga:

c² = a² + b²

25² = 7² + 10²

625 = 49 + 100

625 = 149

625 > 149 → Segitiga tumpul

__

4. ) 8, 15, 17

dengan:

a = 8b = 15c = 17

sehingga:

c² = a² + b²

17² = 8² + 15²

289 = 64 + 225

289 = 289 → Segitiga siku-siku
