PENJELASAN

=> CONDITIONAL SENTENCE TYPE 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan terjadi di masa depan. Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang. Rumus dari kalimat ini adalah: if + simple present, simple future “will”.

=> CONDITIONAL SENTENCE TYPE 2

Second conditional sentences merupakan tipe kalimat pengandaian yang digunakan ketika hasil atau konsekuensi hanya memiliki harapan terwujud walaupun kemungkinannya sangat kecil. Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive. Penggunaan would/could/might berfungsi untuk menjelaskan seberapa besar sebuah kondisi itu akan terjadi.

=> CONDITIONAL SENTENCES TYPE 3

Third conditional sentence merupakan sebuah kalimat pengandaian ketika sebuah kondisi tidak mungkin terwujud sama sekali. Dalam tipe kalimat pengandaian yang ketiga ini, bentuk kalimat menggunaka past perfect yang dilengkapi dengan modal auxiliary seperti would, could, dan should. Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

22. Contoh kalimat conditional sentence type 1 diubah menjadi type 2


PART : -IF CONDITIONAL SENTENCE

-----------------

Jawaban:

TYPE 1

If i don't know the exact way, i will ask people on the street.

Fact : I probably ask people because i'm not sure that i can find the way properly. { Kemungkinan saya tanya orang karena tidak yakin bahwa asaya dapat menemukan jalan dengan tepat )

TYPE 2

If i didn't know the exact way, i would ask people on the street.

Fact : I know the exact way, so i don't need to ask people later . ( Saya tahu jalannya , jadi saya tidak perlu tanya orang saat ini dan ataupun mendatang )

TYPE 3

If i had not known the exact way, i would have asked people on the street.

Fact : I knew the exact way, so i didn't need to ask people yesterday. ( Saya telah mengetahui jalan yang tepat, jadi saya tidak perlu tanya orang kemarin )

Penjelasan:

CONDITIONAL SENTENCE/ ANDAI-ANDAItells about the condition that will be actualized ( possible condition ) or only dreaming ( intention ) because of the particular condition. Consists -If main clause and result clause.Using Comma if the -If clause is started at the beginning sentence.

Commonly, there are 4 types.

1. CONDITIONAL ZERO

is used to describe the  PHENOMENON and GENERAL TRUTH ( THE FACT )

Example :

If it rains, the streets get wet.

2. CONDITIONAL TYPE 1stating the truth  in the present/future ( The planning  will be possible / probably happened )the speaker will actualize his/her dream.using simple present tense and simple future

Pattern :

If +S+ verb (s/es) + O,  S+will+verb1+O/C

Example :

If i have some money, i will buy the English grammar book later.

Fact in the present/future : it's possible that i will buy the English grammar book today because i have enough money.

CONDITIONAL TYPE 2Stating the untruth in the present / futureContrary to the factThe speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/futureusing simple past and past futureKeep in mind, if we use the first and second person  ( I, she, he, it  ) , they must be followed by the linking verb "were." It is the same way to the plural form as the formal writing.

Pattern:

If+S+verb2+O....S+Would+verb1+O/C

Example :

If i were a president, i would build the largest library in the southeast Asia.

Fact in the present/future : I am not a president, i'm a student now .

3. CONDITIONAL TYPE 3Stating  the untruth in the pastContrary to the factusing the past perfect tense and past future perfect

Pattern :

If+S+Had+V3+O....,S+Would+have+V3

Example :

If my money hadn't lost , i would have bought the new cooking recipe.

Fact in the past : i lost my money, so i didn't buy the new cooking recipe yesterday

KESIMPULAN :

Conditional sentence sebagai kalimat andai-andai atau hanya impian jika persyaratan pendukung tidak tercapai.Dalam bahasa inggris terdapat 4 kategori kalimat pengandaian .Conditional Zero membahas tentang fakta/kejadian yang sebenarnya/fenomena alam. Bentuk strukturnya pada umumnya menggunakan simple present.Conditional type 1 membahas tentang prediksi/rencana nyata yang mungkin saja terjwujud karena persyaratannya direalisasikan saat ini atau mendatang. Conditional type 2 membahas tentang kejadian yang kontras ( berkebalikan ) dengan fakta nya yang terjadi saat ini atau mendatang. Dalam istilah ini, si pengucap hanya berandai-andai saja kerena persyaratannya tidak diwujudkan untuk saat ini atau nanti-nanti . Conditional type 3 membahas tentang kejadian yang kontras ( berkebalikan ) dengan faktanya yang terjadi di waktu lampau. Si pengucap mengulas kembali kejadian yang tidak berhasil terwujud di waktu lampau.
