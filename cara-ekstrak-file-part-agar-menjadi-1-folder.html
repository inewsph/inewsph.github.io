3. 1. Cara membuat folder?2. Cara membuat file data (notepad)?3. Cara menyimpan file data?4. Menyalin/memindahkan folder/file (copy/cut)?5. Mengganti nama folder/file(rename)?6. Menghapus nama folder/file (delete)?​


1. Cara membuat folder:

Windows: Klik kanan pada desktop atau di dalam explorer, pilih "New" -> "Folder". Atau tekan "Ctrl + Shift + N".MacOS: Klik kanan pada desktop atau di dalam Finder, pilih "New Folder". Atau tekan "Command + Shift + N".Linux: Buka terminal, ketik "mkdir [nama folder]" lalu tekan enter.

2. Cara membuat file data (notepad):

Windows: Klik kanan pada desktop atau di dalam explorer, pilih "New" -> "Text Document". Ganti nama file dengan ".txt" di akhir. Atau buka program Notepad dan simpan file dengan format .txt.MacOS: Buka program "TextEdit", tulis data dan simpan dengan format .txt.Linux: Buka terminal, ketik "nano [nama file].txt" lalu tekan enter.

3. Cara menyimpan file data:

Windows: Buka file dengan Notepad, ketik data, klik "File" -> "Save" atau "Ctrl + S".MacOS: Buka file dengan TextEdit, ketik data, klik "File" -> "Save" atau "Command + S".Linux: Buka file dengan nano, ketik data, tekan "Ctrl + O" untuk save, "Ctrl + X" untuk keluar.

4. Menyalin/memindahkan folder/file (copy/cut):

Windows: Pilih folder/file, tekan "Ctrl + C" untuk copy dan "Ctrl + X" untuk cut. Tempelkan dengan "Ctrl + V" pada lokasi yang diinginkan.MacOS: Pilih folder/file, tekan "Command + C" untuk copy dan "Command + X" untuk cut. Tempelkan dengan "Command + V" pada lokasi yang diinginkan.Linux: Buka terminal, ketik "cp [sumber] [tujuan]" untuk copy atau "mv [sumber] [tujuan]" untuk cut.

5. Mengganti nama folder/file (rename):

Windows: Klik 2 kali pada nama folder/file, ganti nama dan tekan enter.MacOS: Klik nama folder/file, tekan enter, ganti nama dan tekan enter.Linux: Buka terminal, ketik "mv [nama lama] [nama baru]" lalu tekan enter.

6. Menghapus folder/file (delete):

Windows: Pilih folder/file, tekan "Delete" atau "Shift + Delete".MacOS: Pilih folder/file, tekan "Command + Delete".Linux: Buka terminal, ketik "rm [nama file]" lalu tekan enter.

5. Bagaimanakah cara mengelola file dan folder dengan menggunakan windows explorer jelaskan beberapa fungsi yang di gunakan dalam pengelolaan file dan folder


Berikut ini adalah beberapa cara mengelola file dan folder menggunakan windows explorer:

New folder, berfungsi untuk membuat folder baru.Rename, berfungsi untuk mengganti nama sebuah file atau folder tertentu.Cut, berfungsi untuk memindahkan file maupun folder.Copy, berfungsi untuk menyalin file maupun folder.Paste, berfungsi untuk meletakkan hasil pemindahan atau penyalinan file maupun folder.Delete, berfungsi untuk menghapus file maupun folder.Library, berfungsi untuk menampilkan seluruh folder dan file yang berada dalam drive tertentu.

Pembahasan

Windows explorer merupakan software dari windows yang digunakan sebagai sarana penjelajah file. Software ini berfungsi untuk menyajikan seluruh file maupun folder yang tersimpan dalam sebuah komputer.

Melalui windows explorer, user atau pengguna komputer dapat mengolah, mengelompokkan, hingga menyusun setiap file yang terdapat dalam komputernya. Windows explorer dapat digunakan untuk menggandakan, mengganti nama, mengelompokkan, dan menghapus file.

Folder merupakan suatu wadah yang menjadi tempat beberapa file. Biasanya, folder digunakan untuk mengelompokkan berbagai jenis file berdasarkan kategorisasi tertentu, seperti tanggal, isi, dan sebagainya.

   Klik Folder Baru di kotak dialog Simpan Sebagai.

   Ketikkan nama folder baru Anda, lalu tekan Enter.

   Ganti nama folder baru.

   Catatan: Anda tidak dapat menggunakan garis miring, titik dua, titik koma, garis pisah, atau titik pada nama folder.

   Klik Simpan. Dokumen Anda disimpan ke folder baru.

Buat folder baru sebelum Anda menyimpan dokumen Anda dengan menggunakan File Explorer

   Buka File Explorer menggunakan salah satu cara berikut:

       Tekan tombol logo Windows + E.

       Temukan dari menu Mulai (Windows 7 atau Windows 10).

       Klik ikon folder di taskbar.
