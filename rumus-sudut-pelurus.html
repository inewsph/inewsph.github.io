Resultan Dari Kedua Gaya Tersebut adalah Sebesar 5 N.

_______________________.

_________________.

Resultan dari Dua Vektor Gaya dirumuskan sebagai berikut:

[tex] \bf r = \sqrt{ f_{1} {}^{2} + { f_{2}}^{2} + (2. f_{1}.f_{2}.cos \: \alpha ) } [/tex]

dengan:

R = Resultan Dua Vektor Gaya

F1 = Vektor Gaya 1

F2 = Vektor Gaya 2

a = Sudut yang Dibentuk

_____________.

_____________________.

diketahui:

F1 = 3 N

F2 = 4 N

a = 90° (Tegak Lurus = Siku")

(cos 90° = 0)

--

---

[tex] \bf = \sqrt{ f_{1} {}^{2} + { f_{2} }^{2} + (2. f_{1}. f_{2}.cos \: \alpha ) } \\ \\ \bf = \sqrt{ {3}^{2} + {4}^{2} + (2 \times 3 \times 4 \times cos \: {90}^{o}) } \\ \\ \bf = \sqrt{3(3) + 4(4) + \cancel{(2 \times 3 \times 4 \times 0)} } \\ \\ \bf = \sqrt{9 + 16 }\\ \\ \bf = \sqrt{25} \\ \\ \bf = \sqrt{ {5}^{ \cancel{2}} } \\ \\ \boxed{ \bf r= 5 \: \: n}[/tex]

Maka, Resultan dari dua buah Gaya yang tegak lurus dengan besar masing-masing 3 N dan 4 N adalah sebesar 5 N.

Jawaban:

Untuk mencari pelurus dari suatu sudut, kita dapat menggunakan rumus:

pelurus = 90 - sudut

Sehingga untuk sudut 3,1 derajat, pelurusnya adalah:

pelurus = 90 - 3,1

pelurus = 86,9 derajat

Jadi, pelurus dari sudut 3,1 derajat adalah 86,9 derajat.

Rumus keliling segitiga yaitu:

Keliling = s + s + s atau K = a + b + c

Keliling (Kll) Kll = 2 × (a + b)

Luas (L) L = a × t

5. Rumus yang ada pada Bangun Datar Belah Ketupat

Keliling (Kll) Kll = s + s + s + s

Kll = s × 4

Jawab:

1.  Tuliskan rumus bangun datar 5 saja !

1. Persegi

    Luas = Sisi x Sisi

    Keliling = Sisi x 4

2. Persegi panjang

    Luas = Panjang x Lebar

    Keliling = 2 x ( Panjang x Lebar )  

3. Jajar genjang

    Keliling = 2 × ( Sisi alas + Sisi miring )

    Luas  = Alas x Tinggi

4. Segitiga

    Luas = ½ x a x t

    Keliling = s + s + s

5. Belah ketupat

     Keliling = s + s + s + s

     Luas = ½ × d1 × d2

2. Apa yang dimaksud sudut lancip ?

Sudup lancip adalah sudut yang besarnya < 90° ( Dibawah 90 derajat ).

3. Apa yang dimaksud sudut siku-siku ?

Sudup siku-siku adalah sudut yang besarnya = 90° ( Sama dengan 90 derajat ).

4. Apa yang dimaksud sudut lurus ?
