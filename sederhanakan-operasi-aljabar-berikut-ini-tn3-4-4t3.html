Sederhanakan operasi aljabar berikut ini (tn3)4×4t3!

Hasil penyederhanaannya adalah 4t⁷n¹².

Pembahasan

Penyederhanaan operasi hitung

(tn³)⁴ x 4t³

= t⁴n³ˣ⁴ x 4t³

= 4t⁴⁺³n¹²

= 4t⁷n¹²

Kelas : X (1 SMA)
Materi : Bentuk Pangkat
Kata Kunci : pangkat, positif, negatif

Pembahasan :
Jika a suatu bilangan real dan n suatu bilangan bulat positif, maka
pⁿ = p x p x ... x p
       ____v_____
           n faktor
dengan
n dinamakan eksponen atau pangkat.
p dinamakan bilangan pokok (atau basis atau bilangan dasar).
pⁿ dinamakan bilangan berpangkat.
p x p x ... x p (sampai dengan n suku) dinamakan hasil perpangkatan.

Sifat-sifat bentuk pangkat, antara lain :
1. pᵃ x pᵇ = pᵃ ⁺ ᵇ,
2. [tex] \frac{p^a}{p^b}=p^{a-b} [/tex]
3. (pᵃ)ᵇ = pᵃ ˣ ᵇ,
4. (p x q)ᵃ = pᵃ x qᵃ,
5. [tex] (\frac{p}{q})^a= \frac{p^a}{q^a} [/tex].

Sederhanakan operasi aljabar berikut!
a. q³ x 2q⁷ x (3q)²
b. p x 2q⁷ x p³ x q²
c. 3m³ x (mn)⁴
d. (tn³)⁴ x 4t³
e. (2p³) x p³ x (p²q²)³ x p⁵ x q⁴

Jawab :
Kita ubah variabel x dan b dengan p, serta y dengan q. Supaya tidak keliru antara variabel "x" dengan tanda "kali".

Dengan menggunakan sifat-sifat bentuk pangkat di atas, diperoleh
a. q³ x 2q⁷ x (3q)²
= q³ x 2 x q⁷ x 9 x q²
=  2 x 9 x q³ x q⁷ x q²
= 18 x q³ ⁺ ⁷ ⁺ ²
= 18 x q¹²
= 18q¹²

b. p x 2q⁷ x p³ x q²
= p x p³ x 2 x q⁷ x q²
= 2 x p¹ ⁺ ³ x q⁷ ⁺ ²
= 2 x p⁴ x q⁹
= 2p⁴q⁹

c. 3m³ x (mn)⁴
= 3 x m³ x m⁴ x n⁴
= 3 x m³ ⁺ ⁴ x n⁴
= 3 x m⁷ x n⁴
= 3m⁷n⁴

d. (tn³)⁴ x 4t³
= t⁴ x n¹² x 4 x t³
= 4 x t⁴ x t³ x n¹²
= 4 x t⁴ ⁺ ³ x n¹²
= 4 x t⁷ x n¹²
= 4t⁷n¹²

e. (2p³) x p³ x (p²q²)³ x p⁵ x q⁴
= 2 x p³ x p³ x p⁶ x q⁶ x p⁵ x q⁴
= 2 x p³ ⁺ ³ ⁺ ⁶ ⁺ ⁵ x q⁶ ⁺ ⁴
= 2 x p¹⁷ x q¹⁰
= 2p¹⁷q¹⁰

Sederhanakan operasi aljabar berikut ini.  

a. y³ × 2y⁷ × (3y)² b. b × 2y⁷× b³× y² c. 3m³ × (mn)⁴ d. (tn³)⁴ × 4t³ e. (2x³) × 3(x²y²)³ × 5y⁴

Perpangkatan adalah perkalian berulang dari suatu bilangan yang sama. Bentuk umum dari perpangkatan adalah aⁿ = a × a × a × .... × a, dengan n bilangan bulat positif.

Sifat Perpangkatan

xᵃ × xᵇ = xᵃ⁺ᵇ xᵃ : xᵇ = xᵃ⁻ᵇ (xᵃ)ᵇ = xᵃᵇ (xy)ⁿ = xⁿ . yⁿ

Untuk lebih lengkap sifat perpangkatan bisa dilihat pada lampiran

Pembahasan

Menyederhanakan operasi aljabar

a.   y³ × 2y⁷ × (3y)²

y³ × 2y⁷ × (3y)² = 2 × y³⁺⁷ × 3² × y²

                          = 2 × 9 × y¹⁰ × y²

                          = 18 × y¹⁰⁺²

                          = 18 y¹²

b.   b × 2y⁷× b³× y²

b × 2y⁷ × b³ × y² = 2 × b¹ × b³ × y⁷ × y²

                           = 2 b¹⁺³ y⁷⁺²

                           = 2 b⁴ y⁹

c.   3m³ × (mn)⁴

3m³ × (mn)⁴ = 3 m³ × m⁴ × n⁴

                   = 3 m³⁺⁴ n⁴

                   = 3 m⁷ n⁴

d.   (tn³)⁴ × 4t³

(tn³)⁴ × 4t³ = t⁴  × 4t³

                 = 4 n¹² t⁴⁺³

                 = 4 n¹² t⁷

e.   (2x³) × 3(x²y²)³ × 5y⁴

(2x³) × 3(x²y²)³ × 5y⁴ = 2 x³ × 3 × [tex]x^{2 \times 3} \times y^{2 \times 3}[/tex] × 5 y⁴

                                  = 2 × 3 × 5 × x³ × x⁶ × y⁶ × y⁴

                                  = 30 x³⁺⁶ y⁶⁺⁴

                                  = 30 x⁹ y¹⁰

30. ^sederhanakan operasi aljabar berikut ini. a. y3 × 2y7 × (3y)2 = b. b × 2y7 × b3 × y2. = c. 3m3 × (mn)4.= d. (tn3)4 × 4t3 = e. (2x3) x 3 (x2y2)3 x 5y4. =


Kelas : X (1 SMA)
Materi : Bentuk Pangkat
Kata Kunci : pangkat, positif, negatif

Pembahasan :
Jika a suatu bilangan real dan n suatu bilangan bulat positif, maka
pⁿ = p x p x ... x p
       ____v_____
           n faktor
dengan
n dinamakan eksponen atau pangkat.
p dinamakan bilangan pokok (atau basis atau bilangan dasar).
pⁿ dinamakan bilangan berpangkat.
p x p x ... x p (sampai dengan n suku) dinamakan hasil perpangkatan.

Sifat-sifat bentuk pangkat, antara lain :
1. pᵃ x pᵇ = pᵃ ⁺ ᵇ,
2. (pᵃ)ᵇ = pᵃ ˣ ᵇ,
3. (p x q)ᵃ = pᵃ x qᵃ,

Mari kita lihat soal tersebut.
Sederhanakan operasi aljabar berikut!
a. q³ x 2q⁷ x (3q)²
b. p x 2q⁷ x p³ x q²
c. 3m³ x (mn)⁴
d. (tn³)⁴ x 4t³
e. (2p³) x p³ x (p²q²)³ x p⁵ x q⁴

Jawab :
Kita ubah variabel x dan b dengan p, serta y dengan q. Supaya tidak keliru antara variabel "x" dengan tanda "kali".

Dengan menggunakan sifat-sifat bentuk pangkat di atas, diperoleh
a. q³ x 2q⁷ x (3q)²
= q³ x 2 x q⁷ x 9 x q²
=  2 x 9 x q³ x q⁷ x q²
= 18 x q³ ⁺ ⁷ ⁺ ²
= 18 x q¹²
= 18q¹²

b. p x 2q⁷ x p³ x q²
= p x p³ x 2 x q⁷ x q²
= 2 x p¹ ⁺ ³ x q⁷ ⁺ ²
= 2 x p⁴ x q⁹
= 2p⁴q⁹

c. 3m³ x (mn)⁴
= 3 x m³ x m⁴ x n⁴
= 3 x m³ ⁺ ⁴ x n⁴
= 3 x m⁷ x n⁴
= 3m⁷n⁴

d. (tn³)⁴ x 4t³
= t⁴ x n¹² x 4 x t³
= 4 x t⁴ x t³ x n¹²
= 4 x t⁴ ⁺ ³ x n¹²
= 4 x t⁷ x n¹²
= 4t⁷n¹²

e. (2p³) x p³ x (p²q²)³ x p⁵ x q⁴
= 2 x p³ x p³ x p⁶ x q⁶ x p⁵ x q⁴
= 2 x p³ ⁺ ³ ⁺ ⁶ ⁺ ⁵ x q⁶ ⁺ ⁴
= 2 x p¹⁷ x q¹⁰
= 2p¹⁷q¹⁰
