Jawaban panjang:    

1. Perbedaan antara citra grafis dan citra vektor  

Citra grafis menyimpan gambar dengan susunan piksel, sedangkan citra vektor menyimpan dengan bentuk poligon. Contoh dari citra grafis atau raster atau bitmap adalah file JPEG dan file bitmap (BMP). Sedangkan contoh file vektor adalah Portable Network Graphic (PNG) dan Graphics Interchange Format (GIF).

2. Perbedaan antara mode warna CMYK dan RGB  

CMYK menyimpan warna dengan kombinasi empat warna tinta dalam percetakan yaitu Cyan, Magenta, Yellow (kuning) dan Black (Hitam). Warna-warna ini dipilih untuk warna tinta printer, karena warna ini bisa menutup dengan baik bahan dengan warna terang, seperti kertas.  

Sedangkan RGB menyimpan warna dalam kombinasi tiga warna primer yaitu Red (merah), Green (Hijau), dan Blue (Biru). Warna ini digunakan untuk warna sinar katoda di layar komputer, dan kemudian digunakan juga di layar LCD. Dengan mengkombinasikan 3 warna ini bisa dibuat warna-warna sekunder yang lain.

3. Arti istilah CAD  

Computer Assited Design atau CAD digunakan untuk membantu arsitek atau insinyer untuk membuat desain produk. Sebelum ada CAD, gambar teknis dibuat secara manual dengan pensil dan kertas. Contoh program CAD adalah autoCAD.

4. Program aplikasi garis berbasis vector  

Program-progam ini mampu membuka, merubah dan menyimpan file gambar yang berbasis vektor seperti file GIF dan PNG. Contoh program adalah Adobe Illustrator, corelDraw, dan Inkscape


5. Program aplikasi garis berbasis bitmap
 

Program-progam ini mampu membuka, merubah dan menyimpan file gambar yang berbasis bitmap seperti file BMP dan JPEG. Contoh program adalah Adobe Photoshop, Microsoft Paint dan GIMP

30. Perbedaan antara aplikasi berbasisi vektor dan aplikasi berbasis pixel adalah...


Kelebihan :

1. Vektor

Ruang penyimpanan untuk objek gambar lebih efisien.
Ukuran dan bentuknya dapat diubah tanpa menurunkan mutu tampilannya.
Dapat dicetak pada resolusi tinggi.
Menggambar dan menyunting vektor relatif lebih muda dan menyenangkan.
2. Bitmap

Dapat diberikan efek khusus tertentu sehingga membuat objek tampil sesuai keinginan.
Menghasilkan objek gambar bitmap dari objek vektor dengan cara mudah dan cepat, serta mutu hasilnya dapat ditentukan.
Kekurangan :

1. Vektor

Tidak dapat menghasilkan objek gambar vektor yang prima ketika melakukan konversi objek gambar tersebut dari format bitmap.
2. Bitmap

Mengalami masalah saat melakukan pengubahan ukuran terutama saat objek gambar diperbesar
Efek yang dihasilkan akan terlihat pecah atau berkurang detailnya saat dicetak pada resolusi yang lebih rendah.
