14. Bagaiman Cara integral


Integral = kebalikan dari turunan sehingga
[tex]\displaystyle \int f'(x) \ d x = f(x) + C[/tex]
dimana C adalah konstan

Hukum-hukum integral:
[tex]\displaystyle \int cf(x) \ dx = c \int f(x) \ dx , \ \text{jika c konstan}[/tex]
[tex]\displaystyle \int f(x) \pm g(x) \ dx = \int f(x) \ dx \pm \int g(x) \ dx[/tex]

Integral standar:
[tex]\displaystyle \int x^n \ dx = \frac{1}{n+1}x^{n+1}, \ n \neq -1[/tex]
[tex]\displaystyle \int \cos x \ dx = \sin x + C[/tex]
[tex]\displaystyle \int \sin x \ dx = -\cos x + C[/tex]
[tex]\displaystyle \int e^x \ dx = e^x + C[/tex]
[tex]\displaystyle \int \frac{1}{x} \ dx = \ln{x} + C[/tex]

Untuk Nomor 2 :

Ini adalah Integral dasar... Jadi kita tidak perlu banyak langkah dalam pengerjaannya

Ini adalah Integral dasar

[tex]\int a\ dx = ax + c \\\text{dengan}\ c\ = \text{kostanta}\\\int x^n\ dx = \dfrac{x^{n+1}}{n+1}\\[/tex]

Soalnya di Upgrade dengan adanya operasi penjumlahan

[tex]\int 3x^2 + 2x + 4\ dx = \dfrac{3x^{(2+1)}}{2+1} + \dfrac{2x^{(1+1)}}{1+1} + 4x + C\\= \dfrac{3x^3}{3} + \dfrac{2x^2}{2} + 4x + C\\= x^3 + x^2 +4x + C[/tex]

Nomor 12 :

Masih model yang sama, hanya saja ini dalam bentuk tentunya

[tex]\int\limits^{x=a}_{x=b} f'(x)\ dx\\\\=\left f(x) \right|^a_b\\=f(a) - f(b)\\\\=\int\limits^2_1 (2x^2 + 4x -5)\ dx[/tex]

Pertama - tama, cari dulu Integral dari (2x^2 +4x -5)

Integral dari (2x^2 +4x -5)

[tex]\int(2x^2 +4x -5) dx = \dfrac{2x^{(2+1)}}{2+1} + \dfrac{4x^{(1+1)}}{1+1} -5x + C\\=\dfrac{2x^3}{3} + \dfrac{4x^2}{2} -5x + C\\= \dfrac{2}{3} x^3 + 2x^2 - 5x + C\\[/tex]

Step terakhir adalah

25. cara menghitung integral


1. Aturan sederhana untuk integral ini berfungsi untuk sebagian besar polinomial dasar.Polinomial y = a*x^n.



2. Bagikan (koefisien) a dengan n+1 (pangkat+1) dan tingkatkan pangkat dengan 1. Dengan kata lain, integral y = a*x^n adalah y = (a/n+1)*x^(n+1).



3. Tambahkan konstanta integral C untuk integral tak tentu untuk mengoreksi ambiguitas melekat mengenai nilai yang tepat. Oleh karena itu, jawaban akhir soal ini adalah y = (a/n+1)*x^(n+1) + C.Pikirkan sebagai berikut: ketika menurunkan fungsi, setiap konstanta dihilangkan dari jawaban akhir. Oleh karena itu, selalu mungkin bahwa integral dari sebuah fungsi memiliki beberapa konstanta sembarang.


4. Integralkan istilah yang terpisah dalam suatu fungsi secara terpisah dengan aturan.Sebagai contoh, integral dari y = 4x^3 + 5x^2 +3x adalah (4/4)x^4 + (5/3)*x^3 + (3/2)*x^2 + C = x^4 + (5/3)*x^3 + (3/2)*x^2 + C.
