3. Berapa Mr dari senyawa berikut K2CrO4?


Mr dari senyawa K₂CrO₄ adalah 194

Penjelasan:

Massa Atom Relatif (Ar) adalah perbandingan massa antara atom yang satu terhadap atom lainnya.

Berdasarkan kesepakatan para ahli kimia, dimana pembandingnya adalah 1/12 dari massa 1 atom C-12.

secara matematis dapat dituliskan sebagai berikut:

[tex]\boxed {A_r \ Unsur \ X = \frac{Massa \ Rata \ - \ Rata \ 1 \ atom \ Unsur \ X}{\frac{1}{12} \ Massa \ 1 \ atom \ C \ - \ 12 } }[/tex]

Karena nilai dari 1/12 Massa 1 atom C-12 adalah sama dengan 1 sma, maka:

[tex]\boxed {A_r \ Unsur \ X = \frac{Massa \ Rata \ - \ Rata \ 1 \ atom \ Unsur \ X}{1 \ sma} }[/tex]

Namun singkat kata dari pengertian Massa Atom Relatif (Ar) yaitu nilai Massa  dari 1 atom tertentu yang dapat dilihat nilainya pada Tabel Sistem Periodik Unsur (SPU)

Contohnya adalah Massa Natrium yaitu 22,99

*Angka 22,99 merupakan hasil perhitungan nilai Ar berdasarkan pada rumus diatas.

Nomor massa merupakan hasil pembulatan dari Massa Atom.

Sedangkan untuk Massa Molekul Relatif (Mr) merupakan hasil penjumlahan dari Massa Atom Relatif (Ar) unsur pembentuk senyawa tersebut.

Berdasarkan pada soal jika diketahui senyawa K₂CrO₄ maka kita bisa hitung nilai Mr nya dengan cara berikut:

Mr K₂CrO₄ = (2 x Ar K) + Ar Cr + (4 x Ar O)

Mr K₂CrO₄ = (2 x 39) + 52 + (4 x 16)

Mr K₂CrO₄ = 78 + 52 + 64

Mr K₂CrO₄ = 194

Jadi Mr dari senyawa K₂CrO₄ adalah 194

8. Bilangan oksidasi atom Cr dalam senyawa K2CrO4 adalah


Bilangan oksidasi atom Cr dalam senyawa K₂CrO₄ adalah +6

Penjelasan:

Bilangan oksidasi (Biloks) adalah bilangan yang menunjukkan keadaan elektronik suatu atom, ion atau molekul kekurangan atau kelebihan elektron. Apabila atom, ion atau molekul kekurangan elektron maka biloksnya akan bernilai positif.

Dengan mempertimbangkan keelektronegatifan unsur, dapat disimpulkan suatu aturan untuk menentukan bilangan oksidasi adalah sebagai berkut:

1. Unsur Bebas, atomnya memiliki bilangan oksidasi = 0

2. Flourin, unsur yang paling elektronegatif dan membutuhkan tambahan 1 elektron, atomnya mempunyai bilangan oksidasi -1 pada semua senyawanya.

3. Bilangan oksidasi atom unsur logam selalu bertanda +

    Untuk logam Alkali (Golongan I A) = +1

    Untuk logam Alkali Tanah (Golongan II A) = +2

4. Bilangan oksidasi suatu unsur dalam bentuk ion tunggal adalah sama dengan muatannya

   Contoh:  Cl⁻ = -1

5. Bilangan Oksidasi atom H umumnya = +1, kecuali dalam senyawaan logam, dimana bilangan oksidasi atom H = -1 dikarenakan unsur logam lebih elektropositif dari unsur Hidrogen

Contoh:

Bilangan Oksidasi atom H dalam NaH, BaH₂ = -1

6. Bilangan Oksidasi atom O umumnya = -2, kecuali pada contoh:

  F₂O = +2

  H₂O₂ = -1

  KO₂ = -1/2

7. Jumlah Bilangan Oksidasi atom unsur-unsur dalam suatu senyawa = 0

8. Jumlah Bilangan Oksidasi atom unsur-unsur dalam suatu ion poliatom = muatannya.

Berdasarkan aturan di atas maka kita bisa menentukan Biloks unsur Cr dalam K₂CrO₄, yaitu:

Berdasarkan pada aturan nomer 7, maka jumlah biloks dalam senyawa K₂CrO₄ adalah 0.

berdasar pada rumus molekul di atas kita ketahui terdapat 3 unsur penyusun yaitu K, Cr, dan O sehingga kita bisa menentukan nilai bilangan oksidasi unsur Cr berdasarkan pada nilai bilangan oksidasi unsur K dan O yang didasarkan pada aturan di atas.

Unsur K merupakan unsur logam Utama golongan IA, sehingga:

Berdasarkan pada aturan nomer 3, maka Biloks dari unsur K = +1

sedangkan untuk Oksigen,

Berdasarkan pada aturan nomer 6, maka Biloks dari unsur O = -2

sehingga:

Biloks K₂CrO₄ = 0 maka,

(2 x Biloks K) + Biloks Cr + (4 x Biloks O) = 0

(2 x (+1)) + Biloks Cr + (4 x (-2)) = 0

(+2) + Biloks Cr + (-8) = 0

Biloks Cr + (-6) = 0

Biloks Cr = +6

Jadi, biloks atom Cr dalam senyawa K₂CrO₄ berdasarkan pada perhitungan diatas adalah +6.

Cr2O7²-

= (2 x Cr) + 7 (-2) = -2

= 2Cr - 14 = -2

= 2Cr = -2+14

= 2Cr = 12

= Cr = 12/2

= Cr = 6

Jadi Biloks Cr dalam Cr2O7²- adalah 6.

K2CrO4

Jadi Biloks Cr dalam K2CrO4 adalah 6.

Pasangan senyawa yang memiliki kromium dengan bilangan oksidasi yang sama adalah [tex](Cr(OH)_{4})^-[/tex]  dan [tex]Cr_{2}O_{3}[/tex] . Jadi pilihan jawaban yang benar adalah E.

Pembahasan

∴ Biloks Cr pada  [tex]K_{2}Cr_{2}O_{7}[/tex]  adalah +6.

Diperoleh dari subtitusi biloks yang sudah diketahui pada [tex]K_{2}Cr_{2}O_{7}[/tex]  untuk memperoleh biloks Cr

∵Biloks total  [tex]K_{2}Cr_{2}O_{7}[/tex]  adalah 0 ( aturan biloks 2), Biloks K adalah +1 ( aturan biloks no 5) dan Biloks O adalah -2 (aturan biloks ke 9)

   Biloks  [tex]K_{2}Cr_{2}O_{7}[/tex]   = 0

  (jumlah atom K x biloks atom K) + (jumlah atom Cr x biloks atom Cr) +

  (jumlah atom O x biloks atom O) = 0

  (2 x 1) + (2 x biloks Cr) + ( 7 x (-2)) = 0

  2 + 2 Biloks Cr - 14 = 0

  2 Biloks Cr= 14 - 2

  Biloks Cr= [tex]\frac{12}{2}[/tex]

  Biloks Cr= +6


∴ Biloks Cr pada  [tex]K_{2}CrO_{4}[/tex]  adalah +6.

∵Biloks total  [tex]K_{2}CrO_{4}[/tex]  adalah 0 ( aturan biloks 2), Biloks K adalah +1 ( aturan biloks no 5) dan Biloks O adalah -2 (aturan biloks ke 9)

  (jumlah atom K x biloks atom K) + (jumlah atom Cr x biloks atom Cr) +

  (jumlah atom O x biloks atom O) = 0

Bilang oksidasi adalah bilangan yang menunjukkan jumlah yang dapat dilepas, diterima ataupun digunakan bersama-sama agar dapat membentuk ikatan dengan berbagai unsur lainnya. Bilangan oksidasi lebih dikenal dengan istilah BILOKS.

Untuk mengetahui jumlah bilangan oksidasi suatu unsur atau suatu senyawa. Terlebih dahulu kita harus memahami ketentuan-ketentuan menghitung bilangan oksidasi. Berikut beberapa ketentuan dalam bilangan oksidasi:

Bilangan oksidasi suatu unsur bebas adalah nol ( 0). Contohnya bilangan oksidasi natrium dalam Na adalah nol Bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) Bilang oksidasi suatu ion monoatomik adalah sesuai dengan jumlah muatannya . Contohnya bilangan oksidasi untuk Na dalam  Na+ adalah  +1 Bilang oksidasi suatu ion poliatomik adalah sesuai dengan jumlah muatannya Bilangan oksidasi untuk unsur golongan IA ( Li, Na, K, Rb, Cs, Fr) adalah +1 Bilangan oksidasi untuk unsur golongan IIA (Be, Mg, Ca, Sr, Ba, Ra) adalah +2 Bilangan oksidasi untuk unsur golongan IIIA (B, Al, Ga, In, Ti) adalah +3 Bilangan oksidasi untuk Hidrogen ( H) adalah +1 kecuali untuk senyawa hidrida logam bilangan oksidasi Hidrogen adalah -1 contohnya pada [tex]MgH_{2}[/tex]   Bilangan oksidasi untuk Oksigen ( O) adalah -2 kecuali untuk senyawa pereksida bilangan oksidasi Oksigen adalah -1 contohnya pada [tex]H_{2}O_{2}[/tex] Bilangan oksidasi senyawa halogen ( F, Cl, Br, I) pada umumnya adalah -1

27. bilangan oksidasi Cr pada ion Cr2O72- dan senyawa K2CrO4 secara berurutan adalah.....


Bilangan oksidasi Cr pada ion Cr₂O₇⁻² adalah +6. Bilangan oksidasi Cr dalam senyawa K₂CrO₄ adalah +6

Pembahasan

∴ Bilangan oksidasi ( biloks) Cr pada Cr₂O₇⁻² adalah +6.

Untuk memperoleh bilangan oksidasi ( biloks) atom Cr dapat dilakukan dengan sutitusi dari bilangan oksidasi yang sudah diketahui pada senyawa Cr₂O₇⁻²

∵Bilangan oksidasi ( biloks)  total Cr₂O₇⁻²  adalah -2 ( karena bilang oksidasi ( biloks) suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi ( biloks)  O adalah -2 ( karena bilangan oksidasi ( biloks) untuk Oksigen ( O) pada umumnya  adalah -2 )

Cr₂O₇⁻²  = -2

(jumlah atom Cr x biloks atom Cr) + (jumlah atom O x biloks atom O) = -2

(2 x biloks Cr) + (7x (-2)) = -2

2 Biloks Cr - 14 = -2

2 Biloks Cr= 14-2

2 Biloks Cr = +12

Biloks Cr = +6

∴ Bilangan oksidasi Cr pada K₂CrO₄ adalah +6.

Untuk memperoleh bilangan oksidasi ( biloks) atom Cr dapat dilakukan dengan subtitusi dari bilangan oksidasi ( biloks) yang sudah diketahui pada senyawa K₂CrO₄

∵Biloks total K₂CrO₄ adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 ) dan bilangan oksidasi  K adalah +1 ( karena bilangan oksidasi untuk unsur golongan IA ( Li, Na, K, Rb, Cs, Fr) adalah +1)

K₂CrO₄ = 0

(jumlah atom K x biloks atom K) + (jumlah atom Cr x biloks atom Cr) + (jumlah atom O x biloks atom O) = 0

(2 x 1) + (1 x biloks Cr) + (4x (-2)) = 0

2 + Biloks Cr - 8 = 0

Biloks Cr= 8-2

Biloks Cr = +6
