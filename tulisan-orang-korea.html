6. Bahasa koreanya salma ditulis dalam tulisan korea


Tulisan korea untuk Salma :

살마 (Salma)

Pembahasan

Dalam ilmu kesusastraan Korea abjadnya dikenal dengan Hangeul ( 한글 ). Hangeul terdiri dari sekitar 21/22 abjad, yang diantaranya :

아 dibaca dengan lafal aㅂ dibaca dengan lafal b atau pㅈ dibaca dengan lafal c atau ch atau jㄷ dibaca dengan lafal d어 dibaca dengan lafal e atau oㅍ dibaca dengan lafal ph pengganti fㄱ dibaca dengan lafal g atau kㅋ dibaca dengan lafal khㅎ dibaca dengan lafal h이 dibaca dengan lafal iㄹ dibaca dengan lafal l pengganti rㅁ dibaca dengan lafal mㄴ dibaca dengan lafal n오 dibaca dengan lafal oㅅ dibaca dengan lafal sㅌ dibaca dengan lafal t우 dibaca dengan lafal u야 dibaca dengan lafal ya유 dibaca dengan lafal yu요 dibaca dengan lafal yo으 dibaca dengan lafal eu/u

Kata Salma tersusun dari 5 suku kata yaitu :

S - A - L - M - A

maka hangeulnya disusun dari  :

= ㅅ + 아 + ㄹ + ㅁ + 아= 살마

Catatan :

Jika ada abjad 아 terletak di tengah kata/akhir kata, maka partikel 0 dihilangkan dan pada awal kata boleh tersusun lengkap. Begitupun dengan abjad 우, 유, 으, 어, 이 dan 요Jika ada abjad ㄹ terletak di tengah kata, maka kedudukannya berada di bawah. Begitupun dengan abjad ㄴ, ㄱ, 우, 유, 으, dan 요.

12. tuliskan bahasa korea BUBU dalam tulisan korea!!​


Nama "Bubu" dalam bahasa Korea adalah 부부.

Pembahasan

Huruf Hangeul Korea yang tidak bisa dibaca adalah "ㅇ". Apabila diletakkan pada awal kalimat maka "ㅇ" tidak dapat dibaca, namun jika diletakkan diakhir kalimat maka akan dapat dibaca. Penjelasan :

• Diawal kalimat : "ㅇ" tidak dapat dibaca dan disimbolkan dengan "-".

• Namun jika "ㅇ" diakhir kalimat mungkin dapat dibaca dengan "ng".

"ㅇ" jika diletakkan diawal kalimat dapat digunakan untuk tambahan huruf vokal. Contohnya adalah: "어떻게 지냈어요".Namun, apabila "ㅇ" diletakkan diakhir kalimat dibaca sebagai "ng". Contohnya: 안영 "Annyeong".

Bubu => 부부

ㅂ (b/p)ㅜ (u)ㅂ (b/p)ㅜ (u)

13. bahasa korea nya terimakasih dengan tulisan bahasa koreanya


Bahasa Koreanya Terimakasih adalah :

Khamsamnida (감사 합니다) ➡️ FormalGomawo (고마워) ➡️ NonFormal

Pembahasan

Sejak tahun 2015, bahasa korea menjadi trending di kalangan anak muda karena terbawa akan adanya drama korea (Drakor) yang memiliki daya tarik tersendiri. Sehingga, banyak anak muda Indonesia yang mencoba menghafal beberapa ungkapan-ungkapan sehari-hari dalam bentuk bahasa Korea. Berikut saya cantumkan beberapa ungkapannya :

1. Ungkapan Salam

Salam adalah uangkapan pembukaan dalam pertemuan maupun kegiatan. Dalam bahasa korea, ungkapan ini di notasikan dengan Anyeong haseyo ( 안녕하세요 ) .Namun kata ini digunakan untuk acara nonformal, seperti pertemuan dengan teman sebaya. Untuk ucapan salam secara formal, biasanya orang korea akan menggunakan kata Anyeong Hasimnikka ( 안녕하십니까 ). Contoh penerapannya sebagai berikut :

Anyeong haseyo, Jungkookie! (Hai, Jungkook!)Anyeong hasimnikka, Aboenim (Hai, Pak!)

2. Ungkapan Selamat

Beberapa ungkapan selamat dalam bahasa korea adalah sebagai berikut :

Sangil chukaehamnida ( 생일축하합니다 ), artinya Selamat berulang tahun!Ceulgoun hyuin bonaesipsi-o ( 즐거운휴일보내십시오 ), artinya selamat liburan!Sae haebok manipadeusipsi-o ( 새해 복 많이 받으십시오), artinya selamat tahun baru!oso-o sipsi-o ( 어서오십시오 ), artinya selamat datang!

3. Ungkapan Terimakasih

Beberapa kata yang sering digunakan untuk mengungkapkan rasa terimakasih adalah :

Khamsamnida ( 감사 합니다 ), diperuntukan untuk kegiatan yang berbentuk formal/sakralGomawo ( 고마워 ) , diperuntukan untuk kegiatan/pertemuan yang tidak formal (non-formal)Gomawoyo ( 고마워요 ) , diperuntukan untuk kegiatan/pertemuan dengan teman dekat (lebih non formal)

3. Ungkapan Perasaan

Beberapa kata yang bisa digunakan untuk mengungkapkan perasaan kita adalah :

Himdeuro ( 힘들어 ) : Sangat lelahSil-mang haessoyo ( 실망했어요 ) : Aku kecewaBogoshipoyo ( 보고싶어요 ) : Aku RinduBae-go paeyo ( 배고파요 ) : Aku laparMaeumi apaeyo ( 마음이 아파요 ) : Hatiku sakitGwencanayo ( 괜찮아요 ) : Aku baik-baik sajaJoaeyo ( 좋아요 ) : Aku Senang Nappaeyo ( 나빠요 ) : Aku Kurang Senang
