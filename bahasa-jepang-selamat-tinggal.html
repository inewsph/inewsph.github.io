Sayounara [ さようなら] Merupakan salam Perpisahan , Sayonara Memiliki arti yaitu 'Selamat Tinggal' , Biasanya digunakan ketika Seseorang akan Pergi dalam Jangka waktu yang lama

Ada Pun Mata Ashita :

またあした [ Mata Ashita ] , Memiliki arti yaitu 'Selamat Jumpa lagi Besok' , Yang あした [ Ashita ] artinya yaitu Besok

Tetapi sebenarnya Penggunaan Sayounara dalam bahasa Jepang Jumlah nya terbatas ! tidak sama dengan Sampai Jumpa dalam bahasa Indonesia

Selain Itu Ucapan Selamat ( Salam ) Bukan hanya Sayounara saja , Contoh nya ada : Selamat Pagi , Selamat Malam , Selamat Tidur , Selamat Siang. , Dan Lain lain

Ucapan Perpisahan :

では、また [ Dewa Mata ] Yang artinya yaitu ' Permisi, Sampai jumpa lagi - Yang dewa artinya yaitu Permisi , Dan Mata artinya Yaitu Lagiまたね [ Mata ne ] Yang artinya : Sampai Jumpa lagi , Mata memiliki arti yaitu Lagiじゃあまた [ Jaa-Mata ] Yang artinya Permisi , sampai Jumpa lagi --> Ungkapan Informal Dari dewa mata

Contoh Kalimat :

Mata Ashita Memiliki arti yaitu sampai jumpa lagi besok Ashita Memiliki arti yaitu besok dan mata Memiliki arti yaitu Lagi

12. Bahasa Jepang Hobi : Selamat Pagi : Selamat Tinggal : Terimakasih : Pake Kanji Ya


Hobi : 趣味

Selamat pagi : お早う御座います

Selamat tinggal : 左様なら

Terimakasih : 有り難う御座います

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PEMBAHASAN

趣味 (Shumi)

Artinya adalah "Hobi". Penulisan kata ini dalam Hiragana adalah しゅみ (Shumi), dan dalam Katakana adalah シュミ (Shumi).

--

Salam atau sapaan dalam Bahasa Jepang disebut dengan 挨拶 (Aisatsu). Aisatsu ini sangat penting dan selalu digunakan orang Jepang dalam kehidupan sehari-hari. Beberapa contoh Aisatsu yang akan saya bahas antara lain sebagai berikut.

お早う御座います (Ohayou gozaimasu)

Artinya adalah "Selamat pagi". Ungkapan ini formal, jadi jarang digunakan ke teman. Versi lebih kasualnya dengan menghilangkan 'Gozaimasu'. Penulisan kata ini dalam Hiragana adalah おはようございます (Ohayou gozaimasu), dan dalam Katakana adalah オハヨウゴザイマス (Ohayou gozaimasu).

左様なら (Sayounara)

Artinya adalah "Selamat tinggal". Ungkapan ini digunakan sebagai salam perpisahan, yang dimana kita akan berpisah dengan lawan bicara tanpa tahu kapan akan bertemu lagi. Penulisan kata ini dalam Hiragana adalah さようなら (Sayounara), dan dalam Katakana adalah サヨウナラ (Sayounara).

有り難う御座います (Arigatou gozaimasu)

Artinya adalah "Terimakasih". Ungkapan ini termasuk sopan, dan biasa tidak digunakan ke orang yang dekat atau akrab. Versi lebih kasualnya dengan menghilangkan 'Gozaimasu'. Penulisan kata ini dalam Hiragana adalah ありがとうございます (Arigatou gozaimasu), dan dalam Katakana adalah アリガトウゴザイマス (Arigatou gozaimasu).

17. apa artinya dari bahasa jepang "SAIKO" bahasa jepang nya SELAMAT TINGGAL ? Bahasa jepang nya TELAH MENINGGAL ?


最古 (saiko) artinya adalah tertinggi atau tertua.Bahasa Jepangnya selamat tinggal adalah さよなら (sayonara).Bahasa Jepangnya telah meninggal adalah 死んでしまった (shinde shimatta).

Pembahasan

Aisatsu - Salam

Pembahasan mengenai soal kali ini asalah aisatsu (salam), namun mengenai salam perpisahan atau salam penutup. Salam penutup memiliki banyak frasa dalam bahasa Jepang, antara lain sebagai berikut :

さよなら (sayonara)

Ungkapan salam perpisahan yang satu ini, khusus digunakan ketika seseorang atau diri kita akan pergi jauh dan waktu untuk bertemu kembali kemungkinan akan lama.

またね (mata ne)

kata また (mata) dalam bahasa Indonesia artinya adalah lagi. Sehingga kata ini memiliki arti sampai jumpa lagi dalam salam perpisahan. Kalimat ini merupakan salah satu kalimat informal.

じゃあね (jaa ne)

Salah satu kalimat ini cocok digunakan dalam situasi informal, karena ini merupakan salah satu kalimat yang digunakan untuk orang yang lebih muda atau seumuran dengan kita. Jaa ne memiliki beberapa makna yang berkaitan dengan salam perpisahan yaitu adalah :

Da daah!Sampai jumpa.Bye-bye.

____________________________

Kalimat 'telah meninggal' dalam bahasa Jepang memiliki kalimat-kalimat sebagai berikut, selain 死んでしまった (shinde shimatta) :

死んでしました (shine shimash¡ta), dan死ぬでした (shinu desh¡ta).
