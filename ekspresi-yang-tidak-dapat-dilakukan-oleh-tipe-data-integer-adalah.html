Nilai sebuah data dari sebuah tipe data integer adalah nilai bilangan bulat seperti dalam matematika. Representasi data ini merupakan aktivitas yang dipekerjakan bagaimana nilainya disimpan di dalam memori komputer.

Nilai sebuah data dengan tipe data integer tidak lain adalah bilangan bulat, baik bertanda (negatif), maupun tak bertanda (positif dan 0). Secara internal dalam komputer, nilai data bertipe integer direpresentasikan dengan untaian bit (binary digit, atau digit biner), yaitu 0 dan 1, dengan panjang sesuai definisi dan implementasi tipe data integer oleh bahasa pemrograman atau aplikasi.
 

Pembahasan

Integer, tidak lain adalah bilangan bulat. Pada literatur/referensi berbahasa Inggris, baik dalam bidang matematika maupun informatika atau ilmu komputer, bilangan bulat disebut integer.

Kita tahu bahwa bilangan bulat terdiri atas bilangan bulat negatif, 0, dan bilangan bulat positif. Pada bidang informatika, bilangan bulat atau integer terbagi atas 2 jenis, yaitu:

bilangan bulat bertanda, atau signed integer, yaitu bilangan bulat negatif.bilangan bulat tak bertanda, atau unsigned integer, yaitu 0 dan bilangan bulat positif.

Secara internal dalam mesin komputer, tipe data integer direpresentasikan dengan untaian bit. Berapa panjang untaian bit tersebut, dan seberapa lebar/besar wilayah nilai bilangan bulat yang dapat dimuat, tergantung pada definisi dan implementasi oleh bahasa pemrograman, atau aplikasi yang mengandung fitur pemrograman.

Jika [tex]n[/tex] bit dialokasikan untuk tipe data integer, maka sebuah data atau variabel bertipe integer dapat memuat bilangan bulat dalam rentang [tex]-2^{n-1}[/tex] hingga [tex]2^{n-1}-1[/tex].

Sebagai contoh, pada bahasa pemrograman C++, [tex]\tt sizeof(int)[/tex] mengembalikan nilai 4. Artinya, 4 byte = 4 × 8 bit = 32 bit dialokasikan untuk sebuah variabel bertipe data [tex]\tt int[/tex] atau integer.

Maka, sebuah variabel bertipe data integer pada C++ dapat diisi nilai bilangan bulat dari [tex]-\bf2^{31}[/tex] sampai [tex]\bf2^{31}-1[/tex], atau –2.147.483.648 sampai 2.147.483.647.

Representasi biner untuk tipe data integer dapat berupa little endian atau big endian. Namun, secara umum, pada [tex]n[/tex] bit bilangan biner yang merepresentasikan tipe data integer:

[tex]n-1[/tex] bit digunakan untuk menyimpan nilainya, dan 1 bit digunakan untuk menyimpan tanda (1 jika negatif, 0 jika positif).

Representasi tersebut juga tergantung pada sistem komplemen biner yang digunakan. Sebagai contoh, pada sistem komplemen-1 (1's complement), dengan 8 bit integer, nilai +1 direpresentasikan dengan 00000001. Sedangkan nilai –1 direpresentasikan dengan 11111110, yang merupakan komplemen dari +1 atau 00000001.

20. Ukuran Memori dari tipe data int / Integer adalah​


1. Tipe data Integer / Ordinal

Tipe data ini adalah tipe data yang berupa bilangan bulat yang akan digunakan untuk operasi matematika. Ada beberapa tipe data yang termasuk integer, yaitu antara lain:

a. ShortInt : memiliki nilai range -128 sampai 127 (signed 8-bit)

b. SmallInt : memiliki nilai range antara - 32768 sampai dengan 32767 (signed 16-bit)

c. LongInt : memiliki nilai range -2147483648 sampai 2147483647 (signed 32-bit )

d. Int64 : memiliki range -263+1 sampai 263-1 (signed 64-bit)

e. Byte : memiliki nilai range 0 sampai 255 (unsigned 8-bit)

f. Word : memiliki nilai range 0 sampai 65535 (unsigned 16-bit)

g. LongWord : memiliki nilain range 0 sampai 4294967295(unsigned 32-bit )

Pada pemakaian integer type di implemetasi 16 bit maka:

a. Integer : memiliki range antara - 32768 sampai dengan 32767 (signed 16-bit)

b. Cardinal : memiliki range 0 sampai 65535 (unsigned 16-bit)

Pada pemakaian integer type di implemetasi 32 bit maka:

a. Integer : memiliki range antara -2147483648 sampai 2147483647 (signed 32-bit)

b. Cardinal : memiliki range 0 sampai 4294967295 (unsigned 32-bit)

25. Terangakn nilai sebuah data dari sebuah tipe data integer.


Tipe data integer adalah tipe data numerik yang hanya dapat menyimpan nilai-nilai bilangan bulat, yaitu bilangan tanpa koma. Tipe data integer terbagi menjadi beberapa jenis, yaitu tipe data integer 8-bit (signed char), tipe data integer 16-bit (short), tipe data integer 32-bit (int), dan tipe data integer 64-bit (long long). Masing-masing tipe data integer memiliki jangkauan nilai yang berbeda.

Tipe data integer 8-bit (signed char) dapat menyimpan nilai dari -128 sampai 127.

Tipe data integer 16-bit (short) dapat menyimpan nilai dari -32,768 sampai 32,767.

Tipe data integer 32-bit (int) dapat menyimpan nilai dari -2,147,483,648 sampai 2,147,483,647.

Tipe data integer 64-bit (long long) dapat menyimpan nilai dari -9,223,372,036,854,775,808 sampai 9,223,372,036,854,775,807.

Tipe data integer sering digunakan dalam pemrograman untuk menyimpan nilai-nilai numerik yang tidak memiliki koma, seperti jumlah, skor, atau counter. Tipe data integer juga sering digunakan dalam perhitungan matematis karena lebih efisien dibandingkan dengan tipe data float atau double yang memiliki kemampuan untuk menyimpan nilai numerik dengan koma. Namun, tipe data integer tidak dapat menyimpan nilai numerik yang memiliki koma, sehingga tidak cocok untuk menyimpan nilai-nilai yang memerlukan presisi yang tinggi.
