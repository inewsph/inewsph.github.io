2. sub menu dari menu format yang terdapat dalam MS word



MENU FILE
New: berfungsi untuk membuat file atau workbook baru

Page setup: mengatur tata letak halaman sebelum dicetak

Print preview: menampilkan file yang dicetak pada tampilan monitor

Print: perintah untuk mencetak dokumen melalui printer

Exit: keluar dari program Microsoft Word 

MENU EDIT
Undo: berguna untuk membatalkan perintah yang telah kita lakukan sebelumnya

Redo: berguna untuk membatalkan perintah undo

Cut: berguna untuk memotong (sebagai proses memindahkan teks / obyek dengan cara cut-paste)

Copy: menyalin teks / obyek

Paste: meletakkan objek / teks yang telah dicopy atau dipaste

Paste special: meletakkan salinan / potongan dengan berbagai format

Select all: memilih (memblok) seluuruh dokumen

Find: mencari kata pada kata pada dokumen

Go to: melompat

Web layout: membuat tampilan web

Task pane: menampilkan task pane atau jendela tugas

Toolbar: menampilkan / menyembunyikan toolbar

Ruler: Menampilkan / menyembunyikan bar rumus

Full screen: menampilkan lembar kerja dalam satu layar

Zoom: memilih besarnya tampilan lembar kerja 

MENU INSERT
Break: menyisipkan secsion

Page number: memberi nomor pada halaman

Auto text: menyisipkan teks otomatis

Symbol: menyisipkan simbol

Picture: menyisipkan gambar

Diagram: menyisipkan diagram

Text box: menyisipkan teks box

File: menyisipkan file

MENU FORMAT


Font: memformat font

Paragraph: memformat paragraph

Bullet and numbering: menyisipkan format bullet dan numbering

Border and shading: memformat garis dan bayangan

Columns: menerapkan format berkolom

Tab: memformat tab

12. Untuk menampilkan daftar judul bab dan sub bab yang telah diformat di ms word 2010 dapat dilakukan dengan mengaktifkan menu ...


Untuk mengaktifkan menu daftar judul bab dan sub bab yang ada di microsoft office word 2010 dapat dilakukan dengan meng-klik multilevel list dan untuk membuat list seperti daftar isi dapat menggunakan fitur references, lalu klik table of contents. Setelah itu, klik insert table of contents dan pilih berbagai macam formatnya sesuai yang dibutuhkan.

Pembahasan

Fitur multilevel list di home ditujukan untuk mempermudah pengguna microsoft word untuk membuat judul bab dan sub bab. Selain multilevel list, menu table of contents di references juga dapat digunakan untuk membuat daftar isi.

Fitur-fitur tersebut tidak hanya terdapat di microsoft word 2010 saja, tetapi juga di semua versi microsoft word 2007, 2010, hingga versi terbaru yaitu microsoft word versi 2019.

Untuk mengaktifkan fitur tersebut, pertama-tama, pastikan judul bab ada di heading 1, dan text rata kanan-kiri dengan memakai fitur justify. Penggunaan table of contents sendiri biasanya digunakan untuk membuat daftar isi sebuah makalah atau jurnal, sedangkan multilevel list biasa digunakan untuk membuat sub bab dari bab di dalam makalah/jurnal.
