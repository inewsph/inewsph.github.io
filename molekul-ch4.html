21. Ramalkan bentuk molekul dari CH4 !


Bentuk molekul dari CH₄ adalah tetrahedron. Bentuk ini juga dinamakan bentuk tetrahedral.

Penjelasan dengan langkah-langkah

Untuk mengetahui bentuk molekul, dapat diketahui melalui teori VSPER (Valence Shell Electron Pair Repulsion).  Teori ini memberikan suatu model yang dapat digunakan untuk menggambarkan bentuk-bentuk molekul sesuai gaya tolak-menolak elektrostatik antar pasangan elektron pada kulit valensi atom pusat. Notasi VSEPR adalah :

[tex]\boxed{\sf AX_aE_x ~~dengan ~~E =\frac{elektron~valensi~A -X}{2} }[/tex]

Keterangan:

A : atom pusat X : PEI (Pasangan Elektron Ikatan)E : PEB (Pasangan Elektron Bebas)ₐ : jumlah PEI ₓ : jumlah PEB

Diketahui :

Ar C = 14Ar H = 1
(Ref : Tabel periodik unsur)

Ditanyakan :

Bentuk molekul CH₄ = ?

Penyelesaian :

Konfigurasi elektron C dan H

C = 2.8.4 ==> elektron valensi = 4

H = 1 ==> elektron valensi = 1

Perhitungan PEI

Atom pusat = C
dengan atom terikat berupa 4 atom HX = 4

Perhitungan PEB

Masing-masing atom H perlu 1 elektron untuk memenuhi aturan duplet, maka :
b = 1 E = (4 - 4) : 2
E = 0

Notasi VSPER

AX₄E₀ = AX₄Berdasarkan VSPER bentuk CH₄ berbentuk tetrahedron

Gambar struktur molekul CH₄

[tex]\begin{array}{ccccc}&&H&&\\&&\circ\times&&\\ H&{{\circ} \atop {\times}} &\bf C&{{\circ} \atop {\times}}&H\\&&\circ\times&&\\&&H&&\end{array}[/tex]

Pembahasan

Untuk menyelesaikan soal ini, teman teman brainky akan menggunakan rumu berikut dibawah ini . Di ingat ingat rumusnya agar nantinya mudah untuk menyelesaiakn soal yang seperti ini .

[tex] \\ [/tex]

▪︎ Rumus menentukan mol• [tex]{\boxed{\sf n = \frac{N}{L} }} [/tex]▪︎ Rumus menentukan jumlah partikel• [tex] {\boxed{\sf N = n \: \times L}} [/tex]Keterangannya :n = mol ( mol )N = jumlah partikelL = Bilangan avogadro ( 6,02 × 10²³ )

[tex] \\ [/tex]

▪︎ Rumus menentukan mol• [tex]{\boxed{\sf \: n = \frac{m}{Ar} }}   [/tex]▪︎ Rumus menentukan massa• [tex] {\boxed{\sf m = n × Ar}} [/tex]▪︎ Rumus menentukan Ar• [tex]{\boxed{\sf Ar= \frac{m}{n}}} [/tex]Keterangannya :n = mol ( mol )m = massa ( gram )Mr = Massa molekul relatif ( gram/ mol )

Diketahui :

m = 8 gramMr CH4 = 16

Ditanya :

N .. ?

Dijawab :

[tex] \\ [/tex]

• Menentukan mol

[tex]\sf \: n = \frac{m}{Mr}    [/tex]

[tex]\sf \: n = \frac{8}{16}   [/tex]

[tex]{\boxed{\sf \: n = 0.5\: mol }}[/tex]

[tex] \\ [/tex]

• Menentukan jumlah partikel

[tex] \sf n = \frac{N}{L}[/tex]

[tex] \sf N = n \: \times L[/tex]

[tex] \sf N = 0.5\: \times {6.02 \times 10}^{23} [/tex]

[tex] \sf N = {5×10}^{-1} \: \times {6.02 \times 10}^{23} [/tex]

[tex] \sf N = {6.02 × 5 \times 10}^{23-1} [/tex]

[tex] {\boxed{\sf N = {30.1 \times 10}^{22}  }} [/tex]

jadi, jumlah molekuonya adalah 30.1 × 10²²
