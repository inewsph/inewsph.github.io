Jawaban & Penjelasan:

Perangkat keras komputer yang termasuk sebagai alat keluaran antara lain (c.) monitor dan printer.

Perangkat keras komputer sebagai alat keluaran antara lain :

1.Monitor => adalah sebuah perangkat keras yang digunakan untuk mengeluarkan hasil pemrosesan yang dilakukan oleh komputer berupa informasi yang dibutuhkan oleh orang yang memakai komputer.

2.Printer => adalah perangkat keras keluaran yang digunakan untuk mencetak data seperti gambar/foto, tulisan pada media kertas.

3.Speaker => adalah perangkat keras yang berfungsi untuk mengeluarkan hasil pemrosesan berupa suara dari komputer.

4.Projector => adalah perangkat keras yang mengintegrasikan sumber cahaya, sistem optik elektronik, dan display dengan tujuan untuk memproyeksikan gambar atau video ke dinding atau layar.

5.Network Card => adalah perangkat keras yang digunakan sebagai jembatan penghubung antara komputer server dengan komputer client.

6. Tv Tuner Card => adalah perangkat keras yang digunakan untuk menyiarkan tv pada komputer.

7. Modem => adalah perangkat keras yang digunakan untuk menghubungkan komputer ke jaringan internet. Modem juga bisa sebagai media masukan ataupun keluaran.

8. Router => adalah perangkat keras jaringan yang berfungsi untuk menghubungkan beberapa jaringan yang memiliki akses jaringan sama ataupun berbeda.

9. Microphone => adalah perangkat keras keluaran yang dapat menangkap gelombang suara lalu merubahnya menjadi getaran listrik. Microphone ini memiliki fungsi untuk mengeluarkan suara atau menjadi media rekaman suara yang divisualkan melalui gelombang getaran suara.

10. Sound Card => adalah perangkat keras komputer yang berfungsi sebagai media pengolah data berupa audio atau suara yang menghubungkan antara perangkat penghubung masukan berupa suara (seperti microphone) ke dalam komputer.

11. Headset / Headphone => adalah perangkat keras komputer yang berfungsi sebagai media pengeluaran untuk gelombang suara yang telah disistem oleh komputer.

Pembahasan  Perangkat keras (hardware) adalah komponen fisik dari sebuah komputer seperti keyboard, CPU, mouse, RAM (Random Access Memory), Hard disk (HDD), dan lain-lain. Memory bukan termasuk perangkat keras. Namun, termasuk perangkat keras dengan nama khusus misalnya RAM. Perangkat keras komputer terbagi menjadi 4, yaitu perangkat input, perangkat output, perangkat processing, dan media penyimpan. Perangkat input berfungsi untuk memasukkan data dan program ke dalam komputer. Media penyimpan berfungsi untuk menyimpan data atau program komputer. Contoh: Hard disk, CD, DVD, USB Flash disk, dan lain-lain. Mouse berfungsi sebagai alat pointer untuk mengakses melalui layar monitor. Scanner untuk mengubah data hard copy menjadi soft copy yang dapat diolah di komputer. Joystick adalah tuas kendali yang dapat digerakkan ke segala arah sebagai pelengkap permainan video. Mouse dapat digunakan untuk mengoperasikan beberapa perintah seperti click, double klik, right click, dan drag. Drag dioperasikan dengan mengarahkan pointer mouse pada objek, kemudian menekan dan menahan tombol kiri mouse sambil digerakkan ke tempat yang diinginkan lalu dilepaskan. Keyboard berdasarkan fungsi dasarnya dibagi menjadi 4, yaitu Numeric key, function key, typewriter key dan special function key. Numeric key menampilkan angka, tanda baca titik dan operasi hitung. Perangkat output (output device) berfungsi untuk menyampaikan atau menampilkan informasi. Contoh: Monitor, speaker, dan printer.  ROM singkatan dari Read Only Memory merupakan salah satu memori yang ada dalam komputer yang bersifat permanen dan hanya bisa dibaca saja. Contoh: ROM BIOS (program dasar sistem komputer). RAM (Random Access Memory) dan Hard disk adalah media penyimpan data komputer yang dapat berubah tergantung penggunaan. Memory card adalah media penyimpan data digital pada gadget. Monitor merupakan perangkat output berupa gambar digital (softcopy). Printer merupakan perangkat output berupa gambar atau tulisan cetak (hardcopy). Speaker merupakan perangkat output berupa audio. Sedangkan mikrofon merupakan alat masukan (suara). Printer Dot Matrix menggunakan pita sebagai tinta dan pin sebagai pemukul untuk mencetak. Printer Laser jet merupakan jenis printer yang menggunakan laser pada cetakannya. Printer Inkjet mencetak gambar dan tulisan dengan memancarkan tinta pada kertas.
