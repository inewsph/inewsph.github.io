10. fungsi kalimat i would rather .. than.. dan contohnya


Fungsi dari kalimat I would rather... Than.. Adalah untuk menyatakan apa yang lebih disukai.. Contohnya : I would rather stay home than go fishing.. Itu berarti saya lebih suka tinggal dirumah daripada pergi memancing.. Answer:
________

Would rather ..than digunakan untuk mengungkapkan atau mengekpresikan pilihan yg lebih kita sukai baik yang bersifat khusus(expressing specific preference) maupun yang bersifat umum (expressing general preference).

Examples /contoh :
=> I would rather stay at home than go to the beach(Aku lebih suka tinggal dirumah daripada pergi kepantai)
=> I would rather walk than drive.
(aku lebih suka jalan daripada mengendarai/ naik/mengemudikan kendaraan)

Keterangan:
__________
Yang perlu diingat setelah kata"than" harus menggunakan kata kerja bentuk pertama atau verb 1.

Penjelasan:

digunakan untuk menjawab pertanyaan perbandingan atau untuk memilih antara suatu objek dengan objek lain.

PART = STATING PREFERENCE - WOULD RATHER

-----------------

ANSWER :

1.  She  would rather eat  beef steak  than (eat) fried chicken

2. I would rather watch horror movies than (watch) romance movies

3. He would rather go swimming than read English books.

Note/ Discussion  :

"Would rather" merupakan salah satu ungkapan untuk menunjukkan perasaan suka terhadap sesuatu/seseorang. Kata penghubung perbandingannya menggunakan "than." Harap diingat masing-masing ungkapan preference memiliki kata penghubung perbandingan berbeda-beda.

EXPLANATION :

PREFERENCEUngkapan perbandingan perasaan lebih menyukai terhadap sesuatu/seseorang.Tiap preference memiliki konjungsi masing-masing yang menjadi pasangannya. Konjungsi ini sebagai perbandingan terhadap sesuatu/seseorang. So, harap dihafalkan konjungsi perbandingannya sehingga tidak salah penempatan pada masing-masing pereference.KINDS OF PREFERENCE :

Penggunaan "prefer"  ada 2 jenis yaitu dengan diikuti Gerund { the word "prefer" is one of common verbs should be followed by verb-ing  } dan TO-infinitive .

1. By using PREFER .........TO

PATTERN  :

S+prefer (s)+Gerund/noun +TO+Gerund/noun

Example :

1. I prefer writing to speaking ( by using Verb-ing )

{ saya lebih suka menulis daripada berbicara }

2. They prefer car to bus ( by using noun )

{ Mereka lebih suka mobil dari pada bis }

2. By using WOULD PREFER TO .............RATHER THAN

Penggunaan "prefer" dengan to-infinitive. " would prefer " bisa di kontraksi menjadi "  'd prefer ."

PATTERN :

S+would prefer+to+verb 1+rather than+verb1

Example :

He would prefer to eat fruit salad rather than (to) eat fried chicken.

{ Dia lebih suka makan salad buah daripada makan ayam goreng }

3. By using  WOULD RATHER..............THAN

PATTERN :

S+Would rather+verb1 + than + verb1

Example :

I would rather drink ginger milk than (drink) coffee in the morning.

{ Aku lebih suka minum susu jahe daripada kopi di pagi hari }

4. By using LIKE .............BETTER THAN

PATTERN :

S+like(s)+ noun/gerund+better than+noun/gerund.

Example :

1. I like eating the fruits better than (eating) the meat  ( by using verb-ing )

{ Aku lebih suka makan buah-buahan daripada daging }

2. I like an English grammar book better than a history book (by using noun/noun phrase)

{ Aku lebih suka buku grammar bahasa inggris daripada buku sejarah }
