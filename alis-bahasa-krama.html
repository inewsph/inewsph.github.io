24. buatlah kalimat dalam bahasa Jawa menggunakan Krama dari kata² : 1. rambut 2. mata = bahasa jawa 3. irung 4. alis 5. mulut = bahasa jawa


Nggawe ukara nganggo basa Krama.

1. Rambute mbak Rima sae banget, ngandhan andhan kadhos ombak.

2. Mas Bayu damel kacamripat cemeng amarga saweg sakit mripat.

3. Irunge mbak Ayu mbangir kadhos tiyang bule.

4. Dhik Nisa niku alise ketingal sae, jlarit ngoten.

5. Tiyang niku cangkeme damel ginem sing sae mawon.

Pembahasan

⚫ Basa Krama, yaiku basa sing tembung tembunge nggawe basa Krama, nanging ora dicampur karo basa Ngoko lan basa krama alus.

➢ Panggonane :

wong sing durung kenalwong sadrajat sing durung akrab

➢ Tuladha :

ampeyane bapak sakit amargi saweg dhawah.Tangane Adhik kesleo dados mboten saged mlebet sekolah.

⚫ Jlentrehne ukara ing ngisor.

1. Rambute mbak Rima sae banget, ngandhan andhan kadhos ombak.

➡️ rambut = rambut ( Krama ), rikma ( Krama Alus ).

2. Mas Bayu damel kacamripat cemeng amarga saweg sakit mripat.

➡️ mata = mripat ( krama ), paningal ( Krama Alus ).

3. Irunge mbak Ayu mbangir kadhos tiyang bule.

➡️ irung = irung ( Krama ), Grana ( Krama Alus ).

4. Dhik Nisa niku alise ketingal sae, jlarit ngoten.

➡️ alis = alis ( Krama ), imba ( Krama Alus ).

5. Tiyang niku cαngkeme damel ginem sing sae mawon.

➡️ mulut = cαngkem ( Krama ), tutuk ( Krama Alus ).
