16. Cara menghitung bilangan pangkat perkalian,pembagian,pangkat


Jawaban:

sifat pangkat

Penjelasan dengan langkah-langkah:

bilangan berpangkat jika dikali pangkatnya ditambah. contoh:

[tex] {a}^{2} \times {a}^{5} = {a}^{2 + 5} = {a}^{7} [/tex]

bilangan berpangkat jika dibagi pangkatnya dikurang. contoh: [tex] {a}^{8} \div {a}^{2} = {a}^{8 - 2} = {a}^{6} [/tex]bilangan berpangkat, jika dipangkatkan lagi, maka pangkatnya dikali. contoh: [tex]( { {a}^{4} })^{2} = {a}^{4 \times 2} = {a}^{8} [/tex]
