Rp330.000,00

b) harga setiap kg alpukat?

Rp16.500,00

Penjelasan dengan langkah-langkah:

Diketahui:

Alpukat yang dijual = 20 kg.

Untung = 37,5% atau Rp90.000.

a) berapa penjualan seluruh alpukat?

= (100/37,5 x 90.000) + 90.000

= 240.000 + 90.000

= 330.000

b) harga setiap kg alpukat?

= 330.000 : 20 kg

= 16.500

Jawaban:

keuntungan per alpukat =

[tex] \frac{90.000}{20} = 4.500[/tex]

harga dasar 20 alpukat (modal) =

[tex] \frac{90000}{37.5\%} = 240000[/tex]

maka :

a. harga jual semua alpukat

240000 + 90000 = 330000

b. harga setiap jual setiap alpukat

240000 : 20 = 12000

per alpukat = 12000 + 4500 = 16500

Cintya membeli 2 kg pear,1kg alpukat dan 2kg naga dengan harga rp91.000,00.santi membeli 2 kg pear dan 1 kg alpukat dan 1 kg buah naga dengan harga rp76.000,00.sementara ani membeli 2 kg pear dan 3kg alpukat dan 2 kg buah naga dengan harga rp121.000,00.jika ana ingin membeli 1 kg pear 1kg alpukat dan 1kg buah naga berapakah uang yang harus dibayar ana


Jawab:

Uang yg harus dibayar Ana ialah Rp 53.000,00

Penjelasan dengan langkah-langkah:

Misalkan: 1 kg pear adalah x

1 kg alpukat adalah y

1 kg buah naga adalah z, maka didapatkan persamaan sebagai berikut:

2x + 1y + 2z = 91.000 (persamaan 1)

2x + 1y + 1z = 76.000 (persamaan 2)

2x + 3y + 2z = 121.000 (persamaan 3)

Perhitungannya bisa dilihat di lampiran:

Perhitungan 1: persamaan 1 dikurangi persamaan 2 (di kotak hijau)

Dari perhitungan 1, diketahui bahwa nilai 1z adalah 15.000

Perhitungan 2: persamaan 3 dikurangi persamaan 1 (di kotak biru)

Dari perhitungan 2, diketahui bahwa nilai 1y adalah 15.000

Perhitungan 3: mencari nilai x melalui persamaan 2, yaitu

2x + 1y + 1z = 76.000

2x + 15.000 + 15.000 = 76.000

2x + 30.000 = 76.000

2x = 76.000 - 30.000

2x = 46.000

x = 23.000

Maka 1 kg pear yaitu Rp 23.000,00

Maka 1 kg alpukat yaitu Rp 15.000,00

Maka 1 kg buah naga yaitu Rp 15.000,00

Jadi uang yg harus dibayar Ana = Rp 53.000,00

30. Anis membeli 2 kg salak, 1 kg jeruk, dan 2 kg alpukat dengan harga Rp 75.000. Nisa membeli 2 kg salak, 2 kg jeruk dan 1 kg alpukat dengan harga Rp 70.000. Sedangkan Sani membeli 2 kg jeruk dan 2 kg alpukat dan membayar Rp 70.000. Tentukan harga 1 kg salak! ​


Jawaban:

10.000

Penjelasan dengan langkah-langkah:

Misal : salak = a, jeruk = b, alpukat = c

Maka :

2a + b + 2c = 75.000 .... persamaan 1

2a + 2b + c = 70.000 .... persamaan 2

2b + 2c = 70.000 .... persamaan 3

Eliminasi a dari persaman 1 dan 2

2a + b + 2c = 75.000

2a + 2b + c = 70.000

________________ -

- b + c = 5.000 .... persamaan 4

Eliminasi b dari persamaan 3 dan 4

2b + 2c = 70.000 |×1| 2b + 2c = 70.000

-b + c = 5.000 |×2| - 2b + 2c = 10.000

_____________________________ +

4c = 80.000

c = 80.000/4

c = 20.000

Substitusi c ke persamaan 3

2b + 2c = 70.000

2b + 2(20.000) = 70.000

2b + 40.000 = 70.000

2b = 70.000 - 40.000

2b = 30.000

b = 30.000/2

b = 15.000

Substitusi b dan c ke persamaan 1

2a + b + 2c = 75.000

2a + 15.000 + 2(20.000) = 75.000

2a + 15.000 + 40.000 = 75.000

2a + 55.000 = 75.000

2a = 75.000 - 55.000

2a = 20.000

a = 20.000/2

a = 10.000

Jadi, harga 1 kg salak adalah 10.000.
