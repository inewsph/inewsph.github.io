11. Sebutkan perbedaan format gambar GIF, PNG, JPEG, dan SVG​


Jawaban:

perbedaan format pada image karena penggunaannya berbeda-beda

Penjelasan:

GIF : format gambar yang bisa dianimasikan (bergerak). Mode gambar RGB sehingga penggunaannya untuk ditampilkan dilayar saja (bukan format cetak). Beberapa jenis tidak bergerak (statis), bisa memiliki backround transparan atau tidak. Berlisensi untuk penggunaannya

PNG : hampir sama seperti GIF namun tidak bisa dianimasikan (bergerak) dan tidak berlisensi. Mode gambar RGB untuk penggunaan di layar monitor (bukan format cetak). Paling sering digunakan karena memiliki background yang transparant, beberapa jenis tidak memiliki background transparant.

JPEG/JPG : Format standart image yang paling banyak dipakai. Mode gambar RGB untuk penggunaan di layar monitor, dan Mode gambar CMYK untuk keperluan cetak.

SVG : format gambar vector yaitu berupa pola garis (outline) dan isi warna (fill). Format gambar untuk keperluan editing ulang. Memiliki background transparant

NB: semua format diatas bertipe BITMAP kecuali SVG bertipe VECTOR

Jawaban:

- format gambar GIF (Graphical Interchange Format)

Merupakan format paling sesuai untuk menyimpan gambar sederhana (seperti garis & bentuk dasar) yang tak mengandung banyak warna/ gradasi.

- format gambar PNG (Portable Network Grahic)

Merupakan format yang mampu menyimpan gambar dengan transparansi (misalnya background transparan) atau bagian gambar yang semi transparan.

- format JPEG (Joint Photographic Expert Group)

Merupakan format paling tepat dipakai untuk file foto dan gambar yang memiliki kompleksitas warna tinggi.

- format SVG (Scalable Vector Graphic)

Merupakan format untuk menyimpan file gambar berbasis vektor.
