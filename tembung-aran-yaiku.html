2. Tembung linggane diarani yaiku


Tembung linggane diarani yaiku tembung sing durung owah saka asale (kata asal/kata dasar),  sing durung oleh imbuhan apa-apa, lan tembung kang isih wungkul, isih wantah utawa isih asli.

Pembahasan  

Tembung Lingga (kata dasar)

Tembung lingga digolongake wujud bebas (bentuk bebas). Tembung lingga ana kang dumadi saka sakwanda, rong wanda, utawa telung wanda (suku kata).

Tuladha rong wanda : Ciwit, kursi, meja, deleng, dolan, laku, turu, tandur, cekel, adus, babar,  cokot, tuku, tandhur, lan sak panunggalane.

Tembung lingga sing mung sekecap utawa sak wanda diarani karo tembung wod.

Tuladha : dom, seng, sak, pring, bak, rak, ler, lur, tul, nul, tik, lan sak panunggalane.

16. kang diarani tembung saroja yaiku


           Tembung Saroja, yaiku tembung kang kadiri saking loro tembung, banjur digabung lan saged nuwuhake pangerten ingkang luwih teges.

Conto – contone Tembung Saroja, Tegese, lan Tuladhane :

1.    Abang Mbranang, nduweni teges yaiku Abang Banget utawa abang nemen.

Tuladha : Bocah iku nesu ngasi raine abang mbranang.

2.    Adhem Ayem, nduweni teges yaiku uripe tansah kepenak, tentrem, ugi makmur.

Tuladha : Urip ing ndeso pancen nggarai urip adhem ayem.

3.    Peteng Ndedet, nduweni teges yaiku peteng banget utawa gelap gulita.

Tuladha : Sakiki jam 5 sore suasanane wis peteng ndedet.

4.    Gemah Ripah, nduweni teges yaiku subur.

Tuladha : Sawahe bapakku gemah ripah loh jinawi.

5.    Bapa Biyung, nduweni teges yaiku Wong tua, bapak, ibu, orangtua

Tuladha : Dadi anak kudu ngurmati bapa biyung.

6.    Dowo Tangane, nduweni teges yaiku seneng maling.

Tuladha : Dadi wong sing becik, ojo dadi wong sing dowo tangane.

7.    Padhang Jingglang, nduweni teges yaiku terang banget ora ono peteng – petenge blas.

Tuladha : Tangi – tangi wis padhang jingglang.

8.    Ngalap Berkah, nduweni teges yaiku ngarepake berkah

Tuladha : Ora becik ngalap berkah ing kuburan.

9.    Raja Kaya, nduweni teges yaiku kewan ingon ingon kaya sapi, kebo, jaran, lsp

Tuladha : Pakdheku sakiki ingon – ingon raja kaya.

10. Tahu tempe, nduweni teges yaiku tahu dan tempe.

Tuladha : Kasenenganku namung mangan tahu tempe.

20. tembung aran ono loro yaiku​


Jawaban:

Jenis Tembung Aran

Tembung aran dibedakan menjadi beberapa jenis, yaitu :

Tembung Kang wujud

Tembung Kang ora ono wujude

Tembung Saka barang kang nduweni nyawa

Tembung Aran Kang Wujud

Tembung aran kang wujud yaiku tembung kang bisa ditampa nganggo panca indera. Sing masuk jenis aran kang wujud umpamane :

Tuladha tembung : meja, kursi, koran gunung, lintang, lemah, kali, pot, buku, papan tulis, duit, omah.

Tembung Aran Kang Ora Ono Wujude

Tembung aran ora maujud yaiku tembung kang ora ditampa karo panca indera. Tuladha : ngelmu, kepinteran, pagaweyan, kabecikan, katresnanan.

Penjelasan:

Pengertian Tembung Aran

Tembung aran yaiku arane tembung sing nerangake sekabehane barang kang ono wujude utawa barang kang ora ono wujude.

Ing sajroning tatanan ukara, tembung aran yaiku tembung kang iso dadi jejer lan tembung kang iso dadi lesan.

23. Panganan sehat yaiku panganan sing ngandhut gizi sing diarani papat sehat lima sempurna tembung panganan lan diarani ,yaiku tembung sing oleh wuwuhan


Wangsulan

➡️ Tembung 'panganan' lan 'diarani' oleh wuwuhan ater-ater lan panambang.

Panganan => pangan + panambang -an.Diarani => ater-ater di- + aran + panambang -i

Penjelasan

{Wuwuhan}

1. Ater-ater (Awalan)

→ Ater-ater yaiku wuwuhan sing manggone ana ing sangarepe tembung. Ater-ater iku sabenere ana akeh maceme, ning ing kene mung dirembug sing kerep digunakake kaya dene ater-ater anuswara (n-, m-, ng-, ny-), ater-ater tripurusa (dak-, ko-, di-), ka-, sa-, lan pi-.

» Tuladha :

tandur + n- → nandurwaca + m- → macakopi + ng- → ngopicakot + ny- → nyakotjupuk + dak- → dakjupuktutup + ko- → kotutuptabrak + di- → ditabrakgawe + ka- → kagawelusin + sa- → salusinwales + pi- → piwales

2. Seselan (Sisipan)

→ Seselan yaiku wuwuhan sing manggon ing tengahe tembung. Seselan iku ana papat ing antarane yaiku –in-, -um-, -er-, lan -el-

» Tuladha :

gayuh + -in- → ginayuhgedhe + -um- →gumedhekelip + -er- → kerelipkepyur + -el- → kelepyur

3. Panambang (Akhiran)

→ Panambang yaiku wuwuhan sing manggon ana ing mburine tembung. Panambang iku sabenere ana akeh maceme, ning ing kene mung dirembug sing kerep digunakake kaya dene -ku, -mu, -e, -an, -i, -a, -na, -ana, -en, lan -ake.

» Tuladha :

sepatu + -ku → sepatukukursi + -mu → kursimusega + -e → segane (*dudu segae)tulis + -an → tulisansuwek + -i → suwekituku + -a → tukuasapu + -na → sapokna (*dudu sapuna)tali + -ana → talenana (*dudu taliana)garap + -en → garapengawa + -ake → gawakake (*dudu gawaake)
