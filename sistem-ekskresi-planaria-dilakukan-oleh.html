1. Bagaimana sistem ekskresi pada Planaria ?


planaria mempunyai organ nefridium yang disebut protonefridium, air keluar melewati nefridium. Cacing pipih mempunyai organ nefridium yang disebut sebagai protonefridium. Protonefridium tersusun dari tabung dengan ujung membesar mengandung silia. Di dalam protonefridium terdapat sel api yang dilengkapi dengan silia.
Tiap sel api mempunyai beberapa flagela yang gerakannya seperti gerakan api lilin. Air dan beberapa zat sisa ditarik ke dalam sel api. Gerakan flagela juga berfungsi mengatur arus dan menggerakan air ke sel api pada sepanjang saluran ekskresi. Pada tempat tertentu, saluran bercabang menjadi pembuluh ekskresi yang terbuka sebagai lubang di permukaan tubuh (nefridiofora). Air dikeluarkan lewat lubang nefridiofora ini.
Sebagian besar sisa nitrogen tidak masuk dalam saluran ekskresi. Sisa nitrogen lewat dari sel ke sistem pencernaan dan diekskresikan lewat mulut. Beberapa zat sisa berdifusi secara langsung dari sel ke air.

3. sistem ekskresi pda planaria


dilakukan melalui pembuluh bercabang-cabang yang memanjang pada bagian samping kiri dan kanan di sepanjang tubuhnya. 


Setiap cabang berakhir pada sel-sel api (solenosit) yang dilengkapi dengan silia (bulu getar). Saluran ini disebut protonefridium (proto=sebelum, nephros= ginjal). Silia pada setiap sel api akan selalu bergerak. 

Akibat gerakan silia tersebut, air atau cairan tubuh dan zat sisa yang sudah disaring di dalam sel api akan terdorong masuk ke dalam saluran ekskresi. Dari saluran ekskresi, cairan tubuh dan zat sisa kemudian dikeluarkan dari tubuh melalui suatu lubang yang disebut nefridiofor.


4. jelaskan sistem ekskresi pada cacing planaria


kategori : biologi - ekskresi
kelas : sma
pembahasan :

Cacing planaria merupakan salah satu contoh spesies dari filum platyhelmintes atau filum cacing pipih dari kelas turbellaria atau kelas cacing dengan rambut getar.

 

Karena cacing ini merupakan hewan invertebrate sederhana, maka system ekskresi pada cacing planaria tidak memakai ginjal seperti hewan vertebrata lainnya.

 

Namun Sistem ekskresi pada cacing planaria menggunakan organ khusus yaitu sel api atau flame cell.

 

Sel api atau flame cell ini merupakan alat ekskresi yang paling sederhana yang dimiliki oleh hewan.

 

Fungsinya adalah untuk menarik cairan tubuh yang berisi zat sisa dan kemudian membuangnya keluar dari tubuh.

 

Jumlah sel api atau flame cell pada cacing planaria bisa mencapai ratusan sel.

 

Sel api atau flame cell tersebut berjajar di daerah tepian kanan serta kiri dari tubuh cacing planaria yang kemudian  saling terhubung dengan tubulus atau pipa pipa pembuangan zat sisa ekskresi dan akhirnya bermuara pada lubang pengeluaran yang disebut nefridiofor.

 

Sel api atau flame cell ini dilengkapi dengan struktur khusus mirip rambut atau silia yang kemudian akan bergerak secara ritmik dan terus menerus yang kemudian menimbulkan efek tarikan terhadap cairan tubuh di sekitar sel api tersebut untuk kemudian masuk ke dalam sel api.

 

Cairan tubuh tersebut mengandung zat sisa yang harus dikeluarkan dari tubuh melalui proses ekskresi.

 

Maka cairan tubuh tersebut akan dialirkan ke tubulus atau pipa pembuangan dan selanjutnya dialirkan sampai ke ujung tubulus yang berupa lubang pembuangan atau nefridiofor.
