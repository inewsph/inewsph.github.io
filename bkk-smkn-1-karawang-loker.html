7. pak Amir mendorong loker dengan usaha 900 j dengan denangan gay 600 N.Berapakah perpindahan Loker​


Jawaban:

1,5 Meter

Penjelasan:

Diketahui:

Usaha ( W ) = 900 Joule

Gaya ( F ) = 600 N

Ditanyakan:

Perpindahan ( S ) = ........?

Jawab ⤵️

W = F × S

900 = 600 × S

S = 900 : 600

S = 1,5 meter

______________________

W = Usaha ( Joule )

F = gaya ( Newton )

S = perpindahan ( Meter )
