13. Sebut dan jelaskan beberapa format file pada program adobe photoshop


Beberapa format file pada program Adobe Photoshop adalah sebagai berikut


- PSD,PDD (Photoshop Document). Merupakan format penyimpanan asli dari Photoshop

- PSB (Photoshop Big). Digunakan untuk meningkatkan resolusi maksimum

- BMP, RLE, DIB. Digunakan untuk menyimpan gambar tipe bitmap

- GIF. Digunakan untuk menyimpan gambar tipe GIF. Memiliki kombinasi 256 warna. Biasa digunakan untuk logo, line drawing dan icon

- EPS (Encapsutaed Potscript). Digunakan untuk pencetakan atau pertukaran dokumen antar program grafis

- IFF (Interchange File). Digunakan untuk video

- JPEG. Digunakan untuk menyimpan mode warna RGB, CYMK, Grayscale

- JPEG 2000 (JPF,JPX,JP2,J2C,J2K,JPC). Merupakan standar kompresi untuk meningkatkan rasio kualitas dan kompresi

- JPS (JPEG Stereo). Terdiri dari side by side berdasarkan JPEG

- MPO (Multi Picture Format). Digunakan untuk format 3D

- PCX (Personal Computer Exchange). Berbentuk rester.

- Photoshop PDF. Digunakan oleh Adobe Acrobat

- RAW. Digunakan untuk menyimpan warna RGB, CMYK dan Grayscale

- PXR (Pixar). Digunakan untuk pertukaran file dengan Pixar Image Computer

- PNG (Portable Network Graphics). Digunakan untuk transparasi dalam browser

- Portable Bit Map (PBM,PGM,PPM,PNM,PFM,PAM). Digunakan untuk menyimpan informasi dengan kualitas 1 bit hingga 24 bit

- Scitex CT (SCT). Digunakan untuk image processing pada komputer Scitex

- Targa (TGA,VDA,ICB,VST)

- Tagged Image Format File (TIF,TIFF). Digunakan untuk pertukaran file antara aplikasi dan platform komputer.

30. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.
