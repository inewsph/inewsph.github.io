= 7! / 3!

= 5.040 / 6

= 840 Susunan

> Temen Total huruf = 5Unsur ganda = 2!

= 5! / 2!

= (5.4.3.2.1) / (2.1)

= 120 / 2

= 60 Susunan

> Kokjadi Total huruf = 7Unsur ganda = 2!

= (7.6.5.4.3.2.1) / (2.1)

= 5.040 / 2

= 2.520 Susunan

> Demen Total huruf = 5Unsur ganda = 2!

= 5! / 2!

= (5.4.3.2.1) / (2.1)

= 120 / 2

= 60 Susunan
