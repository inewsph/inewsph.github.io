2,5 rim = 500 x 2,5 = 1250 lembar

1250 : 25 = 50

Jadi, tiap majalah tersebut ada 50 lembar

27. Langkah langkah cara membuat layout text seprerti karan atau majalah


1. Buka fie dokumen Microsoft Word anda yang sudah ada artikelnya.

2. Selanjutnya yaitu membuat tulisan pada artikel menjadi dua kolom caranya yaitu Pilih tulisan anda dari kata pertama sampai kata terakhir, lalu pilih Tab Menu Page Layout -> Columns -> Two.

3. Maka tulisan artikel anda akan menjadi dua kolom.  Selanjutnya yaitu membuat kata awal hurufnya menjadi besar.  Caranya yaitu pilih Tab Menu Insert -> Drop Cap.

4. Selanjutnya silahkan anda masukkan sebuah gambar dengan cara pilih menu INSERT -> PICTURE.  Lalu setelah gambar sudah berada di artikel anda silahkan pilih Tab Manu Format ->Position -> With Text Wrapping.
