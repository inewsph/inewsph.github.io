4. luas segidelapan beraturan adalah 32 akar 2 cm". kelilingnya adalah..


berarti luas sebuah segitiga (dari segi8) adlah :
L = 32√2 / 8 = 4√2
sudut tiap sgitiga pada sgi-8 = 360/8 = 45 derajat, akibatnya :
1/2 r² sin(45) = 4√2
1/2 r² 1/2 √2 = 4√2
r² = 16
r = 4
misal panjang setiap sisi sgi-8, adalah x. maka dg menggunakan aturan kosinus berlaku :
x² = r² + r² - 2r² cos45
x² = 2r² - 2r² (1/2 √2)
x² = 2r² -  r² √2
x² = 2(16) - 16√2
x² = 16(2-√2)
x = √(16(2 - √2))
x = 4√(2-√2)

jadi, keliling segi-8 = 8 (4√(2-√2)) = 32√(2-√2)

Luas segi delapan beraturan

• Jika diketahui sisi - sisinya

L = 2s²(1 + √2)

• Jika diketahui jari - jarinya

L = 2r²√2

maka :

Jika diketahui jari - jarinya adalah 12 cm, maka luas segi delapan beraturan tersebut adalah

L = 2r²√2

L = 2(12)²√2

L = 2(144)√2

L = 288√2 cm²
