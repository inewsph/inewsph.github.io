8. RENSHUUJawablah pertanyaan berikut!1. Q: Asa gohan o tabemasuka. A:...2. Q: Hiru juusu o nomimasuka. A:...3. Q: Yoru nani o tabemasuka. A:...4. Q: Asa nani o nomimasuka. A:...5. Q: Yoru koohii o nomimasuka. A:...​


1. Q : Asa gohan o tabemasuka.

A : Hai, asa wa gohan o tabemasu (はい, 朝はごはんを食べます)

2. Q : Hiru juusu o nomimasuka.

A : Hai, hiru wa juusu o nomimasu.(はい、昼はジュースを飲みます)

3. Q : Yoru nani o tabemasuka.

A : Yoru wa o sushi o tabemasu.(夜はお寿司を食べます)

4. Q : Asa nani o nomimasuka.

A : Asa wa o koohii o nomimasu.(朝はをコーヒーを飲みます)

5. Q : Yoru koohii o nomimasuka.

A : iie, yoru wa koohii o nomimasen.(いいえ、夜はコーヒーを飲みません)

Pembahasan

Asa gohan o tabemasuka (朝ごはんを食べますか) berarti Apa kamu makan nasi di pagi hari?

__________________________

朝 (Asa) berarti pagi, ごはん (gohan) berarti nasi, dan 食べ (tabe) adalah kata dasar dari taberu (食べる) yang artinya makan. Partikel o (を) berfungsi sebagai partikel objek yaitu nasi (gohan) dan partikel -masuka (ますか) berfungsi sebagai pemberi kesan pertanyaan pada suatu kerja yang sedang dilakukan atau akan dilakukan. Jadi, arti dari Asa gohan o tabemasuka adalah Apa kamu makan nasi di pagi hari?. Sehingga, jika memang makan nasi di pagi hari jawabannya adalah Hai, asa wa gohan o tabemasu (Ya, di pagi hari aku makan nasi) dan jika tidak jawabannya adalah iie, asa wa gohan o tabemasen (Tidak, aku tidak makan nasi di pagi hari).

Hiru juusu o nomimasuka (昼ジュースを飲みますか) berarti Apa kamu minum jus di siang hari?

_____________________________

昼 (Hiru) berarti siang, ジュース (Juusu) berarti jus, dan 飲み (nomi) adalah peleburan dari 飲む (nomu) yang artinya minum. Partikel o (を) berfungsi sebagai partikel objek yaitu juusu (jus) dan partikel -masuka (ますか) berfungsi sebagai pemberi kesan pertanyaan pada suatu kerja yang sedang dilakukan atau akan dilakukan. Jadi, arti dari Hiru juusu o nomimasuka adalah apa kamu minum jus di siang hari?. Sehingga, jika minum jawabannya adalah Hai, hiru wa juusu o nomimasu (Iya, aku minum jus di siang hari) dan jika tidak jawabannya iie, horu wa juusu o nomimasen (Tidak, aku tidak minum jus di siang hari).

Yoru nani o tabemasuka (夜何をたべますか) berarti Apa yang kamu makan di malam hari?

_____________________________

夜 (Yoru) berarti malam, 何 (nani) berarti tidak, dan 食べ (tabe) adalah kata dasar dari taberu (食べる) yang artinya makan. Partikel o (を) berfungsi sebagai partikel objek dan partikel -masuka (ますか) berfungsi sebagai pemberi kesan pertanyaan pada suatu kerja yang sedang dilakukan atau akan dilakukan. Jadi, arti dari Yoru nani o tabemasuka adalah Apa yang kamu makan di malam hari?. Sehingga, saya sesuaikan misalnya Saya makan Sushi maka jawabannya Yoru wa o sushi o tabemasu (Di malam hari, saya makan sushi)

Asa nani o nomimasuka (朝何を飲みますか) berarti Apa yang kamu minum di pagi hari?

_____________________________

朝 (Asa) berarti pagi, 何 (nani) berarti apa, dan 飲み (nomi) adalah peleburan dari 飲む (nomu) yang artinya minum. Partikel o (を) berfungsi sebagai partikel objek dan partikel -masuka (ますか) berfungsi sebagai pemberi kesan pertanyaan pada suatu kerja yang sedang dilakukan atau akan dilakukan. Jadi, arti dari Asa nani o nomimasuka adalah Apa yang kamu minum di pagi hari?. Sehingga, saya sesuaikan misalnya Saya minum kopi biar nggak ngantuk di sekolah maka jawabannya Asa wa o koohii o nomimasu (Di pagi hari, saya minum kopi.

Yoru koohii o nomimasuka (夜コーヒーを飲みますか) berarti Apa kamu minum kopi di malam hari?

_____________________________

夜 (Yoru) berarti malam, コーヒー (Koohii) berarti kopi dan 飲み (nomi) adalah peleburan dari 飲む (nomu) yang artinya minum. Partikel o (を) berfungsi sebagai partikel objek yaitu koohii (kopi) dan partikel -masuka (ますか) berfungsi sebagai pemberi kesan pertanyaan pada suatu kerja yang sedang dilakukan atau akan dilakukan. Jadi, arti dari Yoru koohii o nomimasuka adalah Apa kamu minum kopi di malam hari? Kalau saya sih nggak karena malah nanti nggak bisa tidur. Saya kasih alternatif, jika minum jawabannya adalah Hai, yoru wa koohii o nomimasu (Ya, saya minum kopi di malam hari) dan jika tidak maka jawabannya iie, yoru wa koohii o nomimasen (Tidak, saya tidak minum kopi di malam hari).

30. 4. Susunlah kata-kata berikut agar menjadi kalimat yang benar! Tomodachi/o/hiru-gohan/to/tabemasu/yoku​


Jawaban :

Susunan kata-kata tersebut agar menjadi kalimat yang benar adalah Yoku tomodachi to hirugohan o tabemasu. Kalimat ini dalam Bahasa Indonesia artinya "(Saya) sering (makan) makan siang dengan teman".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan

Struktur kalimat dalam Bahasa Jepang sedikit berbeda dengan Bahasa Indonesia. Urutan struktur kalimat dalam Bahasa Jepang bisa bermacam-macam. Dalam kalimat Bahasa Jepang, predikat selalu diletakkan di akhir.

Dalam kalimat Bahasa Jepang juga sering terdapat kata bantu atau biasa disebut partikel. Partikel ini jumlahnya ada banyak dan masing-masing memiliki fungsinya tersendiri.

-

PEMBAHASAN SOAL

Urutan struktur kalimat :

S ➞ K ➞ O ➞ P

S : SubjekK : KeteranganO : ObjekP : Predikat

*Orang Jepang jarang menyebutkan subjek jika subjeknya sudah diketahui oleh lawan bicara, jadi bagian S (Subjek) bisa dihilangkan.

Kosakata :

Tomodachi = TemanHirugohan = Makan siangTabemasu = MakanYoku = Sering

Partikel :

Partikel O : Untuk menandakan objek. Biasa diletakkan di antara kata benda atau objek dengan kata kerja.Partikel To : Untuk menunjukkan kata yang dimaksud sebagai lawan yang diajak. Kalau diartikan seperti "Dengan" atau "Bersama". Misalkan pada kalimat 'A to', maka artinya seperti "Dengan A" atau "Bersama A".

===

Untuk membuat kalimatnya, dimulai dari subjek. Tapi karena subjeknya tidak ada, maka langsung ke keterangannya. Yang berperan sebagai keterangannya adalah Yoku (sering) dan Tomodachi to (Dengan teman). Kemudian untuk objek, yaitu Hirugohan (makan siang). Dan predikatnya adalah Tabemasu (makan).

Jika disusun menjadi kalimat, akan menjadi :

=> Yoku tomodachi to hirugohan o tabemasu.

=> Sering makan siang bersama teman.

===
