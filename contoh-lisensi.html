1. contoh standar lisensi dalam industri ?


Jawaban:

lisensi artinya izin

Penjelasan:

pemberian izin dari pemilik barang atau jada kepada pihak yang menerima lisensi untuk menggunakan barang atau menjualnya

Lisensi adalah izin yang diberikan oleh pemilik merek terdaftar kepada pihak lain melalui suatu perjanjian berdasarkan pada pemberian hak (bukan pengalihan hak) untuk menggunakan merek tersebut, baik untuk seluruh atau sebagian jenis barang dan/atau jasa yang didaftarkan dalam jangka waktu dan syarat tertentu.

Contoh lisensi:
lisensi perangkat lunak (software) komputer misalnya linsensi windows, lisensi program anti virus, lisensi microsoft office dll.

2. pengertian dari program lisensi adalah ??dan apa saja contohnya ??


Lisensi Program atau software license adalah hak yang biasanya dibeli seseorang agar dapat menggunakan aplikasi buatan orang yang dibayar. contohnya adalah Microsoft Office dimana kita harus membayar agar bisa mendapatkan full featurenya dan bukan free trial. Contoh lainnya adalah sistem operasi windows yang mengharuskan kita membayar agar dapat menggunakan software tersebut.

Penjelasan:

1.lisensi commercial

biasa ditemui pada perangkat lunak seperti ms.. dgn windows dan office ny, lotus, oracle dll..

untuk kepentingan commercial.

2.trial software

biasa ditemui pd software sebelum diluncurkan pd masyarakat, atau sudah diluncurkan tetapi memiliki batas masa aktif.

3.non commercial use

biasanya untuk kalangan pendidikan atau yayasan tertentu dibidang sosial.

4.shareware

mengizinkan pemakainy untuk menggunakan, menyalin atau menggandakan tanpa harus meminta izin pemegang hak cipta.

5.freeware

biasanya pada software yang bersifat mendukung, memberikan fasilotas tambahan atau memang free/gratis.

6.royalty-free binaries

serupa dgn freeware hanya saja produk yg ditawarkan adalah library yg berfungsi melengkapi software yg sudah ada dan bukan suatu software yang berdiri sendiri.

7.open source

membebaskan userny untuk menjalankan, menggandakan, menyebarluaskan, mepelajari, mengubahny, dan meningkatkan kinerja software.

24. Apa perbedaan dari lisensi komersial lisensi freeware dan lisensi open source​


Jawaban:

Commercial software adalah software yang dibuat dan dikembangkan oleh perusahaan dengan konsep bisnis, dibutuhkan proses pembelian atau sewa untuk bisa menggunakan software tersebut.

Freeware adalah software yang diizinkan untuk digunakan atau disebarluaskan namun tidak memiliki izin untuk dimodifikasi.

Open Source adalah software yang dapat dilihat kode sumbernya. perlu diketahui software open source bukan berarti software gratis.

Penjelasan:

Lisensi Trial. Maksud dari Trial ini adalah versi demo atau bisa dibilang versi uji coba dari software. Jadi pada software yang menggunakan lisensi trial ini, calon pembeli ataupun pengguna akan mendapatkan software yang masih dalam bentuk demo. Tujuan dibuatkan demo ini agar pengguna dapat merasakan terlebih dahulu pelayanan yang ada pada software atau aplikasi tersebut.
