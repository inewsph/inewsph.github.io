Kita cari sudut apit antara vektor C dan D. 
α = 90+30 = 120°
a. C+D = √C²+D²+2.C.D cos 120
= √4²+5²+2.4.5.-1/2
= √16+25-20 = √21

b. C-D = √C²+D²-2.C.D.cos 120

6. Bagaimana cara menggunakan rumus kosinus dalam menentukan besar vektor resultan


Untuk hal tersebut dapat menggunakan rumus :

[tex]R = \sqrt{ a^{2} + b^{2} + 2. a.b. cos \alpha } [/tex]

Sedangkan untuk selisih vektor menggunakan rumus :

[tex]R = \sqrt{ a^{2} + b^{2} - 2. a.b. cos \alpha } [/tex]
untuk mencari resultan vektor pada kosinus  , bisa di gunakan dengan cara : 

R = √x² + y² + 2xy. cos α

R = resultan vektorF₁ = vektor pertamaF₂ = vektor keduaα = sudut apit antara kedua vektor

________________________________

Diketahui :

[tex]F_1 = 6 \: N[/tex][tex]F_2 = 8 \: N [/tex]α = 90 °

Ditanya :

R ... ?

Dijawab :

[tex]R =\sqrt{{F_1}^{2} + {F_2}^{2} + 2 F_1F_2 \: \: Cos \: α} [/tex]

[tex]R = \sqrt{{6}^{2} + {8}^{2} + 2 \: \: (6 \: . \: 8)\: \: Cos \:90° } [/tex]

[tex]R = \sqrt{36 + 64 + 2 \: \: (48)\: \: \: 0 } [/tex]

[tex]R = \sqrt{100+ 0 \: } [/tex]

[tex]R = \sqrt{100} [/tex]

[tex]{\boxed {R = 10\: N}}[/tex]

jadi, maka resultan kedua vektor tersebut adalah 10 N
