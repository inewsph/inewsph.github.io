6. Umur andi ditambah umur budi asalah 38 tahun . Umur budi ditambah umur chandra adalah 44 tahun . Umur chandra ditambah umur andi adalah 42 tahun . Jumlah umur andi , budi dan chandra adalah​


Diketahui:

Andi + Budi = 38 tahun.

Budi + Chandra = 44 tahun.

Andi + Chandra = 42 tahun.

Ditanya:

Andi + Budi + Chandra = ?

Jawaban:

62 tahun.

Penjelasan dengan langkah-langkah:

Misalnya:

Andi = A.

Budi = B.

Chandra = C.

A + B = 38.

B + C = 44.

A + C = 42.

Caranya menggunakan metode campuran.

A = 38 - B

B + C = 44

A + C = 42

B + C = 44

38 - B + C = 42

B + C = 44

-B + C = 4

2C = 48

C = 24 tahun.

B + 24 = 44

B = 20 tahun.

A = 38 - 20

A = 18 tahun.

Jadi:

18 + 20 + 24 = 62 tahun.
