PENJELASAN

=> CONDITIONAL SENTENCE TYPE 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan terjadi di masa depan. Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang. Rumus dari kalimat ini adalah: if + simple present, simple future “will”.

=> CONDITIONAL SENTENCE TYPE 2

Second conditional sentences merupakan tipe kalimat pengandaian yang digunakan ketika hasil atau konsekuensi hanya memiliki harapan terwujud walaupun kemungkinannya sangat kecil. Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive. Penggunaan would/could/might berfungsi untuk menjelaskan seberapa besar sebuah kondisi itu akan terjadi.

=> CONDITIONAL SENTENCES TYPE 3

Third conditional sentence merupakan sebuah kalimat pengandaian ketika sebuah kondisi tidak mungkin terwujud sama sekali. Dalam tipe kalimat pengandaian yang ketiga ini, bentuk kalimat menggunaka past perfect yang dilengkapi dengan modal auxiliary seperti would, could, dan should. Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

9. Buatlah kalimat promosi makanan dengan menggunakan conditional sentence type 0 dan type 1


Conditional sentence juga dikenal sebagai kalimat kondisional. Dalam Bahasa Inggris conditional sentence adalah kalimat pengandaian. Ada 4 jenis conditional sentence yaitu type 0, 1, 2, dan 3. Berikut akan kita bahas secara singkat 4 jenis tersebut.


Zero conditional (conditional sentence type 0)

Conditional sentence type 0 adalah kalimat kondisional yang digunakan untuk menjelaskan situasi general. Contoh: If water reaches 100 degrees, it boils. Selain itu, kita juga dapat menggunakan grammar ini untuk menjelaskan situasi yang sudah pasti. Contoh: If I eat too much, I get fat.

Berikut susunan untuk membentuk conditional sentence type 0:

If + present simple, (...) present simple.



First conditional (conditional sentence type 1)

Conditional sentence type 1 adalah kalimat kondisional yang digunakan untuk menjelaskan situasi spesifik. Contoh: If it rains, I won't join the BBQ party at the park.

Berikut susunan membentuk conditional sentence type 1:

If + present simple, (...) will + infinitive.


Second conditional (conditional sentence type 2)

Conditional sentence type 2 adalah kalimat kondisional yang digunakan untuk menjelaskan hypothetical situation yang merupakan situasi yang belum tentu terjadi. Kita bisa menggunakan kalimat jenis ini untuk menjelaskan soal mimpi kita di masa depan yang bergantung dengan situasi tertentu. Contoh: If I became a millionaire, I would travel around the world.

Berikut susunan untuk membentuk conditional sentence type 2:

If + past simple, (...) would+infinitive.


Third conditional (conditional sentence type 3)

Conditional sentence type 3 adalah kalimat yang digunakan untuk menyampaikan penyesalan atau situasi masa lalu yang ingin kita ubah namun tidak dapat kita ubah. Contoh: If I had taken a taxi, I wouldn't have been late for the party.

Berikut susunan untuk membentuk conditional sentence type 3:

if + past perfect, (...) would + have + past participle.


Contoh kalimat promosi makanan dengan menggunakan conditional sentence 0 dan 1.


Conditional sentence 0:

1. If it's hot today, buy our newly released apple juice. It can quench your thirst.

2. Are you hungry? If you're hungry, try our high quality steak to satisfy your hunger!

Conditional sentence 1:

1. If it's hot today, our newly released apple juice will quench your thirst.

21. "If I hold down the fn + f5 key, it will reduce the brightness of the laptop screen."1. Conditional Sentence Type 0Answer : 2. Conditional Sentence Type 1Answer : 3. Conditional Sentence Type 2Answer : Fact : 4. Conditional Sentence Type 3Answer : Fact : ​


CONDITIONAL SENTENCE

1. If I hold down the fn + f5 key, it reduces the brightness of the laptop screen. (C.S. type O)

2. If I hold down the fn + f5 key, it will reduce the brightness of the laptop screen. (C.S. type I)

3. If I held down the fn + f5 key, it would reduce the brightness of the laptop screen. (C.S. type II)

Fact : I don't hold down the fn + f5 key, so it doesn't reduce the brightness of the laptop screen.

4. If I had held down the fn + f5 key, it would have reduced the brightness of the laptop screen. (C.S. type III)

Fact : I didn't hold down the fn + f5 key, so it didn't reduce the brightness of the laptop screen.

EXPLANATION

Conditional sentence adalah kalimat andai², di mana suatu keinginan dapat terwujud apabila persyaratannya terpenuhi.

A. Conditional sentence type 0

↪menjabarkan kebenaran umum, selalu benar terjadi [Fact].

Formula :

If+Simple present, .....simple present

If+S+verb¹(s/es)..., S+verb¹(s/es)

Exp.

If the water is heated at 100 degrees Celcius, it boils.

B. Conditional sentence type I

↪menjabarkan keinginan yang kemungkinan terjadi saat ini ataupun nanti-nanti (real possibility) berdasarkan prediksi.

Formula :

If+Simple present, ....Simple future

If+S+verb1(s/es)..., S+will+verb1......

Exp.

If I have enough money, I will buy the new book.

C. Conditional sentence type II

↪menjabarkan keinginan yang diragukan terjadi saat ini atau mendatang (hypothetical situation). Pakai auxiliary BE "were" utk semua bentuk subyek sbg bentuk harapan yg blm tentu terwujud (Subjunctive).

Formula :

If+Simple past, ....Past future

If+S+verb2 ......, S+would+verb1......

Exp.

If I were you, I would tell it to your mom.

D. Conditional sentence type III

↪menjabarkan keinginan yang tidak terwujud di waktu lampau (truly hypothetical). Pengulasannya saat ini hanyalah sebagai dampak dari kejadian di waktu lampau.

Formula :

If+Past perfect,..... Past future perfect

If+S+had+verb3...., S+would+have+verb3....

Exp.

If he had kept the promise, his girl friend wouldn't have been disappointed.

28. perbedaan Conditional sentences type 0 and type 1? berilah rumus dan contoh nya​


Jawaban:

Conditional sentence type 0

Tipe ini biasanya disebut sebagai zero conditional sentences.

Tipe kalimat ini digunakan ketika hasil atau konsekuensi dari kondisi terwujud alias memaparkan sebuah kebeneran (general truth) dan fakta ilmiah.

Kalimat ini biasanya berbentuk present tense dengan rumus: if + simple present, simple present. Contoh kalimat (berdasarkan contoh dari grammarly.com):

If we burn paper, it becomes ash.

I feel sick if I eat too much.

If you don’t brush your teeth, you get cavities.

When people smoke cigarettes, their health suffers.

Conditional sentence type 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan untuk terjadi di masa depan.

Hal ini bisa terjadi karena masih ada kondisi realistik yang masuk akal untuk kemungkinan terjadi.

Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang.

Rumus dari kalimat ini adalah: if + simple present, simple future “will” / imperative dan contohnya adalah:

If I meet him, I will introduce myself.

I will cook dinner tonight if you clean the house.

If you rest, you will feel better.

If you set your mind to a goal, you will eventually achieve it.

If it rains, I will stay at home.

If he gives her chocolate, she will be happy.

If it doesn’t rain, we will go to the library.
