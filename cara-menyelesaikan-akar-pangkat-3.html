3. bagaimanakah cara penyelesaian akar pangkat 3​


Jawaban:

Pangkat tiga adalah mengalikan bilangan yang sama sebanyak tiga kali. Misalkan pangkat tiga dari bilangan 2 maka bentuk pangkat tiganya adalah 23 = 2 x 2 x 2 = 8. Begitu juga untuk bilangan lainnya. Untuk mencari nilai pangkat tiga dari suatu bilangan, sobat idschool hanya perlu mengalikan bilangan yang sama sebanyak tiga kali. Berapapun bilangan yang akan dicari nilai pangkat tiganya. Misalnya untuk bilangan 15, nilai pangkat tiga dari bilangan 15 adalah 3.375. Hal ini sesuai dengan perkalian bilangan 15 sebabnya 3 kali, yaitu 15 x 15 x 15 = 3.375.

18. Tentukan akar atau penyelesaian dari 0 = x pangkat 3 − 3x pangkat 2 − 18x + 40 point banyakk!!! (pakai cara)


x³ - 3x² - 18x + 40 = 0

x³ - 2x² - x² + 2x - 20x + 40 = 0

x²(x - 2) - x(x - 2) - 20(x - 2) = 0

(x - 2)(x² - x - 20) = 0

(x - 2)(x - 5)(x + 4) = 0

x = 2 ; x = 5 ; x = -4
