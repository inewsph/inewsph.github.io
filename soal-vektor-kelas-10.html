Vektor posisi AB = b - a (rumus)

Vektor posisi AB = b - a

(5, 3, 6)= b - (2, -1, 4)

b = (5, 3, 6) + (2, -1, 4)

b = ((5 + 2), (3 - 1), (6 + 4))

b = (7, 2, 10)

Jadi, koordinat titik B adalah (7, 2, 10)

10. Diket: k(1, 2, 0) + m(3, 1, 4) + n(-1, 0, -2) = (4, 4, 2)

Ditanya: k, m, dan n

Jawab:

(k, 2k, 0) + (3m, m, 4m) + (-n, 0, -2n) = (4, 4, 2)

(x, y, z), maka dapat dibentuk persamaan dengan menyatukan x dengan x, y dengan y, serta z dengan z. Lalu, akan diperoleh 3 persamaan:

k + 3m - n = 42k + m = 44m - 2n = 2

Lalu, dapat dilakukan eliminasi atau substitusi untuk mendapatkan masing-masing nilai k, m, dan n:

Eliminasi persamaan 1 dan 2 dengan persamaan 2 dikalikan 3:

(k + 3m - n = 4) - (3 (2k + m) = 4)

(k + 3m - n = 4) - (6k + 3m = 12) =

-3k - n = -8 (misalkan persamaan 4)

Eliminasi persamaan 1 dengan persamaan 2 dengan persamaan 1 dikalikan 2:

(2 (k - 3m - n = 4) - (2k + m = 4)

(2k - 6m - n = 8) - (2k + m = 4)

5m - 2n = 4 (persamaan 5)

Eliminasi persamaan 3 dan 5:

(4m - 2n = 2) - (5m - 2n = 4)

-m = -2

m = 2

Substitusi m = 2 ke persamaan 2:

2k + m = 4

2k + 2 = 4

2k = 2

k = 1

Substitusi m = 2 ke persamaan 3:

4m - 2n = 2

4(2) - 2n = 2

8 - 2n = -2

-2n = -6

n = 3

Jadi, nilai k = 1, m = 2, dan n = 3

2. Diketahui p = -12i + 37j dan q = 18i - 57j. Hasil dari p + q adalah -103i - 20j.

Jawaban: D

Pembahasan

Jika diketahui dua buah vektor pada dimensi 2 dinyatakan dalam vektor arah a = (x₁, y₁) dan vektor b = (x₂, y₂), maka operasi penjumlahan dan pengurangan vektor a dan vektor b dapat dilakukan sebagai berikut.

a + b = (x₁, y₁) + (x₂, y₂) = ((x₁ + x₂), (y₁ + y₂))a - b = (x₁, y₁) - (x₂, y₂) = ((x₁ - x₂), (y₁ - y₂))

Atau jika vektor dinyatakan dalam bentuk kombinasi linear, yaitu vektor arah a = x₁i + y₁j dan vektor b = x₂i + y₂j. Operasi penjumlahan dan pengurangan vektor a dan vektor b dapat dilakukan sebagai berikut.

a + b = (x₁ + x₂)i + (y₁ + y₂)ja - b = (x₁ - x₂)i - (y₁ - y₂)j

↓↓↓No. 1

Diketahui:

a = (-8, 13)

b = (12, 7)

Ditanyakan:

a + b = ...

Penyelesaian:

a + b = (-8, 13) + (12, 7)

a + b = (((-8) + 12), (13 + 7))

a + b = (4, 20)

Jadi hasil dari a + b adalah (4, 20).

No. 2

Diketahui:

p = -121i + 37j

q = 18i - 57j

Ditanyakan:

p + q = ...

Penyelesaian:

p + q = ((-121) + 18)i + (37 + (-57))j

p + q = -103i + (-20)j

p + q = -103i - 20j

Jadi hasil dari p + q adalah -103i - 20j.
