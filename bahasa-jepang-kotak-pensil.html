2. "Apakah itu (jauh) adalah kotak pensil?".Terjemahkan kalimat di atas ke dalam bahasa Jepang!​


"Apakah itu (jauh) adalah kotak pensil?" diterjemahkan ke dalam Bahasa Jepang adalah あれは ふでばこですか (Are wa fudebako desu ka).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

指示詞 (Shiji-shi)

Merupakan istilah umum untuk "Kata tunjuk" dalam Bahasa Jepang. Kata tunjuk ini bisa digunakan untuk menunjukkan berbagai macam hal, seperti benda, orang, tempat, dan lain sebagainya.

Dalam Bahasa Jepang, kata tunjuk dibedakan berdasarkan awalan huruf dari kata nya, yaitu [Ko-So-A-Do], yang perbedaannya adalah sebagai berikut.

Awalan Ko : Digunakan untuk menunjuk sesuatu yang dekat dengan pembicara. Contoh kata nya Kore, Kono, Koko, Kochira, dll. Artinya kebanyakan adalah "Ini" atau "Sini".

Awalan So : Digunakan untuk menunjuk sesuatu yang jauh dari pembicara, tapi dekat dengan lawan bicara. Contoh kata nya Sore, Sono, Soko, Sochira, dll. Artinya kebanyakan adalah "Itu" atau "Situ".

Awalan A : Digunakan untuk menunjuk sesuatu yang jauh dari pembicara juga lawan bicara. Contoh kata nya Are, Ano, Asoko, Achira, dll. Artinya kebanyakan adalah "Itu (jauh)" atau "Sana".

Awalan Do : Digunakan untuk menanyakan sesuatu yang ditunjuk lawan bicara. Contoh kata nya Dore, Dono, Doko, Dochira, dll. Artinya kebanyakan adalah "Yang mana" atau "Mana".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

これ・それ・あれ・どれ

[Kore / Sore / Are / Dore]

[Ini / Itu / Itu (jauh) / Yang mana]

Kata tunjuk ini digunakan untuk menunjukkan benda. Kata-kata ini dapat berdiri sendiri, tanpa harus menerangkan kata benda. Contoh-contoh kalimatnya sebagai berikut.

1. これは テレビです

= Kore wa terebi desu

= Ini adalah TV

2. それは かっこいいです

= Sore wa kakkoii desu

= Itu keren

3. あれは ふでばこです

= Are wa fudebako desu

= Itu adalah tempat pensil

4. どれが いいですか

= Dore ga ii desu ka

= Manakah yang bagus?

Are wa joko - san no kaban desu. 3. Sore wa Budi – san no jogi desu. 4. Sore wa Ditya - san no tokei desu. 5. Are wa Doni - san no fudebakko desu.{ kotak pensil }


Latihan soal A

1. Sore wa dare no kaban desu ka?

2. Sore wa watashi no kaban desu.

3. Sore wa Ali-san no dewa arimasen.

4. Kore wa Ali-san no desu ka?

5. Kore wa Santi-san no enpitsu desu.

Latihan soal B

1. Kore wa Tanaka-san no enpitsu dewa arimasen.

= Ini bukan pensil kepunyaan Tanaka.

2. Are wa Joko-san no kaban dewa arimasen.

= Itu bukan tas kepunyaan Joko.

3. Sore wa Budi-san no jougi dewa arimasen.

= Itu bukan penggaris kepunyaan Budi.

4. Sore wa Ditya-san no tokei dewa arimasen.

= Itu bukan jam kepunyaan Ditya.

5. Are wa Doni-san no fudebako dewa arimasen.

= Itu bukan kotak pensil kepunyaan Doni.

10. "Di atas meja kirana tidak ada kotak pensil dan buku pelajaran"terjemahan bahasa jepang nya adalah???


N5 Level A || Bab: 4

Pembahasan:
• Tata letak

•| R U M U S |•
↪ Kata benda 1 + の + posisi + に + kata benda 2

=== Jawaban ===

↪ テブルの上にキラナの筆箱や鉛筆がない (tebuu no ue ni kirana no fude bako ya enpitsu ga nai): Di atas meja tidak ada pensil dan tempat pensil Kirana ✔

❇ Penjelasan:
↪テブル (teburu): meja
↪の (no): partikel menyatakan kepemilikan.
↪上 (ue): di atas
↪に (ni): di
↪キラナ (kirana): Kirana. (Nama orang luar Jepang harus fitulis pakai katakana)
↪筆箱 (fudebako): tempat pensil
↪や (ya): dan
↪鉛筆 (enpitsu): pensil
↪が (ga): partikel penegas. Di sini berfungsi pengganti partkel "wa"
↪ない (nai): tidak ada. (Perubhaan dari arimasen)
