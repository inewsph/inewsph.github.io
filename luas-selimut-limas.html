8. apabila Limas persegi dengan panjang 10cm dan tinggi selimut/sisimiring 16cm berapakah luas permukaan Limas tersebut?​


Jawab:

Luas permukaan dari limas-limas berikut adalah

a. Luas permukaan = 180 cm²

b. Luas permukaan = 960 cm²

c. Luas permukaan = 336 cm²

d. Luas permukaan = 896 cm²

e. Luas permukaan = 84 cm²

f. Luas permukaan = 384 cm²

Pembahasan

Limas segi empat adalah limas yang alasnya berbentuk segi empat.

Ingat!!

Rumus Luas Permukaan Limas Segi Empat

LP = Luas alas + 4 x luas sisi tegak

a. Diketahui: alas limas berbentuk persegi

                  panjang sisi alas = s = 10 cm

                  tinggi segi tegak = 4 cm

Ditanya: berapa luas permukaan limas?

Jawab: LP = Luas alas + 4 x luas sisi tegak

             LP = (s x s) + 4 x (1/2 x alas x tinggi)

             LP = (10 cm x 10 cm) + 4 x (1/2 x 10 cm x 4 cm)

             LP = 100 cm² + 4 x (20 cm²)

             LP = 100 cm² + 80 cm² = 180 cm²

b. Diketahui: alas limas berbentuk persegi panjang

                  panjang alas = p = 30 cm

                  lebar alas = 16 cm

                  panjang sisi miring pada sisi tegak = 17 cm

Ditanya: berapa luas permukaan limas?

Jawab:

Karena ada 2 bentuk segitiga, maka kita cari terlebih dahulu tinggi sisi tegak limas pada masing-masing segitiga dengan teorema Pythagoras.

segitiga 1 (alas = 16 cm)

t1² = sisi miring² - (1/2 x alas)²

t1² = (17 cm)² - (1/2 x 16 cm)²

t1² = 289 cm² - 64 cm² = 225 cm²

t1 = \sqrt{225 cm^2} = 15 cm

segitiga 2 (alas = 30 cm)

t2² = sisi miring² - (1/2 x alas)²

t2² = (17 cm)² - (1/2 x 30 cm)²

t2² = 289 cm² - 225 cm² = 64 cm²

t2 = \sqrt{64 cm^2} = 8 cm

LP = Luas alas + (2 x luas sisi tegak 1) + (2 x luas sisi tegak 2)

LP = (p x l) + (2 x 1/2 x lebar x t1) + (2 x 1/2 x panjang x t2)

LP = (30 cm x 16 cm) + (2 x 1/2 x 16 cm x 15 cm) + (2 x 1/2 x 30 cm x 8 cm)

LP = 480 cm² + 240 cm² + 240 cm²

LP = 960 cm²

c. Diketahui: alas limas berbentuk persegi

                  panjang sisi alas = s = 12 cm

                  panjang sisi miring pada sisi tegak limas = 10 cm

 Ditanya: berapa luas permukaan limas?

 Jawab:

 Kita cari terlebih dahulu tinggi sisi tegak limas dengan teorema Pythagoras.

 t² = sisi miring² - (1/2 x alas)²

 t² = (10 cm)² - (1/2 x 12 cm)²

 t² = 100 cm² - 36 cm² = 64 cm²

 t = \sqrt{64 cm^2} = 8 cm

 LP = Luas alas + 4 x luas sisi tegak

 LP = (s x s) + 4 x (1/2 x alas x tinggi)

 LP = (12 cm x 12 cm) + 4 x (1/2 x 12 cm x 8 cm)

 LP = 144 cm² + 4 x (48 cm²)

 LP = 144 cm² + 192 cm² = 336 cm²

d. Diketahui: alas limas berbentuk persegi

                  panjang sisi alas = s = 14 cm

                  panjang sisi miring pada sisi tegak limas = 25 cm

 Ditanya: berapa luas permukaan limas?

 Jawab:

 Kita cari terlebih dahulu tinggi sisi tegak limas dengan teorema Pythagoras.

 t² = sisi miring² - (1/2 x alas)²

 t² = (25 cm)² - (1/2 x 14 cm)²

 t² = 625 cm² - 49 cm² = 576 cm²

 t = \sqrt{576 cm^2} = 24 cm

 LP = Luas alas + 4 x luas sisi tegak

 LP = (s x s) + 4 x (1/2 x alas x tinggi)

 LP = (14 cm x 14 cm) + 4 x (1/2 x 14 cm x 25 cm)

 LP = 196 cm² + 4 x (175 cm²)

 LP = 196 cm² + 700 cm² = 896 cm²

e. Diketahui: alas limas berbentuk persegi

                  panjang sisi alas = s = 6 cm

                  panjang sisi miring pada sisi tegak limas = 5 cm

Ditanya: berapa luas permukaan limas?

Jawab:

Kita cari terlebih dahulu tinggi sisi tegak limas dengan teorema Pythagoras.

t² = sisi miring² - (1/2 x alas)²

t² = (5 cm)² - (1/2 x 6 cm)²

t² = 25 cm² - 9 cm² = 16 cm²

t = \sqrt{16 cm^2} = 4 cm

LP = Luas alas + 4 x luas sisi tegak

LP = (s x s) + 4 x (1/2 x alas x tinggi)

LP = (6 cm x 6 cm) + 4 x (1/2 x 6 cm x 4 cm)

                  panjang sisi alas = s = 12 cm

                  tinggi limas = 8 cm

Ditanya: berapa luas permukaan limas?

(t sisi tegak)² = (t limas)² + (1/2 x alas)²

22. diketahui luas limas = 640 cm². luas alas = 160 cm luas selimut = 120 cm ditanya banyak sisi limas​


Jawaban:

8 sisi

Penjelasan dengan langkah-langkah:

Untuk menyelesaikan masalah ini, kita perlu mengetahui jumlah luas sisi limas terlebih dahulu. Jumlah luas sisi limas terdiri dari luas alas dan luas seluruh sisi selimut limas. Kita dapat menghitung luas sisi selimut dengan menggunakan rumus:

Luas Selimut = (keliling alas) x (tinggi limas) / 2

Untuk menghitung keliling alas, kita perlu mengetahui panjang sisi alas (a), karena alas limas berbentuk segi empat sama sisi, maka a dapat dihitung dengan mengakar dari luas alas (A), sehingga a = √A.

Diketahui:

Luas Alas (A) = 160 cm²

Luas Selimut (L) = 120 cm²

Luas Limas (T) = 640 cm²

Maka:

a = √A = √160 cm² = 12.65 cm

Kita dapat mencari tinggi limas (h) dengan menggunakan rumus luas limas:

Luas Limas = (Luas Alas x Tinggi Limas) / 3 + Luas Selimut

640 cm² = (160 cm² x Tinggi Limas) / 3 + 120 cm²

640 cm² - 120 cm² = (160 cm² x Tinggi Limas) / 3

520 cm² x 3 = 160 cm² x Tinggi Limas

Tinggi Limas = 9.75 cm

Setelah mengetahui tinggi dan sisi alas limas, kita dapat menghitung panjang sisi selimut limas (s) dengan menggunakan Pythagoras:

s² = a² + h²

s² = (12.65 cm)² + (9.75 cm)²

s² = 160.0625 cm²

s = √160.0625 cm²

s = 12.65 cm (sudut 2 desimal)

Jumlah sisi limas adalah 4 sisi alas (segi empat sama sisi) dan 4 sisi segitiga pada selimut limas. Sehingga, total sisi limas adalah 8.

Jadi, banyak sisi pada limas tersebut adalah 8 sisi
