Hari ulang tahun (birth day) adalah hari dimana seseorang pada umumnya merayakan hari, tanggal dan bulan saat dia dilahirkan ke dunia.

Kamu bisa mengungkap 'selamat ulang tahun' yang sering digunakan yaitu 'happy birthday' atau disingkat menjadi HBD.

27. Q.8³ - 3² + 7⁴ = ...Nt: Merry Christmas *Untuk yang merayakan :)​


8³ - 3² + 7⁴

= ( 8 × 8 × 8 ) - ( 3 × 3 ) + ( 7 × 7 × 7 × 7 )

= ( 64 × 8 ) - 9 + ( 49 × 49 )

= 512 - 9 + 2.401

= 503 + 2.401

= 2.904

semoga membantu~Bilangan berpangkat

–––––––––––––––––––––––––––––––

Pendahuluan:

Bilangan berpangkat adalah suatu bentuk perkalian berulang dari suatu bilangan yang sama. Bentuk umum perpangkatan adalah aⁿ

Contoh:

[tex] { \boxed{\color{darkblue}\begin{gathered}{ {\rm{ {a}^{n} = \: \: } \rm \underbrace{a \times a \times a \times ... \times a \times a \: \: \: }} }\\ \sf {}^{ \color{red}\: \: \: \: \: \: \: \: \: \: \: \: \: sebanyak \: n \: faktor \: \: \: } \end{gathered}}}[/tex]

a merupakan bilangan pokok (basis) sedangkan n merupakan pangkat (eksponen).

[tex] \: [/tex]

Sifat bilangan berpangkat

Berikut beberapa sifat bilangan berpangkat

[tex] \bf {a}^{0} = 1[/tex]

[tex] \bf {a}^{1} = a[/tex]

[tex] \bf {a}^{m} \times {a}^{n} = {a}^{m + n} [/tex]

[tex] \bf \frac{ {a}^{m} }{ {a}^{n} } = {a}^{m - n} [/tex]

[tex] \bf {a}^{ \frac{x}{y} } = {}^{y}\sqrt{ {a}^{x} } [/tex]

[tex] \bf {a}^{ - n} = \frac{1}{ {a}^{n} } [/tex]

[tex] \: [/tex]

Penyelesaian:

8³ - 3² + 7⁴

= (8 × 8 × 8) - (3 × 3) + (7 × 7 × 7 × 7)

= 512 - 9 + 2.401

= 503 + 2.401

= 2.904

Kesimpulan:

Hasil dari 8³ - 3² + 7⁴ adalah 2.904
