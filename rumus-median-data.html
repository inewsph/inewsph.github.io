28. rumus median data ganjil


Rumusnya adalah:

=> [tex]\frac{xn+1}{2}[/tex]

Contoh:

Median dari 31,32,33,34,35,36, dan 37 adalah ...

=> [tex]\frac{xn+1}{2}[/tex]

=> [tex]\frac{x7+1}{2}[/tex]

=> [tex]\frac{x8}{2}[/tex]

=> [tex]x4[/tex]

Nilai ke empat adalah 34
