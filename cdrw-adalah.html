1. bedakan antara cdroom,cdrw,cdroom drive, cdrw drive!



CDROM: CD Read Only -> hanya untuk melihat CD (tidak DVD)
CDRW : CD Read Write -> dapat untuk melihat dan membuat (burning) CD
DVDROM: DVD Read Only -> hanya untuk melihat DVD (dan CD)
DVDRW: DVD Read Write -> dapat untuk melihat DVD (dan CD) dan membuat DVD.

3. jelaskan perbedaan cdr dengan cdrw?


Pertama yaitu CD-R. CD-R adalah singkatan dari Compact Disk Recordable adalah piringan pipih berbahan polikarbonat berdiameter 120 mm. Untuk mengisi CD-R dengan data diperlukan sebuah perangkat drive khusus. Proses perekamnya juga dilakukan dengan perangkat lunak khusus untuk mem burning data ke dalam CD-R. Data yang ditulis didalamnya bersifat WORM ( Write Once Read Many ) yang menunjukan bahwa data hanya dapat diisi sekali dan selanjutnya hanya bisa dilihat / dibaca. Dapat diartikan jika kita telah burning data ke dalam CD-R, maka data tersebut sudah tidak bisa kita hapus lagi. Bisa dikatakan CD-R hanya dapat digunakan untuk satu kali pemakaian karena kita tidak bisa mengahapus ini data yang ada di dalamnya. Kedua yaitu CD-RW yang merupakan pengembangan dari CD-R, sebenarnya dari bentuk sama dengan CD-R hanya sedikit berbeda dalam kegunaan dalam hal penyimpanan data. Perbedaanya adalah CD-RW ( Compact Disk Recordable and reWriteable ) berbahan CD-RW menggunakan logam perpaduan antara perak, indium, antimon, dan telurrium untuk lapisan perekaman dan dapat diubah datanya berkali - kali, dapat di tambah, diganti atau dihapus sewaktu - waktu. Dapat diartikan jika kita telah memburning data kedalamnya, maka data tersebut masih bisa di hapus. Berbeda dengan CD-R, CD-RW bisa digunakan untuk menyimpan data berulang-ulang. Jadi seandainya  data yang ada pada CD-RW sudah tidak digunakan lagi, kita dapat menghapus dan menggantinya dengan yang baru.

1. Kelebihan dari CD+R dibanding CD-R:

Pada CD+R kita dapat menggunakan space kosong yang masih tersedia di disk.

Sedangkan pada CD-R kita tidak dapat menambahkan data lain jika sudah di gunakan, meskipun masih ada sisa penyimpanan. 

2. Kelebihan CD RW dibanding CD+R:

Pada CD RW, disk ini menggunakan material khusus sehingga datanya dapat dihapus kemudian digunakan untuk menyimpan data baru atau dapat juga di tumpuk dengan data lain.

1. CD-RW atau Compact Disc Rewriteable adalah salah satu bagian hardware komputer yang berfungsi untuk membaca file yang terdapat dalam CD. CDRW dapat ditulis ulang atau berkali-kali (bisa di burning berkali-kali dan bisa dihapus) Sedangkan CD-R atau Compact Disc Read adalah salahsatu bagian hardware komputer yang berfungsi hanya untuk membaca file yang terdapat dalam CD saja dan tidak dapat ditulisi.

2. a. Fungsi monitor adalah untuk menampilkan video dan informasi grafis yang dihasilkan dari komputer melalui alat yang disebut karu grafis (VGA Card).
b. Fungsi dari speaker adalah sebagai alat output suara yang dihasilkan dari komputer. 
c. Fungsi printer adalah sebagai alat output cetak dari dokumen elektronik baik bentuk teks maupun grafis.

3. Bagian utama dari memori adalah Input storage, Program storage, Working storage dan Output storage.

4. Tugas dari CU adalah sebagai berikut:
a. Mengatur dan mengendalikan alat-alat input dan output.
b. Mengambil instruksi-instruksi dari memori utama.
c. Mengambil data dari memori utama kalau diperlukan oleh proses.
d. Mengirim instruksi ke ALU bila ada perhitungan aritmatika atau perbandingan logika serta mengawasi kerja.
e. Menyimpan hasil proses ke memori utama.

5. Fungsi memory card alah sebagai tempat penyimpanan data digital seperti gambar, audio maupun video pada sebuah gadget.

6. RAM atau Random Access Memory adalah suatu bentuk penyimpanan data yang dapat diakses secara acak setiap saat dan dalam urutan apapun dan dari lokasi manapun, juga berbeda perangkat penyimpanan lainnya seperti hard drive. Sedangkan ROM atau Read Only Memory merupakan jenis memori yang hanya bisa dibaca dan tidak bisa diisi atau ditulisi data.

7. Administrator adalah orang yang bertugas untuk mengelola hal-hal yang berhubungan dengan komputer.
