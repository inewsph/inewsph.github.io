17. pada segitiga ABC diketahui besar sudut A = 75 derajat. nilai tan (B+C) adlah


Pada segitiga ABC diketahui besar sudut A = 75 derajat. Nilai tan (B + C) adalah –2 – √3. Hasil tersebut diperoleh dengan menggunakan rumus jumlah dan selisih sudut trigonometri.

sin (A + B) = sin A cos B + cos A sin B sin (A – B) = sin A cos B – cos A sin B cos (A + B) = cos A cos B – sin A sin B cos (A – B) = cos A cos B + sin A sin B tan (A + B) = [tex]\frac{tan \: A + tan \: B}{1 - tan \: A tan \: B}[/tex] tan (A – B) = [tex]\frac{tan \: A - tan \: B}{1 + tan \: A tan \: B}[/tex]

Pembahasan

Diketahui

Dalam segitiga ABC

Sudut A = 75⁰

Ditanyakan

tan (B + C) = ... ?

Jawab

Jumlah sudut dalam segitiga adalah 180⁰

A + B + C = 180⁰

B + C = 180⁰ – A

maka

tan (B + C)

= tan (180⁰ – A)

= [tex]\frac{tan \: 180^{o} - tan \: A}{1 + tan \: 180^{o} tan \: A}[/tex]

= [tex]\frac{0 - tan \: A}{1 + 0 \: . \: tan \: A}[/tex]

= [tex]\frac{- tan \: A}{1 + 0}[/tex]

= [tex]\frac{- tan \: A}{1}[/tex]

= – tan A

= – tan 75⁰

= – tan (30⁰ + 45⁰)

= [tex]-\frac{tan \: 30^{o} + tan \: 45^{o}}{1 - tan \: 30^{o} tan \: 45^{o}}[/tex]

= [tex]-\frac{\frac{1}{3} \sqrt{3} + 1}{1 - \frac{1}{3} \sqrt{3} \: . \: 1}[/tex]

= [tex]-\frac{\frac{1}{3} \sqrt{3} + 1}{1 - \frac{1}{3} \sqrt{3}}[/tex]

= [tex] \frac{\frac{1}{3} \sqrt{3} + 1}{\frac{1}{3} \sqrt{3} - 1}[/tex]

= [tex] \frac{\frac{1}{3} \sqrt{3} + 1}{\frac{1}{3} \sqrt{3} - 1} \times \frac{\sqrt{3}}{\sqrt{3}} [/tex]

= [tex] \frac{1 + \sqrt{3}}{1 - \sqrt{3}}[/tex]

= [tex] \frac{1 + \sqrt{3}}{1 - \sqrt{3}} \times \frac{1 + \sqrt{3}}{1 + \sqrt{3}} [/tex]

= [tex] \frac{1 + \sqrt{3} + \sqrt{3} + 3}{1 - 3}[/tex]

= [tex]\frac{4 + 2 \sqrt{3}}{-2}[/tex]

= –2 – √3
