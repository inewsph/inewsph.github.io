Berikut penjelasan mengenai masing-masing kata yang terdapat dalam kalimat tersebut :

Oniichan「 お兄ちゃん 」

Diartikan berarti kakak (lelaki). Dipakai jika ingin memanggil seorang yang lebih tua seperti kakak dengan gender lelaki. Adanya kata ちゃん (chan) setelah kata お兄 (onii), dikarenakan untuk menunjukkan keakraban.

Jika tidak memiliki hubungan keakraban, maka jangan menggunakan kata お兄ちゃん (oniichan) dan ganti dengan kata お兄さん (oniisan).

Panggilan kakak (lelaki) dalam bahasa Jepang tidak hanya merujuk ke kakak kandung, akan tetapi juga bisa dipakai sebagi panggilan kepada orang yang lebih tua seperti kakak kelas, abang, kakak angkat.

Baka「 ばか 」

Yaitu artinya bodoh. Sejenis bentuk kt kasar yang dipakai dalam merendahkan seseorang.

Arti dari お兄ちゃん ( Oniichan ) adalah kakak laki-laki

Pembahasan

Oniisan/ Oniichan dan Ani keduanya memiliki arti "Kakak" Diantara perbedaan kedua kalimat tersebut seperti berikut; Ani untuk kakak laki-laki sendiri sedangkan Oniisan bisa digunakan kepada kakak laki-laki sendiri maupun kakak laki-laki orang lain

____________________________________

Oniisan memiliki arti "Kakak laki-laki" Anda bisa mengucapkan Oniisan kepada kakak laki-laki sendiri maupun kakak laki-laki orang lain. Bentuk lain ( Bentuk Informal ) dari Oniisan adalah;

Nii-SanOnii chanNii-chan

Atau Bentuk lain ( Formal ) menunjukkan rasa sopan adalah; Onii-sama

Adapun 家族 ( Kazoku );日本語の家族 :

お兄ちゃん ( Onii-Chan ) Arti nya yaitu Kakak Laki-lakiお姉ーちゃん ( Onee-chan ) Merupakan Kalimat yang artinya Kakak PerempuanUntuk Adik laki-laki :弟 (Otouto) yang artinya yaitu Adik ( laki-laki )Untuk Adik perempuan 妹 (Imouto)Yang artinya Adik ( perempuan )Ayah :父( Chichi　) Yang artinya yaitu AyahIbu : 母 ( Haha　) Yang artinya yaitu IbuKakek :祖父 ( Sofu ) Arti : KakekNenek : 祖母 ( Sobo ) Yang artinya yaitu Nenek

Contoh kalimat;

1. 私は兄です ( Watashi wa Ani desu )→Saya adalah ( Seorang ) kakak ( Laki-laki )

Terjemah Per-Kata;

私 ( Watashi ) → Memiliki arti Saya

は ( Wa ) → Merupakan Partikel

兄 ( Ani ) → Memiliki arti Kakak

です ( Desu ) → Tidak memiliki arti berfungsi sebagai penghalus kalimat

2.私のお兄さんはダイチです ( Watashi No Oniisan wa Daichi Desu ) → Kakak saya adalah Daichi

Terjemah Per-Kata;

私 ( Watashi ) → Memiliki arti Saya

の ( No ) → Merupakan Partikel

兄さん( Oniisan ) → Memiliki arti Kakak laki-laki

は ( Wa ) → Partikel

ダイチ ( Daichi ) → Merupakan nama ditulis dalam penulisan Katakana

です ( Desu ) → Tidak memiliki arti berfungsi sebagai penghalus kalimat

私のお兄さんがいません ( Watashi No Oniisan ga Imasen ); Saya tidak mempunyai kakak laki-laki

Terjemah Per-Kata;

私 ( Watashi ); Memiliki arti Saya

の ( No );Merupakan Partikel

兄さん( Oniisan ); Memiliki arti Kakak laki-laki

が ( Ga ); Partikel

25. Yamete,Onichann Onichan itu artinya apa? Kakak atau setan?


Yamete, Oniichan 「やめて, お兄ちゃん」memiliki arti Hentikanlah, kakak (lk) !

(Jangan traveling kau ya).

PEMBAHASAN

Yamete 「やめて」adalah kata kerja bentuk ubahan dalam Bahasa Jepang yang menunjukan permintaan/suruhan untuk menghentikan sesuatu secara sopan. Adapun 2 bentuk lain dari yamete yaitu yamero 「やめろ」dan yameru 「やめる」. Perbedaannya adalah sebagai berikut:

1. Yamete 「やめて」

➢ Kata ini adalah kata kerja perintah/suruhan yang menggunakan akhiran -te. Akhiran -te ini memberikan konsepsi pada kata ini agar menjadi bentuk perintah/suruhan yang sopan. Contoh:

やめて、クラスで戦わないでください。

(Yamete, kurasu de tatakawanaide kudasai!).

Arti: Berhentilah, tolong jangan berkelahi dikelas!

2. Yamero「やめろ」

➢ Kata ini adalah kata kerja perintah/suruhan yang menggunakan akhiran -ro. Akhiran -ro ini memberikan konsepsi pada kata ini menjadi sedikit kasar/tidak sopan. Contoh:

この愚かさを止めろ、ヤシコ。

(Kono oroka sa o yamero, Yashiko!).

Arti: Hentikan kekonyolan ini, Yashiko!

3. Yameru「やめる」

➢ Kata ini adalah kata kerja yang menggunakan akhiran -ru. Akhiran ini memberikan konsepsi bahwa kata kerja ini bersifat dasar. Yameru bisa kita artikan dengan "berhenti", digunakan untuk kalimat non kompleks. Contoh:

大学をやめる。

(Daigaku o yameru).

Arti: Berhenti kuliah.
