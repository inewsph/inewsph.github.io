Penjelasan dengan langkah-langkah:

Minor :

[tex]M_{\textstyle ij} = \det\left(\bold{T_{(n-1)x(n-1)}(\bold{A_{nxn}},$a_{\textstyle ij}$))}\right)[/tex]

yaitu determinan yang dihasilkan dari matriks (n-1) x (n-1) yang didapatkan dengan cara mencoret baris dan kolom A (dengan ukuran nxn) dengan pusatnya yaitu elemen a_ij (sehingga di dapatkan bentuk coretan yang sama persis/mirip huruf T).

Kofaktor atau adjunksi :

[tex]C_{\textstyle ij} = A_{\textstyle ij} = (-1)^{\textstyle i+j} \cdot M_{\textstyle ij}[/tex]

Determinan :

1) Lewat ekspansi baris :

[tex]\displaystyle \det(\bold{A}) = \sum_{\textstyle j=1}^{\textstyle n} a_{\textstyle ij}\cdot C_{\textstyle ij} \; \to \; i \in [1,2,3,...,n] \text{ (pilih salah satu nilai i)}[/tex]

2) Atau bisa juga lewat ekspansi kolom :

[tex]\displaystyle \det(\bold{A}) = \sum_{\textstyle i=1}^{\textstyle n} a_{\textstyle ij}\cdot C_{\textstyle ij} \;\to \;j \in [1,2,3,...,n] \text{ (pilih salah satu nilai j)}[/tex]

yang biasa digunakan di SMA adalah determinan lewat ekspansi baris, dimana nilai i = 1 :

[tex]\displaystyle \det(\bold{A}) = \sum_{\textstyle j=1}^{\textstyle n} a_{\textstyle 1j}\cdot C_{\textstyle 1j} = a_{11}\cdot C_{11}+a_{12}\cdot C_{12}+a_{13}\cdot C_{13} + \hdots + a_{1n}\cdot C_{1n}\\\\\det(\bold{A}) = a_{11}\cdot M_{11}-a_{12}\cdot M_{12}+a_{13}\cdot M_{13}-\hdots + (-1)^{1+n}\cdot a_{1n}\cdot M_{1n}[/tex]

[tex]\bold{A} = \left[\begin{array}{ccc}1&0&0\\0&1&0\\2&0&1\end{array}\right][/tex]

[tex]M_{\textstyle 11} = \det\left(\bold{T(\bold{A},$a_{\textstyle 11}$)})\right) = \det\left(\bold{T(\bold{A},$1$)})\right) = \det\left(\begin{array}{ccc}\textcircled{1}&\Box&\Box\\\Box&1&0\\ \Box&0&1\end{array}\right)\\M_{\textstyle 11} =1\cdot 1 - 0\cdot 0 = 1[/tex]

[tex]M_{\textstyle 12} = \det\left(\bold{\bold{T(\bold{A},$a_{\textstyle 12})$}})\right) = \det\left(\bold{T(\bold{A},$0$)})\right) = \det\left(\begin{array}{ccc}\Box&\textcircled{0}&\Box\\0&\Box&0\\ 2&\Box&1\end{array}\right)\\\\M_{\textstyle 12} = 0\cdot 1 - 0\cdot 2 = 0[/tex]

[tex]M_{\textstyle 13} = \det\left(\bold{T(\bold{A},$a_{\textstyle 13})}$)\right) = \det\left(\bold{T(\bold{A},$0$)})\right) = \det\left(\begin{array}{ccc}\Box&\Box&\textcircled{0}\\0&1&\Box\\ 2&0&\Box\end{array}\right)\\M_{\textstyle 13} = 0\cdot 0 - 1\cdot 2 = -2[/tex]

[tex]\det(\bold{A}) = a_{11}\cdot M_{11}-a_{12}\cdot M_{12}+a_{13}\cdot M_{13}\\\det(\bold{A}) = 1\cdot M_{11}-0\cdot M_{12}+0\cdot M_{13}\\\\\det(\bold{A}) = M_{11}[/tex]

[tex]\boxed{\boxed{\textbf{\Huge{$\boldsymbol{\det(\bold{A}) = 1}$}}}}[/tex]

19. Carilah penyelesaian dari sistem persamaan linier di bawah ini menggunakan metode eliminasi Gaussian melalui operasi baris elementer! (Skor = 35) x₁ + x₂ + x3 + x4 = 0 x₁ + 2x₂ + 3x3 + 4x4 = 0 x₁ + 3x₂ + 5x3 - 4x4 = 0​


Jawaban:

Untuk menyelesaikan sistem persamaan linier tersebut menggunakan metode eliminasi Gaussian melalui operasi baris elementer, kita akan mengubah matriks augmented sistem tersebut menjadi matriks segitiga atas (upper triangular matrix). Berikut langkah-langkahnya:

Matriks augmented sistem:

[1 1 1 1 | 0]

[1 2 3 4 | 0]

[1 3 5 -4 | 0]

Kurangi baris pertama dengan baris kedua, kemudian kurangi baris pertama dengan baris ketiga:

[1 1 1 1 | 0]

[0 1 2 3 | 0]

[0 2 4 -5 | 0]

Kurangi baris kedua dengan dua kali baris pertama:

[1 1 1 1 | 0]

[0 1 2 3 | 0]

[0 0 2 -7 | 0]

Sehingga diperoleh matriks segitiga atas sebagai berikut:

[1 1 1 1 | 0]

[0 1 2 3 | 0]

[0 0 2 -7 | 0]

Terakhir, kita akan menggunakan metode substitusi mundur (backward substitution) untuk mencari solusi dari sistem persamaan linier ini. Berikut langkah-langkahnya:

Dari baris terakhir matriks segitiga atas, kita dapat langsung menentukan nilai x₄ = 7/2.

Substitusikan nilai x₄ = 7/2 ke dalam baris kedua matriks segitiga atas, sehingga diperoleh:

x₂ + 2x₃ + 21/2 = 0

Dari baris ini, kita dapat menentukan nilai x₃:

x₃ = -1/4 x₂ - 21/4

Substitusikan nilai x₃ ke dalam baris pertama matriks segitiga atas, sehingga diperoleh:

x₁ + 3/2 x₂ = 0

Dari baris ini, kita dapat menentukan nilai x₁:

x₁ = -3/2 x₂

Sehingga solusi dari sistem persamaan linier tersebut adalah:

x₁ = -3/2 x₂

x₂ = t (variabel bebas)

x₃ = -1/4 x₂ - 21/4

x₄ = 7/2

dengan t adalah variabel bebas.

Demikianlah penyelesaian dari sistem persamaan linier menggunakan metode eliminasi Gaussian melalui operasi baris elementer.
