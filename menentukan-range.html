2. menentukan domain & range


Domain:
Karena berupa persamaan kuadrat, maka domainnya adalah x bilangan real.
Range:
Tentukan titik baliknya
y = x^2 + 2x + 3
y = (x+1)^2 + 2
(y - 2) = (x+1)^2.
Parabola berpuncak di (-1, 2)
Karena koefisien x^2 = 1 bernilai positif, parabola terbuka ke atas dan y minimum bernilai 2. Jadi rangenya adalah y >= 2.

4. BAGAIMANA CARA MENENTUKAN DOMAIN JIKA RANGE YANG DIKETAHUI DALAM RELASI


contoh:
f(x) = x+1
range = { 1,2,3}
x+1 = 1
x= 1-1 = 0
x+1= 2
x=2-1= 1
x+1=3
x=3-1=2
jadi domain adalah {0,1,2}Setiap fungsi memiliki dua variabel, yaitu variabel bebas dan variabel terikat. Secara harfiah nilai variabel terikat “tergantung” pada variabel bebas. Sebagai contoh, dalam fungsi y = f(x) = 2x + y, x adalah variabel bebas dan y adalah variabel terikat (dengan kata lain, y adalah fungsi dari x). Nilai-nilai valid untuk variabel x yang diketahui disebut “domain/daerah asal.” Nilai-nilai valid untuk variabel y yang diketahui disebut “range/daerah hasil.” [1]

8. bagaimana cara menentukan domain dan range dalam fungsi?


domain = daerah asal
range = daerah hasil
contoh soal
suatu fungsi f(x) = 2x + 3 dengan daerah asal { 1,2,3,4,5} dan daerah kawan { 1 < x < 15} maka daerah hasilnya adalah
f(1) = 2x + 3
       = 2(1) + 3
       = 2 + 3
       = 5
f(2)  = 2(2) + 3
       = 4 + 3
       = 7
f(3) = 2(3) + 3
       = 6 + 3
       = 9
f(4) = 2(4) + 3
       = 8 + 3
       = 11
f(5) = 2(5) + 3
       = 10+3
       = 13
jawabannya { 5,7,9, 11,13}

21. Fungsi yang digunakan untuk menentukan standar deviasi dari suatu data (range) yaitu


Fungsi Untuk Menentukan Standar Deviasi

Standar deviasi memiliki arti yang berupa nilai statistik yang kemudian akan menetukan terhadap penyebaran data yang terdapat pada sebuah sampel. Selain itu, menetukan sebesapa dekat titik data seorang individu terhadap mean menuju nilai sampel. Standar deviasi sendiri adalah kumpulan dari data-data yang mendekati nol maka akan mendekati rata-rata, dan apabila nilai deviasi mendekati satu maka akan menjauhi dari nilai rata-rata.

Di Microsoft Excel sendiri, penggunaan fungsi standar deviasi diciptakan pula, karena sebagai mana yang telah kita ketahui fungsi dari Microsoft Excel ini adalah pengolahan data.

Hal tersebut dapat dituliskan dengan cara menuliskan

STDEV(...)

24. cara menentukan domain dan range dari f(x)=2x+5​


Jika diberikan fungsi f(x) = 2x + 5, maka:

- Untuk menentukan domain caranya adalah cari nilai x berapa saja yang membuat fungsi f(x) terdefinisi atau memiliki nilai.

Karena berapapun nilai x yang diberikan membuat fungsi tersebut memiliki nilai (misal x = -1, 0, 1/2, [tex]\sqrt{2}[/tex], dst) maka domain fungsi tersebut adalah semua bilangan riil (R) atau dituliskan:

[tex]D_f=[/tex] {[tex]x[/tex] | [tex]x[/tex] ∈ R}

- Untuk menentukan range kita tinggal melihat Df nya saja, karena Df adalah semua bilangan riil maka rangenya juga semua bilangan rill.

30. cara menentukan daerah hasil (range) suatu fungsi dari suatu himpunan + penjelasan​


Jawaban:

Sebuah relasi digunakan untuk memasangkan anggota dari sebuah himpunan ke anggota himpunan lainnya. Dalam relasi tidak terdapat aturan khusus dalam memasangkan anggota domain dengan anggota kodomain. Pemasangan anggota domain boleh memiliki lebih dari satu hubungan dengan anggota kodomain. Sebuah relasi ada yang memiliki aturan khusus, yaitu untuk memasangkan semua anggota himpunan daerah asal tepat satu ke daerah kawannya. Relasi dengan aturan demikian disebut sebagai fungsi atau pemetaan. Baca Lebih lanjut mengenai perbedaan antara relasi dan fungsi/pemetaan.
